# Next Workspace

This directory is a transitional workspace for next-only notes and extracted artifacts during the in-place hard cut.

Purpose:
- keep new `next`-only planning and implementation work in one place
- preserve legacy reference material in the same repository while packages become the only active mainline
- avoid scattering temporary hard-cut notes and extracted artifacts back into stable package locations before they are ready

Suggested use:
- add migration notes here
- add temporary hard-cut scripts here when they do not yet belong in `packages/*` or `docs/next-plan/*`
- add extracted `next` resources here before moving them into stable package locations
