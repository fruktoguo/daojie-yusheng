六维属性
优先级最高的怪物真源字段。当前大部分怪物都用这一组属性，保存时会原样写回模板。
数值倍率

装备槽位
这里的装备会参与真实数值计算，也会影响默认掉落补全逻辑。
技能列表
每行一个技能 ID，也支持逗号分隔。保存时会写回 `skills` 字段。
基准数值
这是 `valueStats` 模式的真源；只有模板未直接配置 `attrs` 时，它才会参与推导真实怪物属性。
掉落配置

实际数值预览
右侧只读显示当前模板在服务端同公式下的实际结果。这里不参与保存，用来校对编辑器与真实怪物是否一致。

行为说明
地图编辑只写本地文件，地图本身仍可依赖服务端热重载。
默认模式下，编辑器不会自动拉起或关闭 `@mud/server`，可与 `./start.sh` 并行使用。
如需由编辑器托管主游戏服，请用 `CONFIG_EDITOR_MANAGE_GAME_SERVER=1 ./packages/config-editor/start.sh` 启动。
编辑器前端由 Vite 托管，页面与脚本改动会热更新；本地 API 现在也会随源码变更自动重启。
这个工具不参与线上服务运行，也不对外暴露鉴权入口。
