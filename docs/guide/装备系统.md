# 装备系统

## 装备部位

角色共有 5 个装备槽位：

| 部位 | 说明 |
|------|------|
| 武器 | 主要提供攻击属性 |
| 头部 | 头盔、冠冕等 |
| 身体 | 铠甲、法袍等 |
| 腿部 | 护腿、靴子等 |
| 饰品 | 戒指、项链等特殊装备 |

## 装备品阶

装备和功法一样分为八个品阶：

| 品阶 | 强度 |
|------|------|
| 凡品 | ★ |
| 黄品 | ★★ |
| 玄品 | ★★★ |
| 地品 | ★★★★ |
| 天品 | ★★★★★ |
| 灵品 | ★★★★★★ |
| 圣品 | ★★★★★★★ |
| 帝品 | ★★★★★★★★ |

品阶越高，基础属性越强。每提升一个品阶，属性约提升 20%。

## 装备等级

装备有等级要求，等级越高基础属性越强。每级提供 0.5 点基础价值成长。

## 装备属性

装备可以提供的属性包括：
- 六维属性（体质、神识、感知、根骨、力量、经脉）
- 战斗数值（攻击、防御、命中、闪避、暴击等）
- 特殊属性（修炼速度、移动速度、掉宝率等）

### 属性价值

不同属性的"价值"不同：
- 生命值：每点价值提供 12 点 HP
- 灵力值：每点价值提供 8 点 Qi
- 攻防命闪暴等：每点价值提供 1 点
- 行动次数：每点价值提供 10 点（非常珍贵）

## 装备品质

同品阶同等级的装备，品质可能不同（属性百分比不同），品质越高属性越接近上限。

## 装备特效

部分装备带有特殊触发效果，在战斗中满足条件时自动触发。

## 小贴士

- 品阶是装备强度的决定性因素，优先追求高品阶
- 同品阶下关注品质百分比，越高越好
- 行动次数属性极为珍贵，有就优先装备
- 修炼速度属性对挂机收益影响很大
