# 功法与技能

## 功法系统

功法是角色战斗和修炼的核心。每部功法包含若干层数，修炼层数越高，提供的加成越强。

### 功法品阶

功法分为八个品阶，品阶越高越稀有、越强力：

| 品阶 | 说明 |
|------|------|
| 凡品 | 最基础的功法，入门容易 |
| 黄品 | 普通修士常用 |
| 玄品 | 有一定门槛的功法 |
| 地品 | 较为珍贵 |
| 天品 | 稀有强力 |
| 灵品 | 极为罕见 |
| 圣品 | 传说级别 |
| 帝品 | 最顶级的功法 |

品阶越高，修炼所需的经验越多，但提供的效果也越强。

### 功法境界

每部功法有四个修炼境界，代表你对这部功法的掌握程度：

- **入门** — 刚开始修炼，效果较弱
- **小成** — 修炼到约 1/3 层数
- **大成** — 修炼到约 2/3 层数
- **圆满** — 修满所有层数，效果最强

### 功法经验

修炼功法获得经验的速度受以下因素影响：

- **角色境界与功法品阶的差距**：境界高于功法对应等级时，修炼加速；低于时则减速
- **灵气浓度**：环境灵气越浓，修炼越快
- **装备加成**：部分装备提供功法经验加速

## 技能系统

功法修炼到一定程度会解锁战斗技能。

### 技能释放

释放技能需要满足：
1. **射程范围** — 目标必须在技能射程内
2. **冷却就绪** — 技能不在冷却中
3. **灵力充足** — 有足够灵力支付消耗

### 技能消耗

技能的灵力消耗与功法品阶和角色境界相关：
- 品阶越高的功法，技能消耗越大
- 境界越高，技能消耗也会相应增加

### 吟唱技能

部分强力技能需要吟唱时间：
- 吟唱期间会显示预警区域
- 施法者死亡会打断吟唱
- 被打断的技能不退还已消耗的灵力

## 小贴士

- 优先修炼与当前境界匹配的功法，效率最高
- 高品阶功法虽然强，但修炼慢，前期不一定划算
- 注意技能冷却管理，合理搭配不同冷却时间的技能
- 吟唱技能威力大但有风险，注意站位避免被打断
