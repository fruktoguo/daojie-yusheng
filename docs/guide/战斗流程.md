# 战斗流程

## 战斗概述

《道劫余生》的战斗是回合制实时战斗。每个回合（约 1 秒），角色可以执行攻击、释放技能或移动。

## 一次攻击的完整流程

当你对目标发起攻击时，伤害会经过以下判定链：

```
破防判定 → 闪避判定 → 化解判定 → 暴击判定 → 五行加成 → 防御减伤 → 暴击伤害 → 境界压制 → 最终伤害
```

### 各环节说明

1. **破防判定**：你的破防力对抗目标的化解力，成功则无视部分防御
2. **闪避判定**：目标的闪避对抗你的命中，成功则完全躲避攻击
3. **化解判定**：目标尝试化解你的攻击，降低伤害
4. **暴击判定**：你的暴击率对抗目标的抗暴击，成功则伤害翻倍
5. **五行加成**：技能的五行属性与目标的五行相克关系
6. **防御减伤**：目标的防御力减少最终伤害
7. **暴击伤害**：如果暴击成功，额外乘以暴击伤害倍率
8. **境界压制**：境界高的一方有额外伤害加成/减免

## 技能释放条件

释放技能前会依次检查：
1. 施法者和目标都存活
2. 目标在技能射程范围内
3. 技能冷却已结束
4. 灵力足够支付消耗

## 行动次数

每回合你可以执行的攻击次数取决于"行动次数"属性：
- 默认每回合 1 次行动
- 通过装备和 Buff 可以提升行动次数
- 行动次数越多，每回合输出越高

## 吟唱技能

部分强力技能需要吟唱（前摇）：
- 吟唱期间目标区域会显示预警
- 其他玩家可以看到预警并躲避
- 施法者死亡会打断吟唱
- 被打断后灵力不退还、冷却不重置

## 攻击地块

部分技能可以攻击地面上的建筑或阵法：
- 攻击地块不受境界压制影响
- 不触发闪避、暴击、破防等判定
- 只计算五行加成和额外伤害

## 小贴士

- 境界差距对战斗影响很大，尽量不要越级挑战
- 闪避和命中是对抗关系，堆高一方可以形成优势
- 合理利用吟唱技能的预警来躲避敌人的大招
- 提升行动次数是提高输出的重要途径
