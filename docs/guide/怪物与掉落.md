# 怪物与掉落

## 怪物刷新

### 基础刷新规则

- 怪物被击杀后，经过一段时间会在出生点附近重新刷新
- 每种怪物有固定的刷新间隔

### 清场加速

如果你在规定时间内清完一个刷新点的所有怪物，刷新速度会加快：

- 每次按时清完，刷新速度提升一档（加快约一倍）
- 连续清场可以不断叠加加速，最高加速到原来的 11 倍
- 如果超时才清完，加速重置归零

这意味着：打得越快，怪刷得越快，效率越高。

## 掉落机制

### 掉落概率

每只怪物有自己的掉落表，击杀后按概率掉落物品：

- 每件物品有独立的掉落概率
- 掉落率受你的**掉宝率**属性加成
- 极稀有物品（概率 ≤ 0.1%）额外受**稀有掉宝率**加成

### 掉宝率加成

掉宝率属性的效果：
- 正值时相当于"等效多杀"，比如 +100% 掉宝率约等于杀了两次
- 负值时掉率降低，但不会降到零

### 越级掉落衰减

如果你的等级比怪物高太多（超过 1 级以上），灵石掉落率会打 7 折。这鼓励玩家挑战与自己等级相近的怪物。

### 地块掉落加成

部分特殊地块会提供额外的掉落倍率加成，在这些区域打怪掉落更丰富。

## 击杀经验

击杀怪物可以获得境界修为经验，经验量与怪物强度相关。

## 小贴士

- 找到适合自己实力的刷怪点，保持清场加速效率最高
- 堆叠掉宝率属性可以显著提升收益
- 不要过度越级刷低级怪，灵石掉率会衰减
- 关注地图上的特殊掉落加成区域
