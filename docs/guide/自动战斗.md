# 自动战斗

## 功能概述

自动战斗（挂机）让你的角色自动寻找并攻击附近的敌人，无需手动操作。

## 开启条件

自动战斗需要满足：
- 角色存活（HP > 0）
- 没有正在执行的手动指令
- 没有正在导航移动
- 没有正在吟唱技能

## 两种模式

- **自动战斗**：主动寻找附近敌人并攻击
- **自动反击**：只在被攻击时才还手

## 目标选择

自动战斗会根据评分选择最优目标，评分考虑：

- **仇恨值**：攻击过你的敌人优先级更高
- **距离**：越近的目标评分越高（距离每远一格，评分衰减）
- **不可达惩罚**：无法到达的目标评分大幅降低

### 目标偏好

你可以设置目标偏好来调整优先级：

| 偏好模式 | 效果 |
|----------|------|
| 最近目标 | 优先攻击距离最近的敌人 |
| 低血量 | 优先攻击血量最低的敌人 |
| 高血量 | 优先攻击血量最高的敌人 |
| Boss | 优先攻击 Boss 级怪物 |
| 玩家 | 优先攻击其他玩家（PVP） |

## 静止模式

开启静止模式后，角色不会移动追击敌人，只攻击射程内的目标。适合：
- 守点刷怪
- 避免被引怪到危险区域

## 自动用药

挂机时角色会在血量低时自动使用回复药品，确保不会轻易死亡。

## 行动预算

每回合的攻击次数受"行动次数"属性限制：
- 默认每回合 1 次攻击
- 属性越高，每回合能打出的攻击越多

## 小贴士

- 根据刷怪场景选择合适的目标偏好
- 静止模式适合在安全位置挂机
- 确保背包里有足够的回复药品
- 行动次数是挂机效率的关键属性
