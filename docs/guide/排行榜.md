# 排行榜

## 功能概述

排行榜展示服务器中各方面最强的玩家和宗门，每 10 分钟刷新一次。

## 排行榜类型

共有 8 种排行榜：

| 排行榜 | 排名依据 | 说明 |
|--------|----------|------|
| 境界榜 | 境界等级 + 修为进度 + 底蕴 | 修炼最快的玩家 |
| 击杀榜 | 怪物总击杀数 | 含精英和 Boss 击杀 |
| 财富榜 | 灵石持有量 | 最富有的玩家 |
| PK 榜 | 玩家击杀数 | PVP 最强的玩家 |
| 死亡榜 | 死亡次数 | 死亡最多的玩家 |
| 体修榜 | 体修等级 + 经验 | 体修路线最强者 |
| 属性榜 | 六维属性最高值 | 单项属性最强者 |
| 宗门榜 | 宗门成员数量 | 最大的宗门 |

## 显示规则

- 每个排行榜显示前 **10 名**
- 每 **10 分钟**自动刷新排名

## 小贴士

- 排行榜是展示实力的舞台，也是了解服务器强者的窗口
- 不同排行榜代表不同的游戏方向，选择适合自己的目标
- 境界榜是最核心的排行，代表整体修炼进度
