# 地图与移动

## 地图概述

游戏世界由多张地图组成，每张地图由网格状的地块构成。不同地形有不同的通行难度和特性。

## 地形类型

不同地形的移动难度不同：

| 地形 | 通行难度 | 说明 |
|------|----------|------|
| 道路 | 很快 | 最容易通行的地形 |
| 小径 | 快 | 比道路稍慢 |
| 草地 | 正常 | 普通地形 |
| 地板/门/传送点 | 正常 | 建筑内地形 |
| 山丘 | 较慢 | 需要爬坡 |
| 泥地 | 慢 | 行动受阻 |
| 沼泽 | 很慢 | 严重减速 |
| 墙壁/悬崖/水面 | 不可通行 | 无法穿越 |
| 熔岩池 | 极慢 | 几乎无法通行 |

## 移动速度

### 基础移动
每回合角色获得移动点数，积累足够点数后可以移动一格。

### 移动速度属性
- 移动速度越高，每回合获得的移动点数越多
- 移动速度有**软上限**（500），超过后收益递减
- 即使速度很高，也不会无限快

### 软上限机制
- 速度 ≤ 500：正常生效
- 速度 > 500：超出部分按对数衰减（越高越难提升）

## 寻路系统

点击目标位置后，角色会自动寻找最优路径前往：
- 使用 A* 算法计算最短路径
- 会自动绕开障碍物
- 优先选择移动代价低的地形（走大路比翻山快）
- 最远寻路距离约 96 格

### 路径重算
如果行进途中路径被阻挡（比如有人挡路），系统会自动重新计算路径。

## 地形耐久

地图上的地块有生命值，可以被攻击破坏：
- 被破坏的地块约 **2 小时**后自动恢复
- 受损地块每回合恢复 1% 生命值
- 如果恢复时有东西占据该位置，会延迟恢复

## 小贴士

- 走大路比走野地快很多，规划路线时注意地形
- 移动速度超过 500 后收益递减，不必过度堆叠
- 寻路会自动选择最优路径，但手动规划有时更高效
- 注意避开沼泽等减速地形，尤其在战斗中
