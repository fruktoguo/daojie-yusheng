# 阵法系统

## 功能概述

阵法是消耗灵石布置的大范围效果，可以聚集灵气、稳定地形或设置屏障。阵法是宗门防御和修炼环境营造的核心手段。

## 四种阵法

| 阵法 | 效果 | 用途 |
|------|------|------|
| 聚灵阵 | 向区域内注入灵气 | 提升修炼环境 |
| 固脉阵 | 稳定地形，防止破坏 | 保护重要区域 |
| 太玄封界阵 | 设置边界屏障 | 阻挡外人进入 |
| 护宗大阵 | 宗门专属屏障 | 保护宗门领地 |

## 阵法资源

### 灵石投入

阵法需要消耗灵石来激活和维持：
- 投入的灵石越多，阵法效果越强
- 每颗灵石提供 100 点灵气预算

### 阵盘品阶

阵盘品阶决定灵气预算的倍率：

| 品阶 | 倍率 |
|------|------|
| 凡品 | ×1 |
| 黄阶 | ×2 |
| 玄阶 | ×4 |
| 地阶 | ×8 |

高品阶阵盘能让同样的灵石发挥更大效果。

## 灵气预算分配

激活阵法时，灵气预算在三个方面分配：

- **效果强度**：阵法效果的强弱
- **覆盖范围**：阵法影响的半径大小
- **持续时间**：阵法能维持多久

默认三者均分，你可以根据需要调整比例。

## 维持消耗

阵法激活后每回合消耗灵气维持：
- 激活状态消耗较高
- 休眠状态消耗降为 1/10
- 灵气耗尽时阵法失效

## 小贴士

- 聚灵阵是提升修炼效率的最佳手段
- 高品阶阵盘性价比远高于低品阶
- 根据实际需求调整效果/范围/时间的分配比例
- 宗门应优先布置护宗大阵保护领地安全
- 阵法需要持续投入灵石维护，注意资源规划
