# 灵气场

## 功能概述

灵气场是地图上自然存在或人工营造的灵气浓度区域。灵气浓度直接影响修炼速度，是选择修炼地点的关键因素。

## 灵气衰减

逸散在环境中的灵气会随时间自然衰减：
- 衰减遵循"半衰期"规律：每经过一定时间，灵气浓度减半
- 默认半衰期约为 **86400 回合**（约 24 小时）
- 这意味着无人维护的灵气场会逐渐消散

## 灵气等级

灵气浓度达到一定阈值会提升灵气等级，等级越高修炼加成越大。

## 灵气来源

### 自然灵脉
- 地图上天然存在的灵气富集区域
- 灵气持续产生，不会消散
- 通常是修炼的热门地点

### 人工聚灵
- 通过阵法聚集灵气
- 通过特殊建筑产生灵气
- 需要持续投入资源维护

## 灵气对修炼的影响

在灵气浓郁的地方修炼：
- 境界修为获取速度提升
- 功法经验获取速度提升
- 灵气等级越高，加成倍率越大

## 小贴士

- 优先在灵气浓郁的地点修炼和挂机
- 宗门可以通过阵法和建筑营造高灵气环境
- 自然灵脉是公共资源，可能需要竞争
- 人工灵气场需要持续维护，否则会逐渐消散
