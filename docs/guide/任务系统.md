# 任务系统

## 功能概述

任务系统引导玩家体验游戏内容，完成任务可以获得经验、物品和灵石奖励。

## 任务类型

| 类型 | 说明 |
|------|------|
| 主线任务 | 推动剧情发展的核心任务 |
| 支线任务 | 额外的故事和挑战 |
| 日常任务 | 每日可重复完成的任务 |
| 奇遇任务 | 随机触发的特殊任务 |

## 任务状态

每个任务有四种状态：

```
可接取 → 进行中 → 可提交 → 已完成
```

- **可接取**：满足前置条件，可以从 NPC 处接取
- **进行中**：已接取，正在完成目标
- **可提交**：目标已完成，可以回去交任务
- **已完成**：已提交，获得奖励

注意：如果完成条件不再满足（比如提交物品被消耗了），状态会从"可提交"退回"进行中"。

## 任务目标类型

| 目标类型 | 完成方式 |
|----------|----------|
| 击杀怪物 | 击杀指定数量的目标怪物 |
| 对话 | 与指定 NPC 对话 |
| 提交物品 | 上交指定数量的物品 |
| 学习功法 | 学会指定的功法 |
| 达到境界 | 境界达到指定等级 |
| 突破境界 | 境界突破到指定大境界 |

## 任务导航

接取任务后，系统会提供导航指引，帮助你找到目标位置。支持跨地图导航。

## 小贴士

- 主线任务是了解游戏世界和解锁内容的主要途径
- 日常任务是稳定的资源来源，每天记得做
- 提交物品类任务会消耗你背包中的物品
- 奇遇任务奖励丰厚，遇到不要错过
