# 属性系统

## 六维属性

每个角色拥有六项基础属性，它们是你战斗力的根基：

| 属性 | 名称 | 主要作用 |
|------|------|----------|
| 体质 | constitution | 增加生命上限、物理攻击、物理防御 |
| 神识 | spirit | 增加法术攻击、法术防御、命中 |
| 感知 | perception | 增加闪避、移动速度、抗暴击 |
| 根骨 | talent | 增加生命上限、灵力上限、化解力 |
| 力量 | strength | 增加物理攻击、暴击、破防力 |
| 经脉 | meridians | 增加灵力上限、灵力输出、法术攻击 |

每点属性会为对应的战斗数值提供 **1%** 的百分比加成（感知对移动速度为 0.5%）。

## 属性从哪里来

你的最终属性由多个来源叠加而成，按以下顺序计算：

1. **初始六维** — 角色创建时的基础值（每项默认 10）
2. **境界加成** — 随着境界提升，六维会获得额外加成
3. **功法加成** — 修炼的功法会提供六维加成
4. **装备加成** — 穿戴的装备提供固定六维
5. **Buff 加成** — 丹药、阵法等临时效果

## 战斗数值面板

六维属性最终会转化为具体的战斗数值：

- **生命值（HP）**：角色能承受的伤害总量
- **灵力值（Qi）**：释放技能消耗的资源
- **物理攻击/法术攻击**：决定你造成的伤害
- **物理防御/法术防御**：减少你受到的伤害
- **命中**：影响攻击是否被闪避
- **闪避**：影响你躲避攻击的概率
- **暴击**：影响造成暴击伤害的概率
- **抗暴击**：降低被暴击的概率
- **破防力**：穿透对方防御的能力
- **化解力**：抵抗对方破防的能力
- **移动速度**：每回合能移动的距离

## 小贴士

- 六维属性是"乘区"加成，意味着属性越高，每点带来的绝对收益越大
- 负面效果（如减益 Buff）采用反比衰减，不会让你的属性降到零
- 合理分配属性来源（功法、装备）比单纯堆高某一项更有效
