# 宗门系统

## 功能概述

宗门是玩家组成的组织，拥有独立领地、共享资源和集体防御能力。加入宗门可以获得更好的修炼环境和社交体验。

## 宗门职位

宗门内有以下职位，从高到低：

| 职位 | 说明 |
|------|------|
| 宗主 | 宗门最高领导者 |
| 副宗主 | 协助宗主管理 |
| 太上长老 | 荣誉职位 |
| 长老 | 核心管理层 |
| 内门弟子 | 正式成员 |
| 外门弟子 | 普通成员 |
| 杂役 | 新加入的成员 |

## 宗门权限

不同职位拥有不同权限：
- **护宗大阵操作**：控制宗门防御阵法
- **移除成员**：将成员踢出宗门
- **修改职位**：调整成员的职位等级

## 宗门领地

### 建宗
建立宗门时会清理周围区域作为初始领地。

### 领地扩展
宗门可以扩展领地范围，每次扩展 8 格。

### 领地保护
- 宗门领地内有固有稳定器（半径 8 格）
- 护宗大阵提供额外防御屏障

## 护宗大阵

宗门专属的防御阵法：
- 初始灵力 **100,000**
- 阻挡外敌入侵宗门领地
- 需要持续投入灵石维护
- 只有拥有"护宗大阵操作"权限的成员才能控制

## 宗门玩法

- **共同修炼**：宗门领地通常灵气浓郁
- **集体防御**：护宗大阵保护所有成员
- **资源共享**：宗门成员可以共享修炼资源
- **领地建设**：共同建设宗门设施

## 小贴士

- 加入活跃的宗门可以获得更好的修炼环境
- 为宗门贡献资源可以提升职位
- 护宗大阵是宗门安全的核心，注意维护
- 宗门领地内建造建筑更安全
