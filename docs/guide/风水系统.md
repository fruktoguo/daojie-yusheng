# 风水系统

## 功能概述

风水系统评估你建造的房间和建筑布局的好坏，风水越好，修炼和制作的加成越高。

## 风水等级

风水分数范围为 -1000 到 +1000，对应 11 个等级：

| 等级 | 分数要求 | 效果 |
|------|----------|------|
| 洞天福地 | ≥ 900 | 极强加成 |
| 福地 | ≥ 750 | 很强加成 |
| 大吉 | ≥ 600 | 强加成 |
| 吉 | ≥ 400 | 较好加成 |
| 小吉 | ≥ 200 | 轻微加成 |
| 平 | ≥ 0 | 无加成无减益 |
| 小凶 | ≥ -200 | 轻微减益 |
| 凶 | ≥ -400 | 较大减益 |
| 大凶 | ≥ -600 | 强减益 |
| 灾 | ≥ -800 | 很强减益 |
| 劫 | -1000 | 极强减益 |

## 五行相生相克

风水系统的核心是五行关系：

**相生（有利）：** 木 → 火 → 土 → 金 → 水 → 木

**相克（不利）：** 木克土、土克水、水克火、火克金、金克木

## 房间五行属性

不同功能的房间有对应的五行属性：

| 房间类型 | 五行 |
|----------|------|
| 庭院 | 木 |
| 静室（修炼） | 水 |
| 丹房（炼丹） | 火 |
| 器房（炼器） | 金 |
| 库房 | 土 |
| 卧室 | 木 |
| 宗门大殿 | 土 |

## 如何提升风水

- **五行相生布局**：让相邻房间的五行形成相生关系
- **避免相克**：不要把相克的房间放在一起
- **合理规划**：房间大小、形状、朝向都会影响风水
- **装饰加分**：适当的装饰物可以提升风水分数

## 小贴士

- 丹房（火）旁边放庭院（木）是相生关系，风水加分
- 丹房（火）旁边放器房（金）是相克关系，风水减分
- 追求"洞天福地"需要精心规划整体布局
- 风水影响修炼和制作效率，值得花时间优化
