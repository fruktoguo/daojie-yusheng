# 境界与修炼

## 修仙境界总览

游戏共有 **34 个大境界**，从凡俗到飞升，每个大境界包含若干小等级：

| 阶段 | 境界 | 等级范围 |
|------|------|----------|
| 凡人 | 凡俗 → 先天 | 1-18 |
| 练气 | 练气前期 → 练气后期 | 19-30 |
| 筑基 | 筑基前期 → 筑基后期 | 31-42 |
| 金丹 | 金丹前期 → 金丹后期 | 43-54 |
| 元婴 | 元婴前期 → 元婴后期 | 55-66 |
| 化神 | 化神前期 → 化神后期 | 67-78 |
| 炼虚 | 炼虚前期 → 炼虚后期 | 79-90 |
| 合体 | 合体前期 → 合体后期 | 91-102 |
| 大乘 | 大乘前期 → 大乘后期 | 103-114 |
| 渡劫 | 渡劫前期 → 渡劫后期 | 115-126 |
| 终点 | 飞升 | 127 |

凡人阶段细分为：凡俗、淬体、锻骨、通脉、先天。

## 怎么提升境界

### 积累修为

通过**闭关修炼**积累修为经验。修为达到当前境界上限后，就可以尝试突破。

影响修炼速度的因素：
- **修炼速度属性**：装备和功法提供的 realmExpPerTick
- **灵气浓度**：所在地点的灵气越浓，修炼越快
- **经验加成**：各种 Buff 带来的倍率加成

### 突破条件

每次大境界突破需要同时满足：

1. **修为满额** — 修为经验达到上限
2. **突破材料** — 消耗指定的突破道具
3. **功法要求** — 功法需要修炼到一定层数或品阶
4. **其他条件** — 部分境界要求属性总值、特殊灵根等

## 境界的意义

- **属性提升**：每个境界都会增加六维属性基础值
- **战斗压制**：境界高的一方在战斗中有额外伤害加成
- **解锁内容**：更高境界解锁新地图、新功法、新装备
- **技能强化**：部分技能效果随境界提升而增强

## 底蕴系统

修炼过程中会积累"底蕴"，底蕴越深厚：
- 突破成功率越高
- 突破后获得的属性加成越多
- 代表角色根基是否扎实

## 小贴士

- 不要急于突破，适当积累底蕴对后期发展更有利
- 选择灵气浓郁的地点修炼可以大幅提升效率
- 突破材料可以通过打怪掉落、市场购买或炼丹获得
