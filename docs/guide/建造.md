# 建造

## 功能概述

建造是一种特殊技艺，用于在地图上建造各种建筑物。建筑可以提供修炼加成、防御功能等多种用途。

## 建造流程

1. **选择建筑** — 选择要建造的建筑类型
2. **选择位置** — 在地图上指定建造位置
3. **开始建造** — 角色开始施工
4. **建造完成** — 建筑落成，开始发挥功能

## 建造条件

建造过程中需要持续满足条件：
- 你必须是该建筑的当前建造者
- 建筑必须处于"建造中"状态
- 你必须在建筑附近

如果条件不满足（比如你走开了），建造会暂停，5 回合后自动重试。

## 建筑属性

每个建筑都有：
- **生命值**：默认 100 点，被攻击会损坏
- **建造时间**：不同建筑建造耗时不同
- **拆除时间**：不需要的建筑可以拆除

## 建造技能

建造技能等级影响：
- 建造速度
- 可建造的建筑种类
- 建造成功率

## 小贴士

- 建造需要持续在场，不能中途离开
- 选择安全的位置建造，避免被怪物打断
- 提升建造技能等级解锁更多建筑类型
- 建筑位置很重要，会影响风水和灵气
