# 伤害计算

## 核心机制：对抗率

游戏中的命中、闪避、暴击、破防等判定都使用"对抗率"机制：

- 你的进攻属性与对方的防御属性进行对比
- 差距越大，成功率越高，但存在收益递减
- 即使属性远超对方，也不会达到 100% 成功率

简单理解：属性领先越多优势越大，但永远不会绝对碾压。

## 判定环节详解

### 1. 破防判定

- 条件：你的**破防力** > 目标的**化解力**
- 效果：破防成功后，后续的命中和暴击属性翻倍
- 意义：破防是连锁反应的起点，大幅提升后续判定的成功率

### 2. 闪避判定

- 你的**命中** vs 目标的**闪避**
- 闪避成功：本次攻击完全无效（Miss）
- 如果之前破防成功，你的有效命中翻倍，更难被闪避

### 3. 化解判定

- 条件：目标的**化解力** > 你的**破防力**
- 效果：化解成功后，目标的防御力翻倍
- 意义：化解是防御方的反制手段

### 4. 暴击判定

- 你的**暴击** vs 目标的**抗暴击**
- 暴击成功：伤害乘以暴击伤害倍率
- 如果之前破防成功，有效暴击翻倍

### 5. 防御减伤

- 目标的防御力按对抗率公式减少伤害
- 如果化解成功，防御力翻倍，减伤更多

### 6. 境界压制

- 攻击方境界高于防御方时，伤害额外增加
- 防御方境界高于攻击方时，伤害额外减少
- 境界差距越大，压制效果越明显

### 7. 五行相克

技能带有五行属性时，会根据目标的五行属性产生加成或减益：
- 相克：伤害增加
- 相生：伤害减少

## 伤害类型

- **物理伤害**：由物理攻击力决定，被物理防御减免
- **法术伤害**：由法术攻击力决定，被法术防御减免

## 小贴士

- 破防是最关键的判定，破防成功后命中和暴击都翻倍
- 面对高闪避目标，优先提升破防力而非命中
- 化解力是防御方的核心属性，能让防御翻倍
- 境界压制影响很大，尽量避免跨境界挑战强敌
