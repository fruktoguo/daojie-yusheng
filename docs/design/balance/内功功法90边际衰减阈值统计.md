## 数据与公式来源

- 数据源：`packages/server/data/content/techniques/**/*.json`
- 统计对象：`category === "internal"` 的内功功法，按满层累计 `layers[].attrs`
- 公式来源：
  - `packages/shared/src/constants/gameplay/technique.ts`
  - `packages/shared/src/technique.ts` 的 `calcTechniqueFinalAttrBonus()`

当前功法属性最终加成按品阶和属性分别成池，并对每个品阶池应用软衰减：

```text
raw <= freeLimit:
  final = raw

raw > freeLimit:
  final = freeLimit + decaySpan * ln(1 + (raw - freeLimit) / decaySpan)
```

本文把“达到 90% 衰减”定义为边际收益衰减 90%，也就是新增 1 点同品阶同属性原始池，只剩 0.1 点公式收益。

```text
marginal = 1 / (1 + (raw - freeLimit) / decaySpan)
90% 边际衰减 => marginal = 0.1
raw = freeLimit + 9 * decaySpan
```

注意：`calcTechniqueFinalAttrBonus()` 最终会对每个属性 `Math.floor()`。下表中的“公式后增量”保留两位小数，用于看阈值位置；实际面板会向下取整。
