# 剧情设计

本文档目录用于整理《道劫余生》的主线、时期称呼、章节目标与长期任务结构。

当前先确认以下叙事原则：

- 玩家不是唯一主角，而是乱世中许多入局者之一
- 多个玩家默认真实存在，彼此属于同路人、同道者
- `realmLv 1-9` 以 `外乡人` 为阶段称呼
- `realmLv 10-18` 以 `少侠` 为阶段称呼
- `realmLv 19+` 不再强推线性主线，改为长期任务《寻仙录》
- 世界底色黑暗破败，但整体秩序仍勉强运转，不采用“世界已经毁灭”的写法
- 前期表层是武侠江湖，仙侠内容只作为暗线逐步显形

后续建议按以下层级继续扩写：

- 卷：对应大时期，如 `外乡人`、`少侠`、`寻仙录`
- 章：对应一段明确剧情推进，通常绑定一张地图或一个区域事件
- 节点：对应可实现的任务链、关键 NPC、战斗点、交互点与埋线
