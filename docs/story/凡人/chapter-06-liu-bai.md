# 《寻仙录》第一卷·外乡人 终章 封天

状态：重构初稿

适用阶段：

- 玩家称呼：`外乡人`（本章结束时转为 `少侠`）
- 建议等级：`realmLv 25-30`
- 主要地图：`sky_ruins`（再入最深处）→ `yunlai_town`（回响交付）→ 封天井

说明：

- 凡人篇最终章。不是颁奖，是凡人篇最后一战和凡人世界的告别
- 击败天穹宫主 → 开启封天井 → 镇中回响 → 踏入封天井，进入练气期
- 宫主不是传统"最终Boss"——他是一个从崩断那天起就没有出来过的人

## 1. 本章目标

- 让玩家面对并击败天穹残宫最深处的那个人——天穹宫主
- 打开封天井——凡人世界与修行世界之间的一道旧通道
- 让全卷后果在镇里形成最后一次回声
- 让玩家从封天井踏入练气期——无缝过渡，不是生硬切换
- 凡人篇全卷收束

本章不做：

- 不做宫主长篇独白揭秘
- 不让宫主解释"这一切的真相"
- 不让封天井被NPC解释清楚（它是一个通道，至于为什么被封、通往哪里，没人完整知道）
- 不做煽情告别
- 不让清玄给出最终总结

## 2. 章节摘要

天封核心带回后，云来镇暂时稳住了。但清玄没有说"结束了"——他只说残宫最深处还有一个没出来的人，一口被封住的井。

玩家再次进入天穹残宫，穿过已经清过的外环和已经暗下来的核心区，找到一条之前没注意过的路——通往最深处。那里不像残宫的其余部分那样破碎。它很安静。宫主就在里面。

击败宫主后，封天井解封。宫主不说真相、不交代来历——他只留下了一个空的井口和一句听不太清的话。清玄在井边等着，说这是他送到的最后一步。之后的路，井那边的人会接。

玩家回到云来镇做最后一次交付。沈七娘、兰药师、马会头、老关卒、赵老六——每个人对"你要走了"这件事给出不同的反应。然后玩家回到封天井，踏进去。

凡人篇结束。练气期从井的另一端开始。

## 3. 氛围关键词

- 天穹宫主
- 最深处
- 封天井
- 他不是怪物

- 镇中最后的回声
- 井后面是什么
- 踏入

## 4. 主要角色

### 4.1 天穹宫主

- 身份：天穹残宫最深处的人——或许曾经是人
- 作用：凡人篇最终Boss
- 人设：不说话了——或者说几乎不说话了。从封柱崩断那天起就再没出来过。观星残影说"宫主"像是在说一个职位，不是一个身份。没有人知道他为什么留下来，在守什么，还是被困住了。他自己可能也不清楚了
- 战斗设计方向：宫主的战斗不是狂暴猛攻型。他的招式缓慢、沉重、带着某种重复感——像是他已经把同一套动作做了无数次。他的攻击方式带有封柱崩坏后的残余力量：空间撕裂、星铁碎片、以及一种压得人喘不过气的沉默感。他不是在战斗——他更像是在执行一个早已失效的指令
- 叙事约束（红线）：
  - 宫主在整场战斗中最多只说一句话——被击败时
  - 那句话不能是"真相揭晓"。它应该是碎片化的、可以让玩家记住但无法理解的
  - 建议方向：他说了一句关于井的话——像是"井后面……"然后没说完，或者说出一个没人听过的地名就断了
  - 他不解释自己是谁、在守什么、为什么不出来
  - 击败后他不消失、不爆炸、不化光——他只是停下来了，像一个站了太久终于可以不再站着的人
  - 不要给他名字。他就是"宫主"
- 他留下的碎片：封天井解封（井口的旧封印在宫主被击败后自行消散）。除此之外他什么都没留下——没有遗言、没有遗物、没有真相

### 4.2 清玄执事

- 身份：凡人篇全卷引路人
- 作用：在封天井边完成最后的叙事交接
- 人设：清玄在本章说的话比前面任何一章都少。他在井边等玩家，确认宫主已经被击败，然后说了一句："井那边有人。他们会告诉你前面的路。我送到这里。"
- 叙事约束（红线）：清玄不解释封天井的来历、宫主的身份、道劫是什么。他完成了他的叙事职能——把玩家从"外乡人"推成"能走进封天井的人"
- 清玄的最后一句台词："我送到这里"——和他在练气期开头说的一模一样，形成闭环

### 4.3 沈七娘 / 兰药师 / 马会头 / 老关卒 / 赵老六

- 身份：全卷常驻NPC
- 作用：提供"你要走了"的最后回声——每人一句，每个人用自己最关心的角度理解这件事
- 叙事约束：
  - 沈七娘：不说"保重"，不说"谢谢你"。她说："炕给你留着。万一井那边不收你，这边还有个能缩的地方。"——最朴素的情感收束
  - 兰药师：不说"你救了镇子"。她说："那个伤者收口了。你走之前他让我跟你说——他没记住你叫什么。但他记得你是把他从断道上拖回来的人。"
  - 马会头：不说"你是好汉"。他说："外面现在没那么乱了。不过你走了以后，门还是得有人看。"
  - 老关卒：不说再见。他说："北井那边最近夜里没动静。你也别去看了。"
  - 赵老六：他愣了一下，然后说："你刚来的时候我还嫌你碍事。现在你是要进封天井的人了。"
  - 所有回声不形成一个统一的意义——各人从各自的角度看"你要走了"这件事

## 5. 任务簇

### A 簇：再入残宫线

作用：

- 让玩家从"镇子稳了"回到"最后一战还没打"
- 找到通往最深处的路

推荐任务序列：

1. `talk` — 清玄在天穹残宫入口等你。他手里拿着天封核心，说核心已经安在镇里的旧基座上了，镇子暂时稳了。"但里面那个人还没出来。你要去的话，这次我不进。"

2. `talk` — 穿过已经清过的外环和已经暗下来的旧战场（噬星兽的尸体已经不在了——地上的痕迹还在），在观星残影的位置找到它。它已经更暗淡了，几乎看不清。它说："你见过他了。在第一段残影里——柱前那个人，和最深处的那个，是同一个人。"它说完就散掉了。

3. `kill_collect` — 通往最深处的一段路上仍有残余的天宫傀壳和星铁残卫。清掉它们。这段路上没有残影——因为连残影都不肯进这里。在入口收集最后一批星铁，锻造一件能扛住宫主第一击的护具。

本簇产出的碎片：

- 第一段残影里的那个人就是宫主（观星残影确认——但已散）
- 最深处连残影都不肯进
- 清玄不进——他的叙事边界到此为止

### B 簇：宫主战线

作用：

- 凡人篇最终Boss战
- 不是英雄屠魔——是让一个站了太久的人停下来

推荐任务序列：

1. `talk` — 进入最深处。这里不像残宫的其余部分。它很安静——墙壁上没有裂痕，地面没有星铁碎片。一口旧井封在正中央，井口被某种已经黯淡到几乎看不见的东西覆着。一个人站在井边，背对着你。和你在第五段残影里看到的一模一样。他转过身——不是战斗姿态，更像是被惊动了。他不说话。

2. `kill` — 击败天穹宫主。战斗节奏建议慢而沉重。他的每一击都压着极强的力量但速度不快——他的动作里有某种重复感，像是他已经在脑子里做了无数次。当他的血量降到一半时，井口的旧封开始出现裂纹。当他被击倒时，他没有倒下——他只是慢慢坐下，靠着井沿。然后他说了一句话。很短。可能只有半句。说完之后他不再动了。不是死了——是停下来了。

3. `talk` — 封天井的旧封在宫主停下来后自行消散。你站在井边。井口往下看不到底——但不是黑暗，是一种很深的、不动的东西。清玄从你背后的通道走上来。他看着宫主和井，沉默了很久。然后他说："井那边有人。他们会告诉你前面的路。我送到这里。"

本簇产出的碎片：

- 宫主的战斗方式：缓慢、沉重、重复——像一个执行旧指令的人
- 他的那句半句话——只有半句，不完整，不解释。建议方向："井后面……不是……" 或 "你来了……那他可以……"——玩家可以有各种解读，没有标准答案
- 清玄说"我送到这里"——凡人篇引路人职能正式结束
- 封天井开启——不需要钥匙，不需要仪式。宫主不再守了，它就开了

### C 簇：镇中回响线

作用：

- 在踏入封天井之前，回云来镇做最后一次交付
- 同一件事（你要走了）在不同NPC嘴里是完全不同的回声
- 这是全卷碎片化叙事在终章的最后一次体现

推荐任务序列：

1. `submit_item` — 把该交的东西交了：宫主的遗物（如果玩家捡了什么）、最后一趟路上收回的物件、不需要带进封天井的东西。交付对象是清玄和苏简的旧账（苏简派来的人会在云来镇等）。

2. `talk` — 和五个常驻NPC对话，最后一次。每人一句。不做全卷总结——每人只关心自己关心的那个侧面。没有人在这个时候说出一个"全卷意义"。

本簇产出的碎片：

- 沈七娘："炕给你留着。万一井那边不收你，这边还有个能缩的地方。"
- 兰药师："那个伤者收口了。他没记住你叫什么——但他记得你是把他从断道上拖回来的人。"
- 马会头："外面现在没那么乱了。不过你走了以后，门还是得有人看。"
- 老关卒："北井那边最近夜里没动静。你也别去看了。"
- 赵老六："你刚来的时候我还嫌你碍事。现在你是要进封天井的人了。"

这五个回声：
- 沈七娘关心你能不能回来
- 兰药师关心伤者记不记得你
- 马会头关心门还得有人看
- 老关卒关心那口井
- 赵老六看到了你从"碍事的人"到"要进封天井的人"的变化
- 五句话不形成一个统一的"全卷总结"——它们各说各的

### D 簇：踏井线

作用：

- 玩家踏入封天井，凡人篇正式结束
- 不煽情、不做CG、不给总结陈词
- 封天井是物理通道，走进去就进入练气期

推荐任务序列：

1. `talk` — 回到天穹残宫最深处。清玄站在井边等你。他最后说了一句："井那边有人。他们会告诉你前面的路。我送到这里。"然后他往后退了一步——不挥手、不目送、不煽情。他只是退了一步，让你自己走到井边。

2. `realm_stage` — 在踏入封天井前，确认你已经达到了能够承受井后压力的境界（半步筑基，`realmLv 30`）。这不是系统门槛——这是叙事上的确认：你的身体和气海已经足够稳定了。

3. 踏入封天井。井下不是坠落——是穿过。穿过一段很长的、什么都看不见的过渡。然后另一端的光透进来。你从井口出来时，面前是一个新的地方。这里的空气和云来镇不一样——更沉，但更稳。远处有烟火和棚铺。

凡人篇结束。

本簇产出的碎片：

- 清玄最后一次说"我送到这里"——和练气期开头形成闭环
- 封天井的穿越：不是坠落、不是传送阵、不是加载画面——是一段沉默的穿过
- 井的另一端：栖真渡的方向。一个新的修士据点

## 6. 本章结束状态

凡人篇全卷结束时：

- 天穹宫主停下来了——没有人知道他为什么守、在守什么、为什么停了
- 封天井开了——没有人完整知道井后是什么、为什么被封
- 云来镇暂时稳住了——但能稳多久没人能保证
- 五个人给了五种"你要走了"的回声
- 清玄说完了他的最后一句"我送到这里"
- 玩家踏入了封天井——不再是外乡人，但也不是英雄

全卷的叙事收束不在"真相大白"，在：
- 你从一个被筛进镇的外乡人
- 变成了一个有能力走进封天井的人
- 你手里有碎片、有回声、有伤口和活口
- 但没有标准答案

## 7. 本章不该做的事

- 不让宫主做长篇独白揭秘
- 不让宫主的那句半句话变成可解读的线索（就让它是一句听不懂的残句）
- 不让封天井被任何人完整解释
- 不让清玄在做最后告别时说煽情的话
- 不让五个NPC的回声形成统一的"意义总结"
- 不把踏入封天井做成CG或大场面
- 不解释封天井的另一端具体是什么——留给练气期开局的第一眼

## 8. 文案口径参考

适合终章的NPC说话口径：

- "我送到这里。"（清玄最后一句）
- "炕给你留着。万一井那边不收你，这边还有个能缩的地方。"（沈七娘）
- "他没记住你叫什么——但他记得你是把他从断道上拖回来的人。"（兰药师）
- "你刚来的时候我还嫌你碍事。现在你是要进封天井的人了。"（赵老六）

宫主唯一的台词建议（半句）：

- "井后面……不是……"（断裂）
- 或："你来了……那他可以……"（指向不明）
- 或：说一个没人听过的地名（"栖真"两个字可以说出来——但只说出这两个字，不做任何解释）

不适合终章的写法：

- "原来你守在这里是为了封印道劫的入口！"
- "谢谢你救了云来镇！你是我们的英雄！"
- "真相终于大白了……"
- "再见！一路顺风！"
- 宫主说："让我告诉你一切的真相……"

## 9. 凡人篇全卷叙事闭环

```
序章·入镇   —— 断道、尸体、怪伤、镇门、脚店、北井怪话
第一章·立足  —— 粮路、药路、三种旧北关说法、境界
第二章·追索  —— 青竹林、翻车、竹隐客指方向
第三章·对照  —— 矿洞/遗迹碎片咬合、三方旧留者各说各的
第四章·叩门  —— 兽谷漏口、灵脊岭窄门、行令是承担
第五章·见天  —— 天穹残宫、五段残影、天封核心、宫主与封天井初现
终章·封天    —— 击败宫主、封天井开启、镇中五种回声、踏入井中
            ↓
     练气期开局 —— 从封天井另一端出来，栖真渡在远处
```

卷名仍可沿用 `外乡人`。卷末称呼转为 `少侠`。

全卷核心体验：不是"越来越接近真相"，而是"接触越来越多互相咬合但永远缺口很大的碎片，最后发现前面还有更大的缺口"。
