## 说明

- 重构版章节文档（chapter-00 至 chapter-06）是当前剧情真源文档
- 凡人篇以终章"封天"收束：击败宫主→封天井开启→镇中回响→踏入封天井，无缝过渡到练气期
- 每章文档包含：章节目标、章节摘要、氛围关键词、主要角色（含叙事约束）、任务簇详情（含具体任务序列）、结束状态、该章不该做的事、文案口径参考
- 所有章节统一采用碎片化叙事口径：不设总答案、NPC 各说各的、碎片能咬合但不闭合、章节收束不做总结只推门槛
- 每章末尾的任务簇标注了推荐任务数量和目标类型（talk/kill/kill_collect/submit_item/learn_technique/realm_stage），后续 quest JSON 重写时直接以此为准
