# 《寻仙录》第一卷·外乡人 序章 入镇

状态：重构初稿

适用阶段：

- 玩家称呼：`外乡人`
- 建议等级：`realmLv 1-2`
- 主要地图：镇外断道 → `云来镇`

说明：

- 这是整个前期剧情的开场章
- 本章不负责解释世界，只负责把玩家送进世界
- 不做英雄登场，不做天命揭示

## 1. 本章目标

- 让玩家明确自己不是"天命之子"，只是来到此地讨生活的外乡人之一
- 让玩家尽快理解世道底色：粮紧、路险、命轻、夜里更坏
- 让玩家进入 `云来镇`，拿到在本地立足的第一个身份锚点
- 埋下两条互不印证的暗线：怪伤口与北井旧坊

本章只做三件事：

- 活下来
- 进得了镇
- 看见一点不对劲（但没人肯解释）

## 2. 章节摘要

入夜前，一批外乡人沿着山道赶往云来镇。路被塌方和弃车堵死，前方有人死，后方有人追，天黑以后林子里还会出事。玩家和其他外乡人一起清路、搬货、护送伤者，勉强赶在宵禁前抵达镇门。

镇门不会白开。守门兵和里正不在乎谁受了多少苦，他们只在乎两件事：会不会惹事，值不值得放进来。玩家需要证明自己不是废人，也不是麻烦。等真正进镇后，玩家很快发现，镇里并不比镇外安全多少，只是死法更讲规矩。

## 3. 氛围关键词

- 荒年
- 断路
- 宵禁
- 人多粮少
- 好人不一定有好下场
- 规矩不是为了公道，是为了让地方还能转下去

## 4. 主要角色

### 4.1 赵老六

- 身份：赶路的老车把式
- 作用：引导玩家进入第一个事件现场
- 人设：嘴碎、世故、胆小，但真见血时不跑
- 核心台词方向：他知道这个世道没道理可讲，所以张口闭口都是"先活下来"
- 叙事约束：赵老六只说自己见过的事，不负责解释任何异常的原因

### 4.2 沈七娘

- 身份：云来镇南市脚店掌柜
- 作用：玩家入镇后的第一个稳定落脚点
- 人设：精明、冷硬、愿意做生意，不愿平白发善心。她的善意全是交易
- 核心台词方向：她会收人，但只收能干活的人
- 叙事约束：沈七娘对镇上的怪事心知肚明但从不正面说，只劝人"别多问"

### 4.3 马会头

- 身份：镇门值守头目
- 作用：卡住"进镇资格"，逼玩家完成本章闭环
- 人设：不是纯恶人，但把外乡人当消耗品。嘴上刻薄，实际上维持的是镇门还能转下去的规矩
- 核心台词方向：镇门不是善堂，能进去的人，要么有钱，要么有用
- 叙事约束：马会头对北井旧坊只字不提，问到他脸上也只会说"那不是你该去的地方"

### 4.4 无名伤者

- 身份：在断道边捡到的重伤汉子
- 作用：作为本章第一处异常线索
- 人设：大半程昏迷，只在清醒瞬间说出一两句让人记不牢的话
- 线索作用：其伤口不像常见刀伤，也不像兽咬
- 叙事约束：伤者不知道自己怎么伤的，说不清，也不该说清

### 4.5 老关卒

- 身份：守着北关残墙的老人
- 作用：提供北井旧坊的第一条回避式口风
- 人设：话少、不解释、不让路
- 叙事约束：老关卒只说"那边早就封了"，不说为什么封、谁封的、里面还有什么

## 5. 任务簇

### A 簇：入镇挂名线

作用：

- 先把玩家放进镇内秩序
- 让"落脚资格"成为第一层门槛

推荐任务序列：

1. `talk` — 断道口被堵，赵老六让你帮大家清路。看清倒下的板车、散落的粮袋和两具还没凉透的尸体。其中一具尸体腰牌还在，另一具胸口裂伤发黑，不像寻常兵刃。（教学交互：搬木板、拖障碍、捡干粮）

2. `talk` — 到镇门口，马会头不肯白开门。他先嫌这批外乡人晦气，又嫌伤者要死在门里。最终给出条件：能干活的进，拖后腿的留外头。玩家不是靠主角光环进镇，而是因为在路上做了事、有人证。

3. `submit_item` — 去沈记脚店找沈七娘。她不要可怜人，只要能在后厨扛货跑腿的人。交出路上的凭物（捡回的腰牌、干粮袋或其他证明你有用的东西），拿到第一晚的炕位。

本簇产出的碎片：

- 镇子如何筛人：不是所有人都能进门
- 有些人能进，有些人被留在门外
- 沈七娘的善意是交易，不是施舍

### B 簇：门边杂务线

作用：

- 让玩家先在现实问题里动起来
- 不把开场做成纯对话

推荐任务序列：

1. `kill` — 马会头让你去南门外清灰尾鼠。粮袋边、垃圾堆旁都是，再拖下去流民和守兵今晚都别想睡安稳。这活不难，但脏，正好用来筛掉不肯动手的人。

2. `kill_collect` — 韩铁匠让你去收拾趁夜摸到城边的流匪，把他们的腰牌和还能用的铁件带回来。他不关心你杀没杀够，只关心带回来的东西够不够敲一口锅、打一把刀。

3. `submit_item` — 把收回的腰牌和铁件交给韩铁匠或马会头。各人反应不同：韩铁匠只看铁能不能用，马会头只看死人够不够数。没人夸你，只是在账上记一笔。

本簇产出的碎片：

- 镇子外头并不稳
- 很多规矩都和"今晚能不能撑过去"有关
- 铁匠只认铁，马会头只认数，没人负责关心你

### C 簇：夜里见闻线

作用：

- 留下第一批真正有记忆点的异常碎片
- 让玩家感觉"有问题"但没有任何人给出解释

推荐任务序列：

1. `talk` — 脚店里那名无名伤者快撑不住了。沈七娘让你去照看他，说既然人是你们从断道上捡回来的，就得有人看着。夜里伤者短暂清醒，只说一句断断续续的话，又昏过去。

2. `talk` — 把听见的话转述给沈七娘。她脸色不好，但只让你别乱传。再去找老关卒问北井旧坊的事，老关卒只说"那边早就封了，别过去"，不肯多讲一句。

3. `learn_technique` — 清玄执事听说了你在脚店的事，不追问、不解释，只给你一本最基础的护身功法。他的逻辑很简单：你若连这点东西都学不会，后面的事跟你没关系。

本簇产出的碎片：

- 怪异伤口（不像刀伤，不像兽咬）
- 伤者的呓语（怪，但记不牢）
- 北井旧坊被人刻意回避（不止一个人，而是所有知道的人都不肯说）
- 大家都知道有问题，但没人把话说满

## 6. 本章结束状态

玩家完成本章后，应拿到以下叙事成果：

- 不再只是路过的人，而是在云来镇暂时站住脚的外乡人
- 认识赵老六、沈七娘、马会头、老关卒，知道每个人都有自己的规矩和禁区
- 知道镇门、脚店、南市是自己接下来活动的起点
- 见过一次说不清的异常伤口和一句怪话，但没有任何人能解释
- 被清玄给了一本功法，但不明白他为什么给

对应的情绪收束应该是：

- 不是"我赢了"
- 而是"今晚总算不用死在镇外"
- 同时也是"这地方的问题显然不止是匪患和粮荒"

## 7. 本章不该做的事

- 不解释伤口来源
- 不把北井写成明确副本入口
- 不让任何人告诉玩家"你卷进大事了"
- 不让清玄在本章说出任何关于旧阵、道劫、崩坏的字眼
- 不让任何 NPC 说出"你是特别的人"

## 8. 文案口径参考

适合本章的 NPC 说话口径：

- "外乡人，想活命就别在镇外磨蹭。"
- "这年头，肯搭把手的人不多，肯搭完手还不讨价的，更少。"
- "镇门不是庙门，磕头没用，拿出点用处来。"
- "你先别问那人怎么伤的。问多了，未必活得久。"
- "那边早就封了。别过去。"

不适合本章的写法：

- "你是预言中的人。"
- "这一切只能由你阻止。"
- "你注定要改变天下。"
- "这是道劫的余波。"
- "旧阵已经崩坏了。"

## 9. 这一章的作用边界

这一章只负责把玩家送进世界，不负责解释世界。

它要像一脚踩进泥里：

- 鞋先脏
- 人先冷
- 肚子先饿
- 然后玩家才会开始考虑，自己到底要在这个地方活成什么样
