# 《寻仙录》第一卷·外乡人 第五章 见天

状态：重构初稿

适用阶段：

- 玩家称呼：`外乡人`（本章结束时转换到 `少侠`）
- 建议等级：`realmLv 19-25`
- 主要地图：`sky_ruins`

说明：

- 这是凡人篇主线的最高处
- 前几章在地面和地下摸索残痕，这一章开始碰到更高层留下的遗存
- 天封核心不是万能答案，只是一件抢回来的关键遗存
- 残影见闻是本章的核心叙事机制：玩家看见旧景的一闪，但每次都只有一瞬
- 本章末尾引出两个终章关键：**宫主**（残宫最深处仍未被惊动的存在）与**封天井**（封柱崩断时被震封的旧通道）

## 1. 本章目标

- 把视野从地表和遗迹抬到更高处
- 让玩家看见"劫难留下的不只是地上的坏处，还有更高处掉下来的残余"
- 通过残影见闻让玩家看见旧景的片段——每次都只是一闪，不做解释
- 夺回天封核心——不是救世终战，而是一次勉强抢回关键遗存的行动
- 让玩家第一次听到"宫主"和"封天井"这两个名字——但只听到，不看到

本章不做：

- 不让观星残影把天穹残宫的来龙去脉说完
- 不把天封核心写成万能答案或万能修复件
- 不做"最终真相揭晓"
- 不让玩家在本章见到宫主（只通过残影和口风感知其存在）
- 不让观星残影解释封天井通往哪里

## 2. 章节摘要

练气之后，清玄终于肯让玩家去碰天穹残宫。那地方比矿洞、遗迹和兽谷都更像真相的正面——但也是最容易把人直接碾死的一段路。

观星残影不是人，是封柱崩断前留在宫里的最后一抹残景。它不会回答所有问题——它只会让人看见它还记得的几个片段。玩家先夺回星陨金——一场封柱崩断前下过的"星铁雨"留下的实物，让凡人的手上也能拿一件能触碰天门残物的东西。

玩家深入残宫，途中多次触发残影——旧景一闪而过。击败噬星兽，取回天封核心。但观星残影在交核心时说了一句让玩家记在心里的话：残宫最深处还有一个人——它管他叫"宫主"——从崩断那天起就再没出来过。宫主守着的地方有一口被封住的旧井，观星残影说那井叫"封天井"，但它不知道井后是什么——它只记得封柱崩断前，有人从井那边过来，也有人往井那边去。然后它就不在了。

## 3. 氛围关键词

- 天穹残宫
- 星铁雨
- 星陨金
- 封柱残片
- 残影一闪
- 天封核心
- 宫主还在里面
- 封天井
- 抢回一块能续命的东西

## 4. 主要角色

### 4.1 观星残影

- 身份：封柱崩断前留在宫里的最后一抹残景
- 作用：触发残影见闻、引导回收星陨金和天封核心
- 人设：它不像活人那样思考和回答。它只能"记起"——每次触发都是某个特定条件的回应，而不是对话
- 叙事约束（红线）：观星残影不能让玩家看到完整旧景。每次只是一闪：一只手在补柱、一个人往下跑、半句话被截断。玩家不知道是谁、为什么、后来怎样
- 残影内容建议：
  - 第一段残影：封柱开裂前，有人在柱前做了什么（只看到背影和手，看不到脸）
  - 第二段残影：星铁从高处坠落时，有人在地面往上喊了半句（听不清喊的什么）
  - 第三段残影：噬星兽第一次出现时，不是从外面进来的（似乎是从柱心裂出来的）
  - 第四段残影（仅声音）：风声、金属断裂声、闷响
  - **第五段残影**（摸到核心时触发）：一个人影站在一口井边，背对玩家。不是战斗姿态——更像是在等。残影一闪，人影和井都消失了。不要描述脸、衣着、动作细节。只需让玩家看见"有一个人站在井边，背对着你"
- 观星残影在第五段残影后说出"宫主"和"封天井"：它说那个人影就是宫主，那口井就是封天井。但它只能说出名字——"从崩断那天起他就没出来过。他在守什么、还是在困什么，我不知道。我已经不在了。"
- 观星残影只负责让你看到这些碎片。它不负责讲清来龙去脉

### 4.2 噬星兽

- 身份：守着天封核心的最深层怪物
- 作用：本章的最终战斗目标
- 设定：不是普通妖兽，更像是被封柱崩坏时从核心里裂出来的东西
- 叙事约束：击败它不给剧情对话，不给怪物设定说明。玩家只知道它守着核心

### 4.3 天穹宫主（本章不出现，仅被提及）

- 身份：天穹残宫最深处仍未离开的人——或者说曾经是人
- 作用：终章的最终对手，本章只埋下他的存在
- 观星残影对他的描述：从封柱崩断那天起就再没出来过。它不叫他"王"或"主"，只叫"宫主"——像是在说一个职位，而不是一个身份
- 叙事约束（红线）：本章不让玩家见到宫主。观星残影对宫主的描述必须含混：它不知道宫主在守什么、为什么不出来、现在还是不是活人。只说"从崩断那天起就没出来过"
- 碎片价值：观星残影提到宫主守着一口井——封天井。但"封天井"这三个字是观星残影唯一能给出的信息。井后是什么、井为什么被封、宫主在守还是在困——全都不知道

### 4.4 清玄执事

- 身份：信息控制者
- 作用：本章开篇给许可，本章末尾收核心。听完"宫主"和"封天井"后给出终章方向
- 叙事约束：清玄拿到天封核心后先说"能续一阵子了"。然后听到宫主和封天井的事，沉默了很久，只说："那是你最后一段路。去不去，你自己定。"——他不解释宫主是谁，不解释封天井是什么

## 5. 任务簇

### A 簇：星铁回收线

作用：

- 把"高处留下的后果"落成玩家能接触的实物
- 建立"天上掉下来的东西不是奇宝而是灾后遗存"的认知

推荐任务序列：

1. `talk` — 进入天穹残宫，找到观星残影。它不像活人那样看你——更像一面旧镜在辨认来访者身上有没有相似的残痕。它触发第一段残影：封柱开裂前，一个人站在柱前，手按在柱面上。你看不清脸。残影一闪就没了。

2. `kill_collect` — 观星残影让你先去收星陨金。天宫猎者——被残宫环境扭曲的旧守卫者——守着那些散落的星铁残片。击败它们，收回足够的星陨金。每收一片，你都能感觉到这片金属比寻常铁器沉重得多，表面有烧灼后急速冷却的纹路。

3. `submit_item` — 把星陨金交给观星残影。它触发了第二段残影：星铁从高处坠落时，有人在地面往上喊了半句。你听到的声音像被撕开过——"……快走……柱……"——然后什么都没了。观星残影只说："这些铁是你现在能拿得动的东西。"

本簇产出的碎片：

- 第一段残影：补柱的人（背影+手，无面，无来由）
- 星陨金（实物，烧灼急冷纹路）
- 第二段残影：坠落和喊叫（半句话，被截断，不知对谁喊）
- 观星残影只说这些铁"你现在拿得动"，暗示有更重的东西

### B 簇：残影见闻线

作用：

- 让玩家看见更高层旧景，但每次只是一闪而过
- 这是本章最具辨识度的叙事机制

推荐任务序列：

1. `talk` — 观星残影让你去天穹残宫的外环清理天宫傀壳——那些被封柱崩坏震醒的旧壳。路上触发第三段残影：噬星兽第一次出现，不是从外面进来——它似乎是从柱心被裂出来的。你一眨眼，残影就散了。

2. `submit_item` — 清掉傀壳后，收一些残宫碎片回来。观星残影看到碎片，触发了第四段残影——但这一次它没有让你看到完整画面，只有声音：风声、金属断裂声、然后是一声很低的闷响，像是什么巨大的东西塌进了地里。没有画面。观星残影沉默了一下，只说："封柱崩断之前，天穹残宫先下过一场星铁雨。那是崩前最后一件事。崩本身我没看到——我已经不在了。"

3. `talk` — 你问它"不在了"是什么意思。它没有回答。它只说核心还在最深处，被噬星兽守着。"你要去就拿。但别问那东西的来历。来历我说不清。"

本簇产出的碎片：

- 第三段残影：噬星兽从柱心裂出（不是外来）
- 第四段残影（仅声音）：风声、金属断裂声、闷响（无画面，更抽象）
- 观星残影承认自己没看到崩本身——"我已经不在了"
- 残影越来越短、越模糊，最终只是声音
- 核心的来历——说"说不清"，不说"不能说"

### C 簇：核心夺回线

作用：

- 作为凡人篇最高强度的行动收束
- 玩家不是解决劫难，只是抢回一件还能续用的关键遗存

推荐任务序列：

1. `kill` — 最深处，噬星兽守着核心区。它不是普通的战斗——它的攻击方式带着和封柱崩坏相似的特征：撞击、撕裂、把空间撕开一瞬间的裂缝。击败它。

2. `kill_collect` — 取回天封核心。当你的手碰到核心的一瞬间，最后一段残影触发——但也只有一瞬：一只手（和第一段残影里的那只手可能是同一只）把核心按进柱心，然后一切消失了。

3. `submit_item` — 把天封核心带回给清玄。他看了很久，说："能续一阵子了。"——就这一句。但当你转述观星残影提到的"宫主"和"封天井"时，清玄沉默了更长的时间。最后他说："那是你最后一段路。去不去，你自己定。"——他不解释宫主是谁，不解释封天井是什么。

本簇产出的碎片：

- 噬星兽的攻击方式暗示它和封柱崩坏直接相关（但不解释）
- 最后一段残影：核心被按进柱心的瞬间（和第一段残影呼应，但仍不完整）
- 清玄先说"能续一阵子了"，再听到宫主和封天井后只说"那是你最后一段路"
- 宫主——一个从崩断那天起就没有出来过的人
- 封天井——宫主守着一口被封住的旧井，观星残影不知道井后是什么
- 清玄不给答案也不阻拦——他只确认那是"最后一段路"

## 6. 本章结束状态

- 天封核心交出去了，云来镇暂时从立刻暴毙里退了半步
- 玩家看见过五段残影，但没有一段能拼成完整旧景
- 观星残影说不清，清玄不说全
- 玩家第一次听到"宫主"和"封天井"——残宫最深处还有一个人，守着一口被封住的井
- 清玄说"那是你最后一段路"——不解释、不阻拦、也不鼓励
- 终章的方向已经确立：回去面对宫主，打开封天井
- 但宫主是谁、封天井后面是什么、为什么封柱崩断后他再没出来——全都不明

## 7. 本章不该做的事

- 不让观星残影把天穹残宫的来龙去脉说完
- 不把天封核心写成万能答案或万能修复件
- 不让任何一段残影呈现完整画面
- 不做"原来一切都是因为封柱崩断"式总结
- 不让清玄在拿到核心后说出核心的全部作用
- 不让玩家在本章见到宫主或封天井（只听不看不碰）
- 不让观星残影解释封天井通往哪里、宫主在守什么

## 9. 残影见闻的叙事设计

残影不是过场动画。它们的作用是：

- 让玩家"看到过"但不"看明白过"
- 提供不能互相印证的一闪画面——它们指向同一件事但彼此不闭合
- 越往后越模糊（画面 → 半画面 → 只有声音 → 画面重现但更暗），暗示残影本身也在衰减
- 最后一段残影（摸到核心时）出现宫主背影和封天井——但只看到"有人在井边背对着你"，看不到脸、不知道在做什么
- 第五段残影之后观星残影才说出"宫主"和"封天井"两个名字——但它只能说出名字，说不出来历
