# 云来镇详细格局稿

状态：草案 v1

配套文档：

- [凡人篇地图重设计](./fanren-map-redesign.md)

适用目标：

- 作为 `spawn` 地图的重做蓝本
- 用于后续重画 `packages/server/data/maps/spawn.json`
- 优先兼容当前玩法系统，不预设复杂脚本

## 1. 设计目标

这份稿子不再讨论“云来镇应不应该像个边镇”，而是直接回答“云来镇要怎么画”。

必须同时满足的四个目标：

- 它要像主城据点，不像测试地图
- 它要能承接凡人篇前半主线，不只是功能集散地
- 它要兼容当前 1Hz 格子移动和现有任务系统
- 它要给后续店铺系统、容器系统、更多 NPC 和支线预留空间

## 2. 关键约束

先把不能忘的现实约束写死：

- 当前地图推荐控制在 `64x64` 左右，避免跑图成本过大
- 当前 `spawn` 图仍需承担：
  - 出生点
  - 主线前两段交接
  - 基础功能 NPC 集中
  - 通往荒野和青竹林的外线入口
- 当前早期怪仍需就近布置，不能让新号一进游戏就跑整张图
- 主线仍不应建立在护送、守城、复杂开门、复杂机关联动之上
- 当前 `spawn` 不适合完全做成纯安全图，必须保留“城外缓冲战斗区”
- 可正式依赖以下正在实现的新系统：
  - 地面物品
  - 容器
  - 时间
  - 光照

因此，新的 `云来镇` 不应只是一座城内地图，而应是一张“城内 + 城门外缓冲带”复合地图。

## 3. 总体尺寸与结构

建议最终尺寸：

- `width = 64`
- `height = 64`

地图结构拆成三层：

1. 最外层：地图边界与自然地形边框
2. 中间层：镇外缓冲带
3. 内层：真正的云来镇城内区域

推荐空间比例：

- 城内核心区约占整图 `45%`
- 城外缓冲区约占整图 `35%`
- 墙体、道路、空地和转场区约占整图 `20%`

这样可以同时解决两个问题：

- 镇子终于有“城”的体量
- 早期怪和新手练级点仍能放在近门区域，不需要额外拆新图

## 4. 推荐总平面

## 4.1 城墙包络

建议把主城墙做成近矩形，但不要太工整。

推荐范围：

- 内城墙西界：`x = 10`
- 内城墙东界：`x = 53`
- 内城墙北界：`y = 10`
- 内城墙南界：`y = 47`

视觉处理建议：

- 南墙和东墙较完整，体现仍在使用
- 北墙旧、厚、偏冷，体现旧关道方向的年代感
- 西墙局部可见补修痕迹，呼应棚户和外来人口压力

这样城内净空间约为：

- `44 x 38`

这个体量已经足够做出 6 个功能区，不会再像几栋房子堆在一起。

## 4.2 城外缓冲带

城外不是纯装饰，而是玩法承载区。

推荐划分：

- 南门外：新手缓冲战斗区
- 东门外：运材路和青竹林前置道路
- 东南角：镇外暴徒、夜行刀客活动区
- 西南角：低危垃圾堆、鼠患、临时棚营
- 北侧城外：封闭旧关道，不开放常规战斗

这样处理后，玩家前期在 `spawn` 一张图内就能完成：

- 进镇
- 接主线
- 出门练手
- 回镇交付

不会频繁切图，也不会把怪刷进城里。

## 5. 主路线骨架

## 5.1 三条主路

云来镇至少要有三条一眼能懂的主路线。

### A. 南北主街

作用：

- 南门进镇后的主轴
- 连接南市、仓场、旧坊

推荐走向：

- 从南门门洞正中一路贯穿到北井旧坊外缘

宽度建议：

- 主干路 `3` 格

意义：

- 新玩家第一次进镇就能理解“这条是镇子的中轴”
- 后续主线也最容易围绕它组织

### B. 东西横街

作用：

- 连接西侧棚户、仓场、东门工匠区

推荐走向：

- 从西侧棚户区穿过镇中仓场，再连到东门工匠区和东门

宽度建议：

- 主干路 `3` 格

意义：

- 形成第二条认知轴线
- 让玩家知道东路是工事和外路，不是民居巷子

### C. 环墙巡路

作用：

- 连接门楼、仓场、岗哨和夜巡逻辑

推荐走向：

- 沿内墙留 `1` 到 `2` 格缓冲通道

意义：

- 即使暂时没有巡逻系统，空间上也会让防御结构成立
- 方便以后加巡夜 NPC、封锁门路、夜禁气氛

## 5.2 路口原则

关键路口至少做三个：

- 南门内空场
- 仓场前大路口
- 北井旧坊前岔口

作用分别是：

- 让进城有缓冲
- 让中段任务交接自然集中
- 让异常区与日常区出现明确心理分界

## 6. 六大功能区详细拆分

## 6.1 南门外手区

推荐范围：

- `x = 24-39`
- `y = 44-53`

定位：

- 外乡人第一次看见的镇内空间
- 从城门进入后的第一缓冲区

必须包含的元素：

- 门楼内空场
- 守兵棚
- 公示木牌
- 简易拴马桩
- 外来人登记桌

建议建筑数：

- 小型建筑 `2-3` 栋
- 开放棚架 `2` 组

应放 NPC：

- 马会头或守门头目
- 负责初期主线衔接的差役
- 一个只讲“进镇规矩”的门兵

不建议放：

- 主商业功能
- 大量长期支线

因为这里的本质不是生活区，而是门禁区。

地面物品建议：

- 被丢弃的绳索
- 碎木箱板
- 临时登记木牌

容器建议：

- 门兵补给箱 `1`
- 罚没杂物箱 `1`

昼夜与光照建议：

- 夜里门楼火把必须最稳定
- 门内空场亮度应高于周边巷道
- 这里是夜间回镇时的第一安全感来源

## 6.2 南市脚行区

推荐范围：

- `x = 15-31`
- `y = 34-45`

定位：

- 玩家最常返回的据点区
- 凡人篇的人间层主舞台

建议建筑清单：

- 脚店主楼 `1`
- 后院通铺 `1`
- 小食铺 `1`
- 杂货棚 `1`
- 药铺 `1`
- 零散临时摊位 `2-4`

建议空间感：

- 比其他区域更密、更亮、更吵
- 路不宜完全笔直，可略有挤压感

应放 NPC：

- 沈七娘
- 兰药师
- 素问医女
- 跑腿差使
- 闲话 NPC

建议这里成为：

- 玩家日常补给概念锚点
- 镇内支线最密集的区域
- 主线前半“回镇汇报”的首选落点

地面物品建议：

- 水桶
- 菜篓
- 晾晒药材
- 零散麻袋

容器建议：

- 脚店后院杂箱 `1`
- 药铺药柜 `1-2`
- 杂货棚木箱 `1`

昼夜与光照建议：

- 这是全镇夜间最亮区域
- 脚店门口、药铺门口和摊位区要形成连续暖光带
- 即使夜里，这里也应让玩家觉得“还有人活着”

## 6.3 镇中仓场区

推荐范围：

- `x = 26-42`
- `y = 24-36`

定位：

- 云来镇最核心的功能区
- 所有外线都应该在空间上汇聚到这里

建议建筑清单：

- 大粮仓 `2`
- 木料堆场 `1`
- 货栈 `1`
- 运车停放区 `1`
- 巡检亭 `1`

建议开放空间：

- 仓场中央必须有一块较大的开阔空地

理由：

- 这里会是视觉上的“镇中心”
- 以后无论加公告板、章节地标还是事件提示，都有地方放

应放 NPC：

- 仓头
- 车把式
- 主线中期差役
- 一名能交代“路为什么重要”的老工头

当前过渡期可放：

- 清玄执事

但长期看，清玄不应站在出生点中央石台，而应有独立办公点位，例如仓场旁的公事房或巡检所。

## 6.4 东门工匠区

推荐范围：

- `x = 41-52`
- `y = 24-40`

定位：

- 工事、铁匠、出东路准备区

建议建筑清单：

- 铁匠铺 `1`
- 炉棚 `1`
- 木作间 `1`
- 运材车棚 `1`
- 材料堆场 `1`

建议空间特征：

- 比南市更硬、更开阔
- 地面上能看到木料、矿料、炉灰、废铁等堆放痕迹

应放 NPC：

- 韩铁匠
- 工匠学徒
- 与青竹林 / 矿洞相关的引导 NPC

应放地标：

- 镇防铁砧
- 东路路牌
- 运材车列

这样玩家一靠近这里，就知道“东边出去是干活和历练，不是逛街”。

地面物品建议：

- 木料
- 炉灰
- 矿渣
- 半成品铁器

容器建议：

- 工坊材料箱 `2`
- 废料柜 `1`

昼夜与光照建议：

- 白天靠开阔和作业痕迹识别
- 夜里依靠炉火、炭火和少量工灯形成局部亮区
- 这里的夜间亮度应低于南市，高于棚户

## 6.5 北井旧坊区

推荐范围：

- `x = 24-40`
- `y = 12-23`

定位：

- 云来镇的异常区、沉默区、旧事区

建议建筑清单：

- 围井小院 `1`
- 旧祠 `1`
- 半废弃民居 `3-4`
- 巡夜岗亭 `1`
- 封存杂屋 `1`

空间要求：

- 路要变窄
- 房子要旧
- 空地要少
- 颜色和氛围要比南市冷

应放 NPC：

- 守夜人
- 老住户
- 文籍师叔或旧档案类 NPC

建议玩法定位：

- 前期只做地标、对话、少量任务触点
- 中后期再承接异常线

不要在凡人篇一开局就让这里变成高频任务区，否则气氛会被刷怪节奏冲散。

地面物品建议：

- 井绳
- 破陶片
- 倒掉的供碗
- 湿痕和灰烬

容器建议：

- 供物箱 `1`
- 封存木柜 `1`

昼夜与光照建议：

- 白天可见，但不显眼
- 夜里只保留极少数冷清灯点
- 必须出现大片半暗区，方便让玩家本能地不想久留

## 6.6 西侧棚户区

推荐范围：

- `x = 11-20`
- `y = 22-42`

定位：

- 边镇贫民区、短住区、杂工区

建议建筑清单：

- 棚屋 `5-7`
- 柴棚 `2`
- 破车点 `2`
- 废料堆 `2`

空间要求：

- 巷子窄
- 布局不规整
- 有点乱，但不能走成迷宫

应放内容：

- 低价值补给点
- 初期杂务任务
- 鼠患刷点的视觉来源

注意：

- 这里适合讲“镇里也分三六九等”
- 但不应喧宾夺主到压过南市

地面物品建议：

- 破布包
- 木柴堆
- 旧碗盆
- 破损车轮

容器建议：

- 废弃杂箱 `1`
- 破车暗格 `1`

昼夜与光照建议：

- 光照不均
- 夜里比南市暗很多，但仍要比镇外安全
- 适合作为“勉强算城里”的视觉边界

## 7. 城外缓冲区细化

## 7.1 南门外新手缓冲区

推荐范围：

- `x = 23-40`
- `y = 48-60`

定位：

- 最早期安全边缘战斗区

应放内容：

- 稀疏树丛
- 破栅栏
- 路边废车
- 低危鼠患点

建议刷怪：

- `m_town_rat`

作用：

- 玩家出门几步就能练手
- 交任务路径短

地面物品建议：

- 散落谷粒
- 破箩筐
- 鼠咬布袋

光照建议：

- 夜里仍能吃到门楼外溢的光
- 不应做成完全黑区

## 7.2 西南废棚与排水沟

推荐范围：

- `x = 8-22`
- `y = 48-61`

定位：

- 鼠患、杂物、贫民区外延

应放内容：

- 排水沟
- 倾倒垃圾
- 坍塌棚架
- 遗落货箱

建议刷怪：

- `m_town_rat`

建议用途：

- 作为韩铁匠前期鼠患线的主刷点

容器建议：

- 废棚杂箱 `1`

光照建议：

- 夜里只有零散弱光
- 应明显比南门外主路更脏、更不安全

## 7.3 东南破庙与断路地

推荐范围：

- `x = 42-58`
- `y = 46-61`

定位：

- 镇外暴徒与夜行刀客活动区

应放内容：

- 破庙
- 乱石地
- 废弃车辙
- 半塌路障

建议刷怪：

- `m_town_thug`
- `m_night_bandit`

建议用途：

- 承接镇外暴徒、腰牌类早期任务
- 用空间告诉玩家：东南不是正经出路，而是乱地

地面物品建议：

- 倒塌香炉
- 散乱骨头
- 破碎酒坛
- 丢弃刀鞘

容器建议：

- 破庙供桌暗箱 `1`
- 匪徒私藏箱 `1`

光照建议：

- 夜里主要依赖月光和极弱残灯
- 应成为 `spawn` 图里夜间最危险的战斗角落

## 7.4 东门外运材路

推荐范围：

- `x = 50-61`
- `y = 20-34`

定位：

- 通往青竹林的正式外路

应放内容：

- 木料堆
- 车辙
- 路牌
- 倒木或半废路障

这里不建议放城外新手怪。

原因：

- 它应优先承担“主线出东门去青竹林”的清晰导向
- 不能让玩家第一眼以为东门外只是另一个鼠患区

地面物品建议：

- 木屑
- 车辙痕
- 遗落绳索
- 倒伏小木栏

光照建议：

- 夜间用稀疏路灯和工坊余火保证方向可读
- 但不能亮到失去“出门后就进入野路”的感觉

## 8. 门、portal 与出生点

## 8.1 出生点

建议不再把出生点放在镇中央。

推荐位置：

- 南市脚店附近
- 或南门内空场偏西位置

建议坐标区间：

- `x = 24-29`
- `y = 40-44`

理由：

- 出生后先看到“有人、有店、有灯、有规矩”
- 不会一落地就站在行政中心或异常区
- 也不会离出门练手区太远

## 8.2 通往荒野的 portal

推荐位置：

- 南门外主路尽头

建议坐标区间：

- `x = 30-33`
- `y = 60-61`

这样玩家会自然理解：

- 南门走的是粮路和远路
- 真正离开云来镇，是往南出

## 8.3 通往青竹林的 portal

推荐位置：

- 东门外运材路末端

建议坐标区间：

- `x = 60-61`
- `y = 25-29`

这样玩家会自然理解：

- 东门出去是林路和工事路

## 8.4 通往灵脊岭的 portal

不建议继续裸放在镇中空地。

推荐位置：

- 北侧旧关道封口内侧

建议坐标区间：

- `x = 30-33`
- `y = 8-10`

设计要求：

- 默认视觉上是封闭旧道
- 由残碑、旧门、岗哨组成晚期开启区域

这样即使技术上已经有 portal，空间上也不会破坏凡人篇节奏。

## 11. Landmark 设计

云来镇至少应有以下 8 个稳定地标：

1. 南门门楼
2. 南市脚店
3. 镇中仓场
4. 镇防铁砧
5. 东门运材路牌
6. 北井围栏
7. 旧关残碑
8. 集议石台或告示台

地标原则：

- 每个地标都要能用一句话说清用途
- 玩家在聊天或攻略里能直接用地标称呼位置
- 地标不一定都强功能，但都要强辨识

## 12.1 容器与地面物品的总原则

云来镇接入容器和地面物品后，必须遵守以下规则：

- 地上的东西要像“刚有人在这里生活或干活”
- 容器里的东西要像“本来就该收在这里”
- 不要把容器做成遍地奖励箱
- 主城里的大部分容器应服务于可信度，而不是纯奖励
- 同一区域的地上物和容器内容要统一口径

示例：

- 南市药铺门前可以有晾晒药材，药柜里是草药和包布
- 东门工坊地上可以有矿渣和木料，材料箱里是铁料和工具
- 北井旧坊地上可以有供碗和湿痕，供物箱里是残香、符纸和旧杂物

## 12. 宝箱与固定拾取点预留

虽然现在还没有完整容器系统，但在格局稿里要先预留点位。

推荐首批预留位：

- 南市脚店后院杂箱 `1`
- 仓场封存箱 `2`
- 西棚户废车暗格 `1`
- 北井旧坊供物箱 `1`
- 东门工匠区废料柜 `1`

使用原则：

- 不全部做成高价值奖励
- 大部分应服务于探索感和任务感
- 镇内点位应符合“这东西为什么会在这里”

## 12.2 时间与光照的总原则

云来镇一旦接入时间和光照，主城体验应至少分成四档：

### 清晨

- 人少、冷、刚开门
- 南市开始有炊烟
- 仓场开始动起来

### 白日

- 可见度最高
- 镇内秩序最明确
- 最适合新玩家形成空间认知

### 黄昏

- 火把和灯笼开始接管空间
- 城门、南市、仓场的层次会更明显
- 适合做“该回镇了”的心理提示

### 夜间

- 门楼、南市、部分仓场维持亮区
- 棚户区和旧坊进入半暗区
- 东南乱地、北井旧坊成为高压感来源

这会让同一张 `spawn` 图在不同时间段有不同叙事重量：

- 白天是生活据点
- 夜里是勉强压住不安的边镇

## 13. 具体落图顺序建议

如果后续开始改 `spawn.json`，建议按以下顺序落图：

1. 先确定 `64x64` 外框和城墙范围
2. 先画三条主路
3. 再画南门、东门、北旧关
4. 再落六大功能区轮廓
5. 再放核心建筑
6. 再摆 NPC
7. 最后补城外缓冲怪区和 portal

不要反过来从摆怪和摆房子开始，否则很容易又画成一张功能测试图。

## 14. 最终体验目标

重做后的云来镇，玩家第一次进入时，应立刻得到这几个印象：

- 这地方有人活着
- 这地方活得不轻松
- 这里有规矩，而且规矩很硬
- 这里有吃饭睡觉做工的地方，不只是领任务的地方
- 这里的路通向外面的麻烦，也通向更远的命

如果这些印象立住了，凡人篇前半就有了真正的据点基础。
