# 《寻仙录》第一卷·外乡人 第三章 对照

状态：重构初稿

适用阶段：

- 玩家称呼：`外乡人`
- 建议等级：`realmLv 8-13`
- 主要地图：`black_iron_mine`、`ancient_ruins`

说明：

- 这是凡人篇里碎片第一次开始互相咬合的章节
- 但碎片之间的咬合感指向的是"它们是同一类事留下的"，而不是"这是一件事的完整经过"
- 矿洞和遗迹不是"前因后果"关系，而是同一场旧灾在两个不同地点的不同后果

## 1. 本章目标

- 把矿洞与遗迹并列起来，让玩家拿到足够多能互相对上的碎片
- 让三名旧留者（锻炉余火、守纹残念、碑灵学士）各说各的，彼此不完全一致
- 让玩家意识到"不同地方留下的是能互相印证的后果"，但仍拿不到完整解释
- 明确清玄在本章末不能做世界观定义

本章不做：

- 不把矿洞负责"前因"、遗迹负责"后果"写得过分工整
- 不让清玄在本章末尾下一个世界观定义
- 不让任何一个旧留者把话说完

## 2. 章节摘要

青竹林给了方向，真正的残片却埋在矿洞和断碑遗迹里。清玄先让玩家去玄铁矿洞见锻炉余火——一种不肯散的旧念头，守着一条早该断掉的矿路。玩家从矿洞活尸手里抢回失落信标核心后，矿路短暂地亮过一次，让玩家看见旧矿道里曾经有过的规模。

清玄只看了一眼信标就把玩家赶去断碑遗迹。他说"矿路告诉你的是一半，遗迹里的能对上另一半"。守纹残念是守门的壳，碑灵学士是记字的鬼，矿监残魂还以为自己仍在值夜。三个旧留者各提供了一段说法，三段能咬合出"旧阵崩坏以后往外漏余波"的大轮廓，但三段的细节彼此冲突。

清玄听完玩家带回的所有碎片后，第一次承认云来镇外的怪事不是偶然。但他不给定义，只说"先入先天"。

## 3. 氛围关键词

- 失落信标
- 遗迹钥石
- 旧矿路亮一次
- 死人比活人记性好
- 各说各的
- 碎片咬合但不闭合

## 4. 主要角色

### 4.1 锻炉余火

- 身份：旧矿炉前不肯灭的一缕残念
- 作用：提供矿洞侧的第一批碎片
- 人设：执着、顽固，只关心矿路还能不能亮。不在乎来的人是谁
- 叙事约束：锻炉余火只知道矿路的事。它不会说"这是旧阵崩坏造成的"，只会说"矿路不是自己死的，是有人让它死的"。它对遗迹的事完全不知道
- 提供的碎片：信标核心、矿路"不是自然死掉"的证词

### 4.2 矿监残魂

- 身份：死了还以为自己在值夜的旧矿监
- 作用：提供矿洞侧的第二批碎片（旧矿册）
- 人设：糊涂、重复、分不清现在和从前
- 叙事约束：矿监残魂掌握矿册上的名字——一批人从未"出矿"但也没被算死。它不解释为什么
- 提供的碎片：旧矿册、未出矿者名单

### 4.3 守纹残念

- 身份：断碑遗迹的守门意志残留
- 作用：提供遗迹侧的第一批碎片
- 人设：沉默寡言，只知道"门后还有东西"，但不肯说是什么
- 叙事约束：守纹残念承认自己只是守门的壳，真正的东西在门后
- 提供的碎片：遗迹钥石、关于"门后"的回避式描述

### 4.4 碑灵学士

- 身份：旧碑里残留的记字者意志
- 作用：提供遗迹侧的第二批碎片（旧碑文）
- 人设：啰嗦、挑剔、怕被人遗忘。对文字有执念
- 叙事约束：碑灵学士记得很多旧碑文，但碑文本身就是残的。它说的和矿监残魂说的有时对不上
- 提供的碎片：断纹石片、残碑文字

### 4.5 清玄执事

- 身份：信息控制者
- 作用：本章收束——承认怪事不是偶然，但不做定义
- 叙事约束（红线）：清玄不能在本章末尾说"这是旧阵崩坏以后一路往人间漏出来的余波"。他只能说"这些事不是偶然"和"你先入先天"

## 5. 任务簇

### A 簇：矿路残灯线

作用：

- 用矿洞的"还能不能重新亮一下"做视觉记忆点
- 让锻炉余火说出矿洞侧的第一层事实

推荐任务序列：

1. `talk` — 进入玄铁矿洞，找到锻炉余火。它看起来像一簇不肯灭的旧焰，声音从炉壁里透出来。它说矿路不是自己死的，是有人让它死的。你若想知道矿路和镇里到底断在什么地方，先把失落信标核心抢回来。

2. `kill_collect` — 深入矿道，击退盘踞在信标附近的矿洞活尸。这些活尸有些穿着旧矿工的衣裳，动作不像是纯粹的野兽。夺回失落信标核心。路上你会注意到矿壁上有大面积的焦痕——不是火烧的焦，更像是被什么东西烫过后又迅速冷却留下的。

3. `submit_item` — 把信标核心交给锻炉余火。炉火把它按进旧槽，矿路短暂地亮了一次——只有几息，但足够让你看见这条旧矿道曾经有多深、多宽，绝不是一个边镇小矿该有的规模。然后灯火灭了。锻炉余火只说了一句："你也看到了。"

本簇产出的碎片：

- 矿路不是自然死掉的
- 旧矿道规模远超边镇小矿（暗示曾服务于更大的体系）
- 矿壁焦痕（不解释成因）
- 矿路短暂亮一次的视觉记忆

### B 簇：遗迹守门线

作用：

- 让遗迹呈现"有人一直在守某种东西"的感觉
- 让守纹残念和碑灵学士提供遗迹侧的第一层事实

推荐任务序列：

1. `talk` — 清玄把信标核心收回后，让你立刻去断碑遗迹。守纹残念在遗迹入口等着，它不像锻炉余火那样热情——它只想确认你是不是又一个来撬门的。"你若只是来翻东西的，不用进去了。"

2. `kill_collect` — 碑灵学士告诉你，要进内层先得把外面会动的石卫清掉。这些石卫不是傀儡，更像是旧阵残留的守卫意志，动作僵硬、目标单一，重复着某个早已失效的指令。带回来的断纹石片上刻着你不认识的字。

3. `submit_item` — 碑灵学士认出石片上的字是旧纹，但有些字它自己也读不出来。"这条纹是守纹，意在封存。但封的是什么东西，我记不全了。"它给的说法只到一半。

本簇产出的碎片：

- 遗迹不像普通废墟，更像被故意封存
- 石卫仍在执行旧指令
- 碑灵学士记不全——连死人都不掌握全貌
- 封的是什么东西？没有人说出答案

### C 簇：旧账对照线

作用：

- 让矿洞和遗迹的碎片互相咬合
- 三方（矿监残魂、碑灵学士、守纹残念）给出不一致的判断
- 这是碎片化叙事在本章的核心体现

推荐任务序列：

1. `submit_item` — 矿监残魂见到信标核心亮过一次后，终于肯翻它的旧矿册。它指着册上的一批名字说"这些人从未出矿，但名字也没算死"。它不解释这意味着什么，只说那是它最后一次值夜前的事。同时交出一份矿晶核作为对照凭证。

2. `talk` — 把矿监残魂的说法带过去问碑灵学士。碑灵学士说它不认得那些人，但它记得"有一批人是在守东西，不是挖东西"。矿洞的真正用途和遗迹一样，都是守。至于守的是什么，碑灵学士说"你问守纹残念"。

3. `talk` — 守纹残念听完两边的话，沉默了很久。它最后只说了半句："他们在守的东西和我在守的东西，是同一件东西的两道门。第一道门早就坏了。第二道门也快了。"

本簇产出的碎片：

- 矿监残魂版本：有一批人从未出矿（人事记录，不完整）
- 碑灵学士版本：矿洞的真正用途是守，不是挖（认知判断，基于碑文）
- 守纹残念版本：矿洞和遗迹守的是同一件东西的两道门（最接近全貌，但只说半句）
- 三段能咬合出轮廓（都在守、都坏了、互相关联），但细节冲突（矿监不知道自己不是矿工，碑灵不知道守的具体是什么）

### D 簇：先入先天线

作用：

- 把"知道得差不多了"转换成"你还得先练"
- 清玄不做世界观定义

推荐任务序列：

1. `learn_technique` — 清玄听完你从三处带回的所有碎片后，沉默了很久。然后他说了一句："这些事不是偶然。"——这是他到目前为止说得最重的一句话，但他没有继续往下说。他给了你一门更高阶的功法。

2. `realm_stage` — 清玄让你先入先天。"你现在知道的，已经够让你在兽谷里活着进去。还不够让你活着出来。先把内外一炁练到先天。"

本簇产出的碎片：

- 清玄终于承认怪事不是偶然（但仍不说是什么）
- 修行门槛再次被推到前面——不是不想告诉你，是你还扛不住
- 兽谷是下一站

## 6. 本章结束状态

- 玩家第一次拿到足够多的相邻碎片
- 碎片能互相对上（同一类事的不同后果），但无法拼成定论
- 矿监残魂/碑灵学士/守纹残念各说各的，细节彼此不完全一致
- 清玄首次松口承认怪事不是偶然，但他不做定义
- 玩家从异常见闻者，变成开始被异常后果推着往前的人

## 7. 本章不该做的事

- 不要把矿洞负责"前因"、遗迹负责"后果"写得过分工整
- 不要让清玄在本章末尾下一个世界观定义（"旧阵崩坏余波外泄"之类）
- 不要让三个旧留者的版本互相印证成一套完整答案
- 不要解释"他们在守的到底是什么"
