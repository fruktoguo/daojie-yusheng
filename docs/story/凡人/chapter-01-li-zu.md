# 《寻仙录》第一卷·外乡人 第一章 立足

状态：重构初稿

适用阶段：

- 玩家称呼：`外乡人`
- 建议等级：`realmLv 2-3`
- 主要地图：`wildlands`、`yunlai_town`

说明：

- 本章从"镇内落脚"推进到"镇外生存"
- 核心不是调查异常，而是看清边镇秩序正在被什么消耗

## 1. 本章目标

- 把"活下来"变成"如何持续过日子"
- 让玩家进入荒野压力场，理解镇子靠外面的路和货活着
- 为同一件事（旧北关）提供至少三种互相不完全一致的说话
- 让玩家感觉到：药铺和脚店对同一名伤者的态度完全不同，各有各的道理

本章不做：

- 不把荒野匪患写成单纯反派清剿
- 不把旧北关写成一个马上要进去解谜的秘密点
- 不给"旧北关到底发生了什么"的标准答案

## 2. 章节摘要

沈七娘把玩家推出镇门，说脚店不养只会在镇里晃的人。去荒野找洛猎户，把南门外的活路和死路认明白。玩家开始处理野彘、匪徒、毒蛛，每一项都不是为了除害，而是为了维持镇子的粮路、药路和安保。

与此同时，序章里那名无名伤者的怪伤口始终没个说法。兰药师只想先把人保住，沈七娘只想先别惹麻烦，文籍师叔却觉得这事和旧北关的旧账能对上——但他的说法和老关卒、野修客的说法全都不一样。

## 3. 氛围关键词

- 粮路
- 荒野生计
- 买命钱
- 旧北关
- 药不白给
- 不同人嘴里的同一件事

## 4. 主要角色

### 4.1 洛猎户

- 身份：荒野猎户，靠山吃山的边缘人
- 作用：把玩家拉进荒野生存链
- 人设：不谈侠义，只谈肚子。话糙理不糙
- 叙事约束：洛猎户对旧北关的事知道得不多，但他知道"那边林子里常有不干净的动静"

### 4.2 兰药师

- 身份：云来镇药铺主人
- 作用：代表"先把人保住"的现实主义
- 人设：冷脸、实际，药不轻给。但她真把伤者当人看，这一点和沈七娘不同
- 叙事约束：兰药师对伤口的看法是"不像寻常兵器，但我也没见过"，不给出超自然判断
- 与沈七娘的分歧：兰药师认为伤者值得救，沈七娘认为不值得倒贴药钱

### 4.3 文籍师叔

- 身份：镇中为数不多还在翻旧档的人
- 作用：提供旧北关的第二种说法（偏旧档记录）
- 人设：书生气、迂阔、记性好但判断力一般
- 叙事约束：文籍师叔的说法基于旧档，但旧档本身就是残的

### 4.4 野修客

- 身份：不入镇、不入派的荒野独行者
- 作用：提供旧北关的第三种说法（偏亲历）
- 人设：语气冷淡，说话只说半句。不肯教人，但也不害人
- 叙事约束：野修客说法最接近真实，但他不肯说完，玩家拿不到定论

### 4.5 清玄执事

- 身份：出现但不出手的信息控制者
- 作用：本章收束时出现，继续压话头
- 叙事约束：清玄听完玩家带回的几种说法后，只说"先把骨头练硬"，不做裁决

## 5. 任务簇

### A 簇：粮路维持线

作用：

- 把日常维持写成剧情主轴
- 让玩家感受到：镇子不是封闭孤点，而是被荒野粮路吊着命

推荐任务序列：

1. `talk` — 沈七娘把你推出镇门，让你去荒野找洛猎户。她说得很直白："脚店里的人要吃饭，饭要从南门外进来。你先把这条路认明白，再回来跟我谈别的。"

2. `kill_collect` — 洛猎户不谈侠义，只谈肚子。野外最近猎物少、人却多，若想让镇里少几个饿死鬼，就得先把獠牙野彘打下来。把还能卖的牙带回去。路上你会看见沿路残缺的车辙和弃置的货袋——有些明显不是自然破损。

3. `submit_item` — 把野彘牙带回给沈七娘。她只看货，不问你路上看见了什么。但当你说起车辙和货袋时，她顿了一下，只说"那条路本来就不好走"，不再往下接。

本簇产出的碎片：

- 镇子靠外面的路和货活着
- 荒野沿路的破损车辙和弃置货袋（不是自然烂的）
- 沈七娘对异常迹象的刻意回避

### B 簇：药路续命线

作用：

- 强化"劫难不是大爆炸，而是慢慢拖垮一切"
- 让药铺和脚店对同一件事呈现不同态度

推荐任务序列：

1. `talk` — 兰药师让你去看那名无名伤者。她已经尽力用药吊着他的命，但伤口始终不收口。她问你人在哪捡到的，问完只皱眉。"这伤不像寻常兵器打的，但我也没见过。"——她不说更多，只让你去取阴沼丝配药。

2. `kill_collect` — 去荒野阴沼杀毒蛛，收回足够的阴沼丝。这不是简单的取药跑腿——阴沼附近的地面有些地方塌陷得不自然，有几处像是被什么东西从下方掏空的。不必解释，只让玩家看见。

3. `submit_item` — 把阴沼丝交给兰药师。她配药时你去沈七娘那里问伤者的事。沈七娘的态度和兰药师完全不同："药钱谁出？救回来他能干活？不能干活的人，救回来也是多一张嘴。"——同一个伤者，两个人看出两种立场。

本簇产出的碎片：

- 兰药师和沈七娘对同一名伤者态度截然不同
- 兰药师认为人值得救，沈七娘认为资源不该倒贴
- 阴沼附近的塌陷（不自然，但不解释）
- 伤者的怪伤口仍没好，也不解释

### C 簇：旧关口风线

作用：

- 提供第一组互相冲突的旧事版本
- 这是碎片化叙事在本章的核心体现

推荐任务序列：

1. `talk` — 文籍师叔听你提到无名伤者的伤口后，让你去问老关卒。老关卒守着北关残墙，只说了一句："旧北关当年不是单纯守不住，是里头的人先不对了。"再问就不肯说了。

2. `talk` — 把老关卒的话带回给文籍师叔。他翻出旧档，给出第二种说法："旧档上写的是粮道断、人散了，但没写为什么散。不过有一页提过，散之前北关守兵已经连续几天听见墙外有回声，像是有人在学他们说话。"

3. `talk` — 去荒野找野修客，问他旧北关的事。他冷笑着给了第三种说法："你们镇里的人只会翻纸。旧北关不是守不住，是有人故意让它守不住的。至于为什么，你自己想。"他不肯多说，走了。

本簇产出的碎片：

- 老关卒版本：里头的人先不对了（口传，惜字如金）
- 文籍师叔版本：旧档记载墙外有回声，人在学人说话（文档，有缺口）
- 野修客版本：有人故意让它守不住（亲历边缘，不说完）
- 三个版本能咬合（都指向"旧北关不是普通失守"），但无法拼成定论

### D 簇：境界站稳线

作用：

- 把练体、锻骨写成"继续往前的资格"
- 不是奖励，而是门槛

推荐任务序列：

1. `learn_technique` — 清玄听完你带回的三种说法，不表态、不裁决。他只说你现在还只配在镇外跑腿，然后给你一门更硬的功法。

2. `realm_stage` — 把身子练到锻骨境。清玄的原话是："若想碰那口井背后的东西，先把骨头练到能扛住荒野的下一层恶意。"

本簇产出的碎片：

- 修行不是奇遇奖励，而是继续做事的门槛
- 清玄知道些什么，但他不裁决任何人的说法
- 他只负责把你推到能继续往前走的程度

## 6. 本章结束状态

- 玩家具备稳定处理镇外事务的资格
- 故事从"先站稳脚跟"推进到"边镇结构正在受损"
- 玩家手里握着关于旧北关的三种不同说法，没有标准答案
- 伤者还活着，伤口还没好，没人能说清
- 沈七娘和兰药师对"值不值得救"各有立场，玩家自己判断

## 7. 本章不该做的事

- 不把荒野匪患写成单纯反派
- 不把旧北关写成马上要进的地图
- 不让任何一个 NPC 说出"旧北关的真相"
- 不让清玄在本章末做任何总结
