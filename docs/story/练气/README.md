# 练气期文档

适用范围：

- `realmLv 19-30`
- 对应《寻仙录》进入修士圈后的第一段正式成长线
- 采用"短主线 + 五行五事件 + 归藏收束"的叙事结构

## 剧情设计

- [练气期剧情设计](./练气期剧情设计.md) — **剧情叙事真源**（v3）
  - 核心剧情：五行错层带正在失控，失控的源头在归藏脉窟
  - 与凡人篇的叙事接缝（封天井过渡、清玄退场）
  - 主要角色（老铁头、陆微、苏简、郭火、许灯、石姑 + 五行失控者五人）
  - 主线三节点：
    - **出井入渡**（2簇）— 从封天井出来 → 栖真渡落脚 → 听脉 → 察觉异常
    - **五行收束**（5条并行线，每图2簇）— 五个失控者各有一个独立事件
      - 裂锋原：断锋道人（金精抽疯）
      - 青萝谷：缠根姑（人藤共噬）
      - 寒汐泽：镜途客（倒影反吞）
      - 赤陨庭：燃骨僧（炉心焚身）
      - 厚脉岭：镇岳先生（残纹反激）
    - **归藏压基**（2簇）— 五块残片拼出旧阵轮廓 → 进入归藏脉窟 → 面对未成道基 → 石姑收束
  - 全期红线与文案口径
