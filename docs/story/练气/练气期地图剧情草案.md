# 练气期地图剧情草案

状态：草案 v3.1

适用范围：

- `realmLv 19-30`
- 五行五图覆盖 `19-29`，归藏脉窟负责最终收束
- 采用“短主线 + 五行任选成长 + 一张混行收束图”的结构

## 1. 设计结论

练气期不继续走凡人篇那种长主线推图，而是改成：

- 1 个低阶修士主城
- 5 个五行主题地图
- 1 个练气末段的五炁混行终图
- 1 条只负责串场、定方向、把人送进归藏收束的短主线

核心目标不是“再讲一大段真相线”，而是让玩家在练气期真正感到：

- 自己已经进了修士圈，但仍只是最底层的低阶修士
- 灵根差异会直接影响地图体验、修炼风险和装备选择
- 练气修行不是单纯吸更多灵气，而是学会识脉、纳气、洗气、锻气、承气
- 最后一段的门槛不是输出更高，而是能把五炁压成初步可承载的道基雏形

一句话概括：

- 凡人篇解决“你能不能碰到门”
- 练气期解决“你进门以后会不会先把自己修坏”

## 2. 与现有版本的衔接

当前版本已经做到：

- `灵脊岭 -> 练气境`
- `天穹残宫 -> 天封核心`
- `q_foundation_oath` 以“练气末段收束”作为旧版终点占位

练气期新内容建议这样接上：

1. `天穹残宫` 仍作为练气开场楔子保留
说明玩家已经见过更高层的残败，但还没有资格去处理

2. `道基初誓` 改成过渡任务
由清玄或相关引路 NPC 把玩家正式引去新的修士据点

3. 练气期主题从“追旧灾真相”切到“先把自己修成能承灾的人”
这样能明显区分凡人篇与练气期的节奏和内容重心

## 3. 实现约束

这轮设计仍要服从当前项目的稳定实现能力。

### 3.1 地图与任务约束

- 地图仍是静态 JSON 配置的格子地图
- `1Hz tick` 下不适合做超大空旷图
- 任务推进仍以 `talk / kill / collect / realm_stage` 为主
- 不把推进建立在护送、多段机关、动态改图、复杂潜入脚本上

### 3.2 设计口径

- 五行地图都做成“外环适应区 + 中段压力区 + 内圈收口区”
- 首领统一放在地图内的核心区，不另外拆五张独立 Boss 房
- 最终第七张图承担统一收束，而不是让五行图各自结束后直接散掉
- 主线只要求玩家完成若干关键目标，不要求五张五行图线性全清
- 五张五行图统一覆盖 `19-29` 级怪物带，沿主路逐段递增
- 五张五行图都固定有两个传送点：一个回 `栖真渡`，一个通往 `归藏脉窟`

## 4. 总体结构

### 4.1 主城更名

原暂定名：`五脉集`

调整后名称：`栖真渡`

调整理由：

- “五脉集”功能很清楚，但名字太直白，像系统占位名
- 练气期据点不该直接叫“城”，也不该像普通坊市地名那样没气质
- “栖真”能表达低阶修士暂避乱世、求一线真修之路
- “渡”既有交通中枢感，也有“过此处才真正离开凡尘边地”的阶段意味

`栖真渡` 不是仙城，也不是宗门山门，而是：

- 低阶修士的前线据点
- 五行材料、残卷、药品和低阶法器的交换地
- 从凡人据点转入修士活动圈后的第一座真正修士主城

### 4.2 七张地图关系

```text
云来镇 / 灵脊岭 / 天穹残宫旧线
            ↓
          栖真渡
  ├─ 金：裂锋原
  ├─ 木：青萝谷
  ├─ 水：寒汐泽
  ├─ 火：赤陨庭
  ├─ 土：厚脉岭
  └─ 终图：归藏脉窟
```

- 五张五行图都应允许不同灵根玩家进入，但内部路线、掉落和危险点明显偏一行
- 五张五行图的主路怪物都按 `19-21 -> 22-24 -> 25-27 -> 28-29` 逐段上升
- 每张五行图入口附近都放回 `栖真渡` 的传送点，深处核心区都放通往 `归藏脉窟` 外围的传送点
- `归藏脉窟` 是练气期真正意义上的终图，负责回收五行成长线

### 4.4 炼气期主线简要

主线保持很短，只给 4 个泛化目标：

1. 到 `栖真渡` 听脉，确认自己现在不是“练气了就能乱冲”，而是先得稳住气海
2. 在五张五行图里任选路线推进，把角色从 `19` 级一路练到 `29` 级左右
3. 收集炼气末段需要的定形、洗脉、承载、合炁材料
4. 从任一五行图深处进入 `归藏脉窟`，完成最终收束

主线不要写成长剧情长跑，只要反复强化一个认知：

- 练气期不是在赶路，是在把自己修成一个不会先崩掉的容器

## 5. 主城：栖真渡

### 5.1 剧情定位

天穹残宫坠落后，附近地脉被撕成五行错层带。一批逃出来的低阶修士、散修匠人、伤残门人和灵材商贩，围着一条尚能勉强平衡五炁的旧渡口重新搭起聚落，这里便是 `栖真渡`。

它和云来镇的区别必须很明确：

- 云来镇重在凡人生存
- 栖真渡重在低阶修士怎么继续修下去
- 云来镇是边镇
- 栖真渡是乱世里的修士前线补给中枢

### 5.2 分区建议

- `听脉台`
玩家初到练气期的第一落点，用来解释灵根、五行纳气和各图路线差异

- `渡心堂`
主线交接点，负责记录五行图推进、材料交付和归藏准入

- `百工巷`
低阶法器、护具、药材与残卷交易区

- `静气坊`
恢复、洗杂、功法校验、闭关文案区

- `行修街`
散修摆摊、零工发布、残篇流通、消息交换

- `压基坪`
后期目标区域，前期可以看到，但只有完成主线后才正式启用

### 5.3 关键 NPC

| NPC | 身份 | 作用 |
| --- | --- | --- |
| 闻脉师·陆微 | 主引路人 | 负责解释五行环境、地图路线和练气期修行重点 |
| 渡记官·苏简 | 记录修士 | 负责主线收束、材料登记、终图准入 |
| 百工老匠·郭火 | 炼材匠人 | 负责装备、材料回收和低阶法器口径 |
| 静坊执事·许灯 | 调息管事 | 负责药品、净气、稳脉相关任务 |
| 守坪人·石姑 | 压基坪守人 | 负责练气末段的压力感与最后收束 |

### 5.4 主城气质要求

- 不做成富足安稳的仙门
- 要有避难点、前线补给点和低阶修士混居感
- 视觉上偏旧木台、补丁阵、临时棚铺、风旗和药烟

## 6. 五行五图设计

## 6.1 金行：裂锋原

### 地图定位

- 关键词：定形、凝锋、破甲、收束
- 更适合偏攻击、破防、单体爆发路线的玩家
- 也是玩家最容易一眼看懂路线倾向的一张图

### 空间结构

- `外环碎岩带`
视野较开，怪物以游荡兵魂和残兵碎灵为主，给玩家熟悉金行节奏

- `中段金砂原`
金砂横扫、残刃插地，直线路径多，怪物硬碰硬压力明显

- `内圈断阵地`
旧阵切开的锋痕和斩裂台地，精英怪开始成组出现

- `收口区：断兵冢口`
不用另切地图，直接作为本图最深处的核心区

### 传送点与等级坡度

- `回渡残阵`
放在入口外环，直接回 `栖真渡`

- `归藏锋门`
放在断兵冢口后侧，直通 `归藏脉窟` 外围

- 怪物等级按主路递增：
`19-21` 外环碎岩带 -> `22-24` 中段金砂原 -> `25-27` 内圈断阵地 -> `28-29` 断兵冢口

### 主要怪与首领

- 普通怪：`断刃游魂`、`裂甲砂兵`、`金砂锋魈`
- 精英怪：`断碑锋傀`、`残旌铁卫`
- 区域首领：`庚金镇墓将`

### 资源与掉落方向

- 基础材料：`庚砂`、`残兵铁片`
- 稀有材料：`锋纹残晶`
- 装备方向：破甲、暴击、收束单体伤害、短时爆发

### 剧情作用

- 让玩家明白金行不是“只是更疼”
- 真正的金行修炼，是把散乱灵气压成可用的锋形

## 6.2 木行：青萝谷

### 地图定位

- 关键词：养脉、生长、牵引、续航
- 更适合回复、持续战、保命和修炼向路线
- 是练气前段最适合做“调息体验”的地图之一

### 空间结构

- `谷口药圃`
安全采集区，主要承担低压采药与入门怪

- `藤桥岔谷`
多岔路和回环路径，体现木行“被牵着走”的路线语言

- `深处妖圃`
妖藤与木魅盘踞，采集和战斗开始交织

- `收口区：心藤坳`
作为本图最终压力区，由巨大妖藤盘据

### 传送点与等级坡度

- `回渡藤门`
放在谷口药圃，直接回 `栖真渡`

- `归藏木隙`
放在心藤坳深处，直通 `归藏脉窟` 外围

- 怪物等级按主路递增：
`19-21` 谷口药圃 -> `22-24` 藤桥岔谷 -> `25-27` 深处妖圃 -> `28-29` 心藤坳

### 主要怪与首领

- 普通怪：`青萝小妖`、`寄脉藤童`、`吸灵花妖`
- 精英怪：`枯荣藤卫`、`碧髓树魈`
- 区域首领：`噬脉藤母`

### 资源与掉落方向

- 基础材料：`青髓藤`、`养脉叶`
- 稀有材料：`生灵木心`
- 装备方向：生命回复、真气恢复、持续恢复、养脉效率

### 剧情作用

- 告诉玩家木行不是“奶量高”这么简单
- 它真正考的是你能不能让气脉活下来，不被慢性侵蚀拖死

## 6.3 水行：寒汐泽

### 地图定位

- 关键词：洗练、净化、流转、化解
- 是练气期最重要的调和地图
- 更适合处理中途出现的杂气冲突和异常状态压力

### 空间结构

- `浅泽外圈`
可探索的浅滩与低雾桥，路线不难，但地貌辨识度要强

- `双流水道`
两条相互缠绕的主水路，让玩家感到越走越冷、越深越静

- `月井湿宫`
内层区域偏封闭，怪物控制和减速明显增强

- `收口区：月井寒台`
作为本图深层核心区，围绕一口古月井展开

### 传送点与等级坡度

- `回渡雾桥`
放在浅泽外圈，直接回 `栖真渡`

- `归藏井门`
放在月井寒台后侧，直通 `归藏脉窟` 外围

- 怪物等级按主路递增：
`19-21` 浅泽外圈 -> `22-24` 双流水道 -> `25-27` 月井湿宫 -> `28-29` 月井寒台

### 主要怪与首领

- 普通怪：`寒汐蜉灵`、`浅泽雾鬼`、`冰纹水卒`
- 精英怪：`月沼鲛影`、`雾桥巡灵`
- 区域首领：`玄溟螭影`

### 资源与掉落方向

- 基础材料：`寒魄露`、`月井冰砂`
- 稀有材料：`净潮水精`
- 装备方向：净化、减伤、真气回复、控制抗性、稳定循环

### 剧情作用

- 把“会吸气”推进到“会不会把杂气洗出去”
- 是最适合承担中段净化与稳定循环的一张图

## 6.4 火行：赤陨庭

### 地图定位

- 关键词：淬炼、爆发、烈性、以险换强
- 更适合高风险高收益、短时间强化路线
- 也是最能做出“练气不是站桩修仙”的一张图

### 空间结构

- `焦土外庭`
给玩家熟悉火行环境伤害和快节奏怪群

- `熔裂廊道`
之字形向下切层，越往深处越像走进一座坠火坑炉

- `陨火内院`
高密度精英区，怪群带灼烧与爆发压制

- `收口区：炉心台`
整张图的核心压力点，视觉上像未冷却的古炉残庭

### 传送点与等级坡度

- `回渡火阶`
放在焦土外庭，直接回 `栖真渡`

- `归藏炉隙`
放在炉心台下层，直通 `归藏脉窟` 外围

- 怪物等级按主路递增：
`19-21` 焦土外庭 -> `22-24` 熔裂廊道 -> `25-27` 陨火内院 -> `28-29` 炉心台

### 主要怪与首领

- 普通怪：`焦土火蛾`、`赤烬蜥幼`、`熔纹火蜥`
- 精英怪：`炽骨巡卒`、`流火戍灵`
- 区域首领：`离火炉君`

### 资源与掉落方向

- 基础材料：`陨火砂`、`炎髓炭`
- 稀有材料：`炉心赤晶`
- 装备方向：灼烧、攻速、短时增伤、低血爆发、淬体收益

### 剧情作用

- 让玩家明白火行不是“高攻模板”
- 真正的火行问题是你扛不扛得住烈气入体后的反噬

## 6.5 土行：厚脉岭

### 地图定位

- 关键词：承载、镇压、稳脉、立基前夜
- 是练气末段最关键的一张地图
- 更适合防御、护盾、抗打断和厚重型路线

### 空间结构

- `山脚采脉区`
承担材料采集和中低强度刷怪

- `坠石狭道`
路线变窄，开始强调击退、卡位和承压

- `石柱高坪`
高压精英区，视觉上像被硬生生抬起来的断层山台

- `收口区：镇岳石府前庭`
不另开图，直接作为本图最终压力区

### 传送点与等级坡度

- `回渡石坪`
放在山脚采脉区，直接回 `栖真渡`

- `归藏岳门`
放在镇岳石府前庭，直通 `归藏脉窟` 外围

- 怪物等级按主路递增：
`19-21` 山脚采脉区 -> `22-24` 坠石狭道 -> `25-27` 石柱高坪 -> `28-29` 镇岳石府前庭

### 主要怪与首领

- 普通怪：`黄岩鼍兽`、`沉脉石奴`、`坠岩甲灵`
- 精英怪：`厚土负碑者`、`裂岭重卫`
- 区域首领：`玄岳地傀`

### 资源与掉落方向

- 基础材料：`承脉石`、`厚岩核`
- 稀有材料：`镇岳石胆`
- 装备方向：护盾、减伤、抗打断、承载上限、稳脉收益

### 剧情作用

- 明确告诉玩家，立基之前最重要的不是吸更多
- 而是身体、经脉和气海能不能扛住你已经吸进去的东西

## 7. 第七张终图：归藏脉窟

### 7.1 地图定位

- 关键词：五炁混杂、旧阵回流、炼气末段总收束
- 这是练气期真正的最终地图
- 玩家不是来学某一行，而是来验证自己是否能让五炁共处而不崩

### 7.2 剧情背景

残宫坠落后，最深处有一段旧时代的聚脉窟未完全崩碎。那里原本是低阶弟子试立道基的地方，如今阵法残缺，五行倒灌，成了高等级练气修士也未必敢久留的混脉险地。`栖真渡` 的人一直把它当成真正的最后一道门槛。

### 7.3 空间结构

- `乱流甬道`
入口区，五行轮流压上来，用来检查玩家有没有明显短板

- `偏脉回廊`
五个小段拼成的环路，每一段分别偏金木水火土，但不像单行图那样纯粹

- `归藏心庭`
中段平台，开始出现复合怪和双属性精英

- `压基古阵`
最终区域，旧阵残存，还能勉强把五炁往一个容器里压

### 7.4 入口结构与等级带

- 五张五行图深处的 `归藏传送点` 全部汇入 `乱流甬道`
- `乱流甬道` 和 `偏脉回廊` 主要放 `29` 级怪
- `归藏心庭` 与 `压基古阵` 放 `30` 级精英和最终首领

### 7.5 主要怪与首领

- 普通怪：`错脉游灵`、`逆流残侍`、`五炁砂偶`
- 精英怪：`双相脉兽`、`乱基守遗`
- 最终首领：`未成道基`

`未成道基` 的设计口径：

- 不做人格化魔王
- 更像当年失败立基后留下的一团半成灵胎与阵残合物
- 它体现的是“练气修坏以后会变成什么”

### 7.6 资源与掉落方向

- 基础材料：`五炁尘`、`归藏脉核`
- 稀有材料：`半基灵胚`
- 装备方向：混行增益、五行平衡、立基准备、综合续航

### 7.7 剧情作用

- 回收前面所有五行路线
- 让玩家在进入筑基前，真正经历一次“五行不是五张图，而是一体气海”的确认
- 在击破 `未成道基` 后，正式把角色推到 `练气十二层`

## 8. 炼气期新装备与新功法

这一期不需要把装备做成海量散件，建议先按“系列明确、流派清楚、来源稳定”投放。

## 9. 任务组织建议

主线保持短，五行图承担大部分内容厚度。

建议主线节点只保留：

1. `入渡听脉`
进入 `栖真渡`，完成第一次听脉，拿到 `五脉引气诀`

2. `五图修行`
在五张五行图中任选推进路线，沿主路打到更深层区域，持续换装和换功法

3. `备材入藏`
凑齐炼气末段需要的材料，并解锁任一地图深处的归藏传送点

4. `归藏压基`
进入 `归藏脉窟`，击破 `未成道基`，完成炼气期最终收束

支线重点则由五行图各自承担：

- 金行问“你能不能把气收成形”
- 木行问“你能不能把脉养住”
- 水行问“你能不能把杂气洗净”
- 火行问“你能不能扛住烈气反噬”
- 土行问“你能不能让气海承得住”

## 10. 这版结构的好处

- 地图数量清楚，正好对应你要的七张主地图
- 主线不长，但阶段目标明确
- 五行成长、装备、功法、材料和药品都有稳定落点
- 练气末段有独立终图，不会让练气期尾声显得太散
- `栖真渡` 比旧名更像正式地名，也更符合低阶修士前线据点气质
