# 寒汐泽地图设计

## 1. 总体定位

- 地图名：`寒汐泽`
- 五行属性：水
- 关键词：旋入、环绕、洗练、净化、寒滞

寒汐泽不该是横着切几段的水泽图。

水行真正该有的感觉，是人从外围踏进去以后，被整片水势一点点拽着转，绕三四圈，越转越冷，越转越静，最后才被送到月井核心。

所以寒汐泽这次直接做成一张完整大图：

- 从外围进入
- 主路按圆环向内旋
- 中间不是普通沼地，而是一整片新的 `寒沼`
- 玩家体验应该像被一股冷水慢慢卷进井心，不是一路横推到终点

## 2. 地块语言

这张图现在用这套地块：

- `,`：浅滩、退潮露地、泽边缓区
- `:`：雾桥、湿滑堤道、环形次路
- `=`：较稳的主路、旧井堤、外圈进入道
- `;`：冷泥滞带，表示寒气已经开始压脚
- `%`：旧沼区、寒湿压迫带
- `~`：深水主隔断，用来切出旋转感
- `o`：井栏残构、石桩、道标
- `寒`：新的 `寒沼` 地块，位于中心寒区

### 寒沼的用途

寒沼不是普通沼泽换个颜色。

它在设计上的作用是：

- 作为寒汐泽中心区的独立辨识
- 让玩家一眼知道“这里已经不是外围湿地，而是月井真正的冷核”
- 在视觉和玩法上都把中心区从普通 `%` 沼地里拔出来

## 3. 空间结构

寒汐泽现在改成完整大图，结构是：

1. `回渡外滩`
从外围左侧切入，先给玩家一个还能稳住脚步的入口带

2. `第一环`
开始围着外圈转，水线和堤道第一次成形

3. `第二环`
深水隔断明显增加，主路不再直给

4. `第三环`
已经能感到月井在中间压着，人开始被整张图的重心往内拖

5. `第四环`
路线进一步收紧，外围感彻底消失

6. `月井寒核`
中心一整片寒沼，外圈用井堤和残构围住，形成最终核心

## 4. 设计判断

这张图最重要的不是“水多不多”，而是三个词：

- `旋`
路线必须让人产生明确的旋入感

- `绕`
不是直冲，而是环着核心不断调整方向

- `冷`
越靠近中心，越有一种被月井寒意压住的感觉

如果后面继续补细节，优先补的是：

- 寒沼边缘的井台层次
- 环道上不同圈层的桥、堤、湿宫残构
- 月井中心和外围之间的压迫差

不是继续横切成很多散图。

## 5. 本轮落地口径

本轮先只做一张完整大地图的纯地块版本：

- 不放怪物
- 不放灵气
- 不放传送
- 不放其他运行元素

先看环绕感、旋入感和寒沼中心够不够成立，再补玩法层。
