## 1. 文档口径

这份是策划文档，不按配置文件写法展开。

每只怪统一写以下内容：

- 等级、品阶、血脉
- 装备搭配
- 六维
- 数值倍率
- 技能
- 掉落

说明：

- 这里不直接写运行时最终面板
- 六维仍沿用当前项目怪物计算口径，只是在文档里写成中文：
  - `体` = 体魄
  - `神` = 神识
  - `身` = 身法
  - `根` = 根骨
  - `悟` = 悟性
  - `运` = 气运
- 数值倍率统一按下列顺序写：
  - `血` = 生命倍率
  - `气` = 灵力倍率
  - `物` = 物攻倍率
  - `法` = 法攻倍率
  - `物防` = 物防倍率
  - `法防` = 法防倍率
  - `命` = 命中倍率
  - `闪` = 闪避倍率
  - `暴` = 暴击倍率
  - `破` = 破招倍率
  - `化` = 化解倍率
  - `灵出` = 灵力输出倍率
  - `速` = 移速倍率
- 这批装备和技能都按炼气期新内容单独命名，不复用凡人期旧装备和旧技能名
