# 练气期装备草案

状态：草案 v1

适用范围：

- `realmLv 19-30`
- 对应 [练气期地图剧情草案](./练气期地图剧情草案.md)
- 对应 [练气期地图怪物草案](./练气期地图怪物草案.md)

## 1. 设计口径

这份文档只写练气期玩家装备，不写怪物穿戴。

统一约束：

- 继续兼容当前项目的 `5` 个装备槽位：武器、头部、身上、下装、饰品
- 装备成长采用 `栖真渡过渡 -> 五行专精 -> 归藏收束` 三段结构
- 不做旧凡人篇装备换名复用，这一批全部是练气期新装备
- 装备效果先按“静态词条 + 流派定位”设计，暂不强绑复杂特效
- 每张五行图都给出一整套 `5` 件代表装备，方便后续落成实际装备池

练气期装备要解决的不是“数值更大”，而是“你开始按五行路线穿装”。

## 2. 整体成长路线

| 阶段 | 对应区域 | 装备作用 | 主风格 |
| --- | --- | --- | --- |
| 过渡期 | `栖真渡` | 先把凡人期散装换掉，稳住练气前段气海 | 稳气、导气、保命 |
| 专精期 | 五行五图 | 按金木水火土路线做明显分流 | 各行偏科成长 |
| 收束期 | `归藏脉窟` | 把单一路线装备收回到练气末段的混行需求 | 合炁、承压、定形 |

## 10. 装备分流建议

如果玩家不想纯走单行路线，建议这样混搭：

- 金 + 火：走爆发、破招、收头
- 木 + 水：走法修、控制、续航
- 土 + 金：走近战抗压、定点拆精英
- 土 + 木：走稳扎稳打、长战线
- 归藏件：优先替换饰品和身上，再考虑武器

## 11. 后续落地建议

如果下一轮要继续往实际配置收：

- 栖真渡先做 `5` 件通用过渡装
- 每张五行图优先落 `2` 件最能体现流派差异的核心件
- `归藏脉窟` 至少保留 `3` 件真正值得毕业前更换的收束装备
- 装备名尽量与地图材料、怪物掉落和功法名保持同一套口径
