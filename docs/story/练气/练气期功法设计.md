# 练气期功法设计

状态：草案 v3

适用范围：

- `realmLv 19-30`
- 对应 [练气期地图剧情草案](./练气期地图剧情草案.md)
- 对应 [练气期装备草案](./练气期装备草案.md)
- 对应 [练气期技能草案](./练气期技能草案.md)

## 1. 设计结论

练气期的核心不是“限制玩家只能学几门功法”，而是：

- 所有功法都能学
- 但路线不能靠“无脑全学就毕业”成立

也就是说，练气期要做的不是职业锁死，而是 build 成形。

一句话结论：

- 凡人期更多是“学会几门能打的本事”
- 练气期开始变成“你把哪些功法练成一套能自洽的气海结构”

因此这段的分方向，应该主要通过以下几件事实现：

- 功法获取难度不同
- 功法养成成本不同
- 同方向功法、技能、装备围绕同一类路线 Buff 联动更强
- 不同方向混搭有收益，但不会比成体系 build 更稳定

## 2. 为什么不能用“限制可学数量”来做分流

修仙题材里，“见法就不能学”通常不够顺。

更自然的表达应该是：

- 你可以广泛涉猎
- 但不可能在同一阶段把所有路子都练到同样深
- 真正决定强度的不是“你有没有这本书”，而是“你把哪条路练成体系”

这样更符合练气期的味道：

- 散修、低阶修士本来就会到处收残卷、换法门
- 能学很多，不代表能都精
- 同修多门不是问题，问题是你的气海最终按什么路数运转

所以练气期应当保留“所有功法都能学”的开放性，同时让“全学”只停留在收集层，不自动等于最优 build。

## 3. 核心思路

### 3.1 学会不等于成型

练气期首先要把两个概念拆开：

- 学会：玩家拥有这门功法，能继续投入修炼
- 成型：这门功法已经进入当前 build 的核心循环

允许玩家把功法都学到手，但 build 是否成立，要看：

- 当前主修投入是否足够
- 配套功法之间有没有联动
- 装备、地图、灵根、技能链是否在同一条线上

### 3.2 路线靠“投入重心”形成，不靠硬锁形成

练气期仍然可以保留“主修”概念，但这里的主修更适合定义成：

- 当前主要投入功法经验的对象
- 当前 build 的联动中心
- 当前突破和地图推进最依赖的一门功法

它不应该被写成“只有主修才生效，其余全部失效”。

更合理的口径是：

- 所有已学功法都可以提供成长
- 但主修功法承担最高投入、最高联动密度和最高阶段价值
- 其他功法更多承担补短板、开技巧、补 tag、补特定场景的作用

### 3.3 分方向主要靠三层压力

练气期的 build 分流，建议主要由三层压力共同完成。

第一层：获取压力

- 五行功法书分布在不同地图、不同怪群、不同深度区
- 玩家理论上都能去拿，但早期不可能无代价横扫所有线
- 路线从“你先去哪一图、先刷哪门法”就已经开始分化

第二层：养成压力

- 功法经验、修炼时间、材料、地图推进都会形成投入竞争
- 玩家可以学很多，但在 `19-30` 这一小段里，不可能把五行功法都练成同强度
- 这会自然逼出“主轴 + 辅轴 + 补位”的 build 结构

第三层：联动压力

- 同方向功法、技能、装备围绕同一类路线 Buff 时，要明显强于无脑拼盘
- 玩家若只是把所有功法都学一遍却没有联动设计，收益应该很散
- 真正强的是“围绕一套 Buff、一套属性、一套节奏”堆出来的组合

## 4. 练气期的功法结构

### 4.1 通用起步功法

名称沿用现有方案：

- `听脉定息诀`

定位：

- 所有人都能拿到
- 负责练气前段的稳气、调息、保命
- 是通用底盘，不是长期唯一答案

它的作用不是替代路线，而是让玩家进入练气期时先有一个“能跑起来的底座”。

### 4.2 五行主轴功法

五行功法仍然是练气期的主体内容，但这里的“主轴”不等于“排他”。

建议按当前草案保留：

| 方向 | 功法 | 地图 | 主战斗气质 | 主属性倾向 |
| --- | --- | --- | --- | --- |
| 金 | `庚华凝锋诀` | `裂锋原` | 单体压制、破甲、收头 | 体魄、身法、根骨 |
| 木 | `青木养脉篇` | `青萝谷` | 续航、吸取、磨战 | 神识、根骨、悟性 |
| 水 | `寒汐洗脉法` | `寒汐泽` | 控场、减速、法修游斗 | 神识、身法、悟性 |
| 火 | `离炎锻息诀` | `赤陨庭` | 爆发、压血、强收尾 | 体魄、神识、身法 |
| 土 | `厚土承元功` | `厚脉岭` | 抗压、站场、反打 | 体魄、根骨、悟性 |

设计目标：

- 每门都能学
- 但每门都应该更适合某一类 build 核心
- 玩家在这一阶段通常会自然形成 `1 主轴 + 1 补强方向 + 若干低投入泛修`

### 4.3 归藏收束功法

`归藏合脉经` 仍然应该放在练气末段，主要服务 `练气十一层-练气十二层` 的最终收束，但它的意义不是“终于给你一个万能功法”。

更准确的定位应该是：

- 把前面已经形成的 build 往练气末段的立基前夜收
- 提高混行环境下的稳定性
- 让多方向涉猎的玩家有一个收束接口

它不是把前面五行路线推翻重练，而是把已有积累压成更稳的雏形。

## 5. build 应该怎么形成

### 5.1 先靠来源形成方向

五行图不是五个平铺的素材池，而是五条不同的成长入口。

因此功法获取上建议明确拉开：

- 入口附近掉基础页、残篇、低层修炼材料
- 中段掉完整主修书或关键升层材料
- 深处和首领掉高联动部件、强化残卷、后段 build 核心件

这样一来，玩家即便理论上都能学，也会因为先刷哪张图而自然偏向某条线。

### 5.2 再靠联动形成强弱

真正把路线做出来的，不是“只准学这个”，而是“同线联动更值”。

建议练气期每个方向至少建立一套清晰的路线 Buff，并让核心技能围绕 Buff 层数运作：

| 方向 | 共享 Buff | build 目标 |
| --- | --- | --- |
| 金 | `锋痕` | 压单体、拆精英、斩杀 |
| 木 | `缠脉` | 长线作战、越拖越稳 |
| 水 | `寒印` | 法修游斗、减益叠压 |
| 火 | `灼脉` | 快节奏压血、强收尾 |
| 土 | `镇脉` | 稳态站场、反手压制 |

要求：

- 每条路线至少要有 `1` 个稳定叠层技能、`1` 个按层数强化的主输出技能、`1` 个收束或消耗技能
- 同方向技能、装备、怪物掉落词条优先服务同一个路线 Buff
- 技能效果默认直接受 Buff 层数影响，例如增伤、破甲、减速、吸取、延长持续时间或额外结算
- 同 Buff 之间收益明显高于无关联拼装
- 不同方向混搭也要能玩，但收益要更偏功能互补，而不是全能碾压

### 5.3 混搭 build 要有，但要讲道理

练气期不该把玩家锁成纯色路线。

更好的状态是：

- 纯一路线有最稳定、最顺手的闭环
- 两路混搭有鲜明风格
- 多路泛修可以玩，但不会是最省心的解

建议优先鼓励的混搭：

- 金 + 火：破甲后收头，适合爆发斩杀
- 木 + 水：控场加续航，适合法修长线
- 土 + 金：站住后压单体，适合拆精英
- 土 + 木：承伤加续航，适合稳扎稳打

这类 build 的重点不是“两个方向都学了”，而是“两个方向的属性和路线 Buff 能接起来”。

## 6. 如何避免“无脑全学就最强”

既然核心是“所有功法都能学”，那系统上就要避免学得多自动等于最优。

建议主要靠以下几条约束，而不是靠禁学：

### 6.1 获取节奏约束

- 不同功法分散在不同图和不同深度
- 想全拿，首先就要横跨多张图和多轮刷取
- 这本身就会拉开路线成型时间

### 6.2 修炼投入约束

- 功法经验和修炼时间是有限资源
- 玩家可以样样会一点，但当期境界里只能重点练少数核心法门
- build 的差异首先来自“你把哪些法门练深了”

### 6.3 联动密度约束

- 一条路线内的技能、词条、装备、材料要围绕同一个路线 Buff 彼此互相抬
- 纯拼盘如果没有共享 Buff 或明确的叠层收益，收益就会散
- 这样“全学”更多是广，而不是强

### 6.4 成长衰减约束

现有系统里已经有“功法总属性按品阶分组后做软衰减”的基础。

这条很适合继续保留其设计意义：

- 鼓励玩家多学
- 但不鼓励单纯靠堆满同品阶功法数量把面板硬灌爆

这样可以把“广泛涉猎”保留下来，同时避免“全家桶堆属性”直接成为最优解。

## 7. 与现有系统的衔接

这套思路和当前项目的已有基础是能接上的，不需要改成完全不同的功法系统。

当前已有可直接利用的点：

- 已有“设为主修”的概念，可以把它继续定义成当前投入重心
- 已有独立功法经验推进，可以天然制造养成竞争
- 已有功法属性累计和软衰减，可以限制纯数量堆叠
- 已有技能从属于功法的结构，便于把路线 Buff 联动写在功法体系里

因此后续落地建议不是“限制玩家只能生效两门”，而是：

1. 保持所有功法都可学习
2. 强化主修功法在当期 build 中的效率和联动中心地位
3. 用地图来源、材料来源和掉落深度拉开各路线成型速度
4. 把五行路线 Buff 联动真正写进技能、装备和怪物掉落
5. 用属性软衰减和养成成本压住“全学全练就是最优”

## 8. 对现有练气期功法池的整理结论

结合当前草案，练气期第一版功法池本身已经够用：

| 类型 | 功法 |
| --- | --- |
| 通用起步 | `听脉定息诀` |
| 五行主轴 | `庚华凝锋诀`、`青木养脉篇`、`寒汐洗脉法`、`离炎锻息诀`、`厚土承元功` |
| 终图收束 | `归藏合脉经` |

这套池子的问题不在于“还不够多”，而在于还没把 build 规则写清楚。

当前真正该补的是：

- 各功法的获取分层
- 各方向的共享路线 Buff
- 同属性和同 Buff 层数的联动收益
- 混搭路线的合理上限

## 9. 本轮定稿口径

本轮先把以下结论定下来：

- 练气期核心必须是“所有功法都能学”
- 分方向不靠禁学，而靠获取难度、养成投入和联动收益
- 主修是投入重心，不是排他开关
- 同方向 build 必须通过共享路线 Buff 明显强于无脑拼盘
- 混搭 build 要有空间，但要靠属性和 Buff 讲逻辑
- `归藏合脉经` 负责练气末段收束，不负责把前中期分流抹平
- 现有 [练气期技能草案](./练气期技能草案.md) 后续应继续按“功法联动 build”而不是“硬职业锁”去细化
