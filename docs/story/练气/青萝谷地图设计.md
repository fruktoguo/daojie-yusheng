# 青萝谷地图设计

状态：草案 v1

适用范围：

- 对应 [练气期地图剧情草案](./练气期地图剧情草案.md) 里的木行图
- 用于后续拆分 `verdant_vine_vale` 的六段子图与总图拼合

## 1. 总体定位

- 地图名：`青萝谷`
- 地图 ID：`verdant_vine_vale`
- 五行属性：木
- 等级带：`19-29`
- 核心体验：养脉、续航、牵引、慢压、回气

青萝谷不是“站着回血”的图，而是让玩家在被藤、叶、雾、药圃一路牵着走的过程中，学会边打边养、边耗边稳。

路线语言不做大开大合，而是：

- 主路不断被藤桥、药圃畦道、藤墙口、湿地边线牵住
- 视线不一定窄，但脚下总在被引导
- 越往深处，战斗和采集越不分家

## 2. 六段结构

青萝谷按六部分走：

1. 入口：`回渡药圃`
2. 第一段：`引藤坡`
3. 第二段：`双桥岔谷`
4. 第三段：`噬灵妖圃`
5. 第四段：`盘根内坳`
6. 出口：`心藤坳口`

## 3. 各段设计

### 3.1 入口：回渡药圃

定位：

- 让玩家一进图就知道这里是木行图
- 不是先打，而是先看见药圃、藤架、灵植和能喘气的外谷

空间：

- 地图右下侧入场，接 `栖真渡`
- 入口不是一条硬路，而是药圃畦道加藤棚走廊
- 畦道之间能穿，但会被藤架自然分成几条细路

视觉组件：

- 药圃田畦
- 藤棚
- 小水沟
- 晒叶台
- 回渡藤门

功能：

- 放回城传送点
- 放第一组最低压怪
- 放低级木灵气，偏 4 级
- 放基础采集点，主要是 `养脉叶`

怪物带：

- `19-21`
- 主打 `青萝小妖`

### 3.2 第一段：引藤坡

定位：

- 从药圃过渡到真正的木行路线
- 开始让藤蔓把路“拽住”

空间：

- 不是直坡，而是缓坡加弯折
- 主路被几道横藤切成 S 形推进
- 左右各有一两块小空地，像给人喘气，但怪会从侧面压过来

视觉组件：

- 横生老藤
- 低湿草坡
- 半塌木栏
- 引脉石桩

功能：

- 开始出现双线可选走法，但最后会重新汇回主路
- 放几格木灵气和无属性灵气重叠点
- 让玩家知道同一点位可以是“稳气点”

怪物带：

- `21-23`
- `青萝小妖`
- `寄脉藤童`

### 3.3 第二段：双桥岔谷

定位：

- 青萝谷最核心的“被牵着走”区段
- 真正建立木行地图的路线辨识度

空间：

- 两条藤桥上下并行
- 下桥偏稳，绕得远，怪少一点
- 上桥偏险，短，但桥窄且容易被夹
- 两桥之间有视线联系，但不是处处可直接互跳

视觉组件：

- 双层藤桥
- 枯木立柱
- 雾藤吊根
- 桥下湿地

功能：

- 这里是地图的中段分岔
- 允许玩家自己选“稳打”还是“快推”
- 中段开始放更明显的木灵气密度，主要 4 级，少量 5 级

怪物带：

- `22-24`
- `寄脉藤童`
- `吸灵花妖`

### 3.4 第三段：噬灵妖圃

定位：

- 战斗和采药开始真正缠在一起
- 看起来资源很多，但一站住就容易被木妖拖节奏

空间：

- 由几块药圃和藤墙口拼成
- 不是整片大平地，而是一个个小圃块
- 圃块之间有细道和泥水边，走起来容易被绕背

视觉组件：

- 妖化药圃
- 吸灵花丛
- 绞藤墙
- 孢雾角

功能：

- 采集点和怪区高度重叠
- 让玩家体会木行图的“资源越好，压力越缠”
- 可放 `青髓藤`、`养脉叶`

怪物带：

- `25-27`
- `吸灵花妖`
- 精英开始出现 `枯荣藤卫`

### 3.5 第四段：盘根内坳

定位：

- 进入首领前的内收压力段
- 不再是绕，而是越来越被根系和老藤逼成窄口

空间：

- 外面还算开，越往里越被盘根圈住
- 主路形成几次“外宽内窄”的喉口
- 偏向持续压血、拖节奏，不是一下秒人

视觉组件：

- 盘根墙
- 老藤拱门
- 木瘤地包
- 枯枝屏风

功能：

- 精英主要区
- 放更高密度的 5 级木灵气
- 让玩家在进首领前就感受到“续航检查”

怪物带：

- `27-29`
- `枯荣藤卫`
- `碧髓树魈`

### 3.6 出口：心藤坳口

定位：

- 首领区
- 视觉上像整个青萝谷的根心被一株老藤盘住

空间：

- 外圈是半封闭藤环
- 内圈是相对完整的一块坳地
- 不是特别大，但要有足够走位空间，避免只剩站桩互殴

视觉组件：

- 心藤主根
- 藤母盘座
- 坳口木环
- 归藏木隙

功能：

- 放首领 `噬脉藤母`
- 放去 `归藏脉窟` 的传送点
- 放整图最稳定的一组 5 级木灵气

怪物带：

- `29`
- 首领：`噬脉藤母`

## 4. 主路节奏

青萝谷的主路节奏按这个坡度走：

- 外谷药圃：先稳气、低压熟悉
- 引藤坡：开始被路线牵引
- 双桥岔谷：开始做路线选择
- 噬灵妖圃：资源和战斗真正缠住
- 盘根内坳：持续压血压气
- 心藤坳口：最终首领与出口

## 5. 灵气口径

青萝谷只放木属性灵气，不混别的五行灵气。

布法：

- 入口到双桥：以 4 级木灵气为主
- 妖圃到内坳：4 级和 5 级混合
- 坳口和首领前：以 5 级木灵气为主

同时保留部分无属性灵气点，用来做“稳气落脚点”。

## 6. 后续落地顺序

后续建议按这个顺序继续：

1. 先做 `回渡药圃`
2. 再做 `引藤坡`
3. 再做 `双桥岔谷`
4. 再做 `噬灵妖圃`
5. 再做 `盘根内坳`
6. 最后做 `心藤坳口`

做完六张子图后，再拼回 `verdant_vine_vale` 总图。
