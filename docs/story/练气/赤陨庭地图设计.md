# 赤陨庭地图设计

## 1. 总体定位

- 地图名：`赤陨庭`
- 五行属性：火
- 关键词：下坠、淬炼、爆发、烈压、以险换强

赤陨庭不能做成一条平推火路。

火行真正该有的感觉，不是地上随便摆点红色地块，而是人一脚踏进来以后，能明显感觉自己在往一座失控的古炉里下沉。

所以这张图这次直接做成一张完整大图：

- 从外侧火阶切入
- 主路沿着坑炉的断层台地一层层往里压
- 不是平面推进，而是回折、下切、再收口
- 中心不是普通焦地，而是一整片新的 `熔池`

玩家体验应该像是在坠进一座还没冷透的古炉残庭，越往里走，越热，越窄，越逼着人提速和冒险。

## 2. 地块语言

这张图现在用这套地块：

- `=`：较稳的火阶、断层台道、主推进线
- `:`：裂火侧路、热缝捷径、局部斜切线
- `,`：冷却焦土、外庭灰地
- `;`：灼热灰带、半熟焦泥、贴着熔裂带的危险缓冲区
- `^`：抬起的焦坡、塌起的炉渣坡、外圈坠层
- `崖`：断层炉壁、残庭围壁、不可走的坍塌边
- `o`：炉石、砸落陨块、挡线障碍
- `熔`：新的 `熔池` 地块，位于中层熔裂带和炉心区

### 熔池的用途

熔池不是把水换个颜色。

它在设计上的作用是：

- 把赤陨庭的炉心和普通焦土地带彻底区分开
- 让玩家一眼知道这里已经不是“烫一点”的火地，而是古炉真正没熄掉的熔心
- 用一整圈熔裂带把主路压成回折下切，而不是平着走到终点

## 3. 空间结构

赤陨庭现在改成完整大图，结构是：

1. `回渡火阶`
从上方切入，先落到还能站稳脚的外圈火阶

2. `焦土外庭`
外圈较宽，先让人看见整座坑炉的轮廓和第一圈熔裂带

3. `第一层熔裂台`
路线沿右侧下切，开始出现明显的熔池隔断

4. `第二层陨火廊`
主路改成横向回折，走位空间被压缩，已经有“被古炉往里卷”的感觉

5. `第三层内院`
线路再次收口，只剩几条能稳定贴着熔裂边推进的台道

6. `炉心台`
中心保留一块还能落脚的台面，四周全是熔池，用来立住整张图最终的火行核心

## 4. 设计判断

这张图最重要的不是“红不红”，而是三个词：

- `坠`
要让玩家感觉自己是在往坑炉里掉，不是在野外横穿

- `压`
越往里，主路越少，熔裂带越明显，空间要越收越紧

- `烈`
中心一定要比外层更躁、更硬、更像一口还在喘火的古炉

如果后面继续补细节，优先补的是：

- 每一层台地之间的断层感
- 熔池边缘的焦黑、破炉石和陨火残构
- 炉心平台和外层焦土在温度感上的强烈区分

不是继续把火图做成一块普通焦地。

## 5. 本轮落地口径

本轮先只做一张完整大地图的纯地块版本：

- 不放怪物
- 不放灵气
- 不放 NPC
- 不放地标
- 不放归藏出口

先看“坑炉下沉感”和“熔池炉心”成不成立，再补玩法层。
