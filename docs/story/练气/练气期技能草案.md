# 练气期技能草案

状态：草案 v1

适用范围：

- `realmLv 19-30`
- 应先结合 [练气期功法设计](./练气期功法设计.md) 阅读
- 对应 [练气期地图剧情草案](./练气期地图剧情草案.md)
- 对应 [练气期地图怪物草案](./练气期地图怪物草案.md)

## 1. 设计口径

练气期的技能继续按“功法带技能”去设计，不单独拆成纯技能池。

统一原则：

- 每门功法都要明显服务一条五行路线，而不是全都做成万金油
- 这些技能默认从属于“方向化功法”，玩家可以逐步广泛习得，但强度组织应靠路线联动而不是硬性禁学
- 每条路线统一围绕一个可叠层的共享 Buff 运作，技能效果直接受 Buff 层数影响
- 技能写成当前系统容易落地的类型：单体、直线、范围、自身增益、目标减益
- 练气期重点是“识脉、纳气、洗气、锻气、承气、合气”，不是继续重复凡人期的刀剑拳脚
- 五行图掉落的是分流功法，`归藏脉窟` 给最后的混行收束功法

## 2. 主城通用功法：听脉定息诀

定位：

- 练气期通用过渡内功
- 给所有玩家一个先稳住气海的起步选择

功法方向：

- 主加根骨、神识
- 小幅提升灵力恢复与灵力上限
- 不走偏科，适合刚进 `栖真渡` 时使用

技能设计：

| 技能 | 类型 | 作用 |
| --- | --- | --- |
| `平潮息` | 自身增益 | 短时间提升灵力恢复和化解，适合开怪前稳气 |
| `听脉步` | 自身增益 | 小幅提速并提高闪避，解决练气前段跑位问题 |
| `回渡护元` | 自身增益 | 获得一层短时护体，同时提高法防，适合抗第一轮伤害 |

## 3. 金行功法：庚华凝锋诀

来源：

- `裂锋原` 精英和首领掉落

定位：

- 单体爆发
- 破甲断势
- 适合近身压制与先手拆精英

功法方向：

- 主加体魄、身法
- 次加根骨
- 强调命中、破招、暴击联动

技能设计：

| 技能 | 类型 | 作用 |
| --- | --- | --- |
| `凝锋刺` | 单体 | 给目标叠加“锋痕”，作为金行路线的共享 Buff |
| `分砂断甲` | 直线 | 正面一线破开护体，并按“锋痕”层数额外削物防 |
| `断碑斩` | 单体重击 | 高额收头技，对“锋痕”层数越高的目标伤害越重 |

## 4. 木行功法：青木养脉篇

来源：

- `青萝谷` 精英和首领掉落

定位：

- 持续作战
- 养脉续航
- 适合单刷和长线拉扯

功法方向：

- 主加神识、根骨
- 次加悟性
- 强调生命恢复、灵力恢复、法防和持续压制

技能设计：

| 技能 | 类型 | 作用 |
| --- | --- | --- |
| `牵萝缠脉` | 单体减益 | 让目标叠加“缠脉”，并压低移速、闪避和化解 |
| `回青息` | 自身增益 | 一段时间内持续回复生命和灵力，攻击“缠脉”目标时回复量更高 |
| `枯荣逆抽` | 单体 | 对“缠脉”层数越高的目标抽取更多生命与灵力，越拖越强 |

## 5. 水行功法：寒汐洗脉法

来源：

- `寒汐泽` 精英和首领掉落

定位：

- 控场游斗
- 寒蚀减益
- 法修节奏流

功法方向：

- 主加神识、身法
- 次加悟性
- 强调冷却速度、命中、闪避和法术压制

技能设计：

| 技能 | 类型 | 作用 |
| --- | --- | --- |
| `寒汐印` | 单体 | 造成水行伤害并叠加“寒印”，压低目标命中和闪避 |
| `月井回波` | 直线 | 一道回卷水炁扫过前方，对“寒印”层数越高的目标减速越重 |
| `净潮封脉` | 范围 | 在小范围内压低敌人法防和冷却速度，并按“寒印”层数延长压制 |

## 6. 火行功法：离炎锻息诀

来源：

- `赤陨庭` 精英和首领掉落

定位：

- 快节奏爆发
- 压血收尾
- 高风险高收益

功法方向：

- 主加体魄、神识
- 次加身法
- 强调法攻、暴击、灵力输出速率和爆发收头

技能设计：

| 技能 | 类型 | 作用 |
| --- | --- | --- |
| `陨火烙脉` | 单体 | 给目标叠加“灼脉”，后续火行技能更容易打穿法防 |
| `赤线穿庭` | 直线 | 一线火炁贯穿前方，对“灼脉”层数越高的目标额外提高暴击收益 |
| `焚息诀` | 范围爆发 | 引爆周围目标身上的“灼脉”，层数越高结算越猛 |

## 7. 土行功法：厚土承元功

来源：

- `厚脉岭` 精英和首领掉落

定位：

- 承压站场
- 反打拆阵
- 适合硬抗精英与门口 Boss

功法方向：

- 主加体魄、根骨
- 次加悟性
- 强调生命、双防、化解和稳定输出

技能设计：

| 技能 | 类型 | 作用 |
| --- | --- | --- |
| `镇脉印` | 自身增益 | 短时间提高双防和化解，并让自身攻击更容易给目标叠加“镇脉” |
| `裂岭推山` | 直线 | 一道厚土劲向前压出，对带“镇脉”的目标额外压低移速和破招能力 |
| `承元落碑` | 范围重击 | 在前方砸下一记厚重镇击，对“镇脉”层数越高的目标伤害越重 |

## 8. 终图功法：归藏合脉经

来源：

- `归藏脉窟` 精英和首领掉落

定位：

- 五行混行
- 练气末段收束
- 不依赖单一五行标记

功法方向：

- 主加体魄、神识、根骨
- 次加悟性
- 强调物法双修、灵力循环、抗压和终段稳定性

技能设计：

| 技能 | 类型 | 作用 |
| --- | --- | --- |
| `合炁印` | 单体 | 在目标身上打出“杂相”标记，并把已有五行路线 Buff 汇成混行压制 |
| `归流轮转` | 直线 | 顺着主路放出一线轮转气流，对带“杂相”的目标附加额外压制 |
| `半基压窍` | 范围终结 | 对范围内所有带“杂相”的目标进行一轮强压，作为练气末段大招 |

## 9. 流派组合建议

如果玩家不准备纯修一门，可以这样搭：

- `听脉定息诀 + 庚华凝锋诀`：适合近战先手拆人
- `听脉定息诀 + 青木养脉篇`：适合稳态续航和单刷
- `寒汐洗脉法 + 青木养脉篇`：适合法修控场拖长线
- `离炎锻息诀 + 庚华凝锋诀`：适合高爆发收头
- `厚土承元功 + 任一五行功法`：适合作为抗压底盘
- `归藏合脉经`：建议作为 `练气十一层-练气十二层` 的收束主修，不建议太早拿来当纯起步功

## 10. 后续落地建议

如果下一轮要继续往配置走，建议顺序如下：

- 先落 `听脉定息诀`，保证 `19-21` 级不会出现功法断档
- 五行功法先各做 `2` 个核心技能，再补第三个收束技能
- `归藏合脉经` 要最后做，因为它最依赖前面五行路线 Buff 的口径统一
- 技能命名尽量和地图材料、装备名、怪物词汇保持同一套五行语言
