# 练气期地图剧情草案

状态：草案 v2

适用范围：

- `realmLv 19-30`
- 练气期到筑基前期的地图与剧情骨架
- 采用“短主线 + 五行支线主体 + 聚落中枢回收”的设计

## 1. 设计结论

练气期不再延续凡人篇那种长主线推图结构，而是改成：

- 1 个低阶修士据点
- 5 个五行主题区域
- 5 个独立 Boss 房
- 1 条只负责串场、解锁、收束的短主线

核心目标不是“再讲一大段线性主线”，而是让玩家在练气期真正体验到：

- 已经踏进修士圈子，但仍只是低阶修士
- 不同灵根面对不同五行灵气时，修炼收益与风险明显不同
- 练气修行的重点是识脉、纳气、调和、稳住气海
- 筑基不是单纯堆更多灵气，而是把练气期吸纳的五行灵气压成能承载的道基雏形

因此，练气期的主体内容应从“继续往更高危地图刷怪”转为：

- 围绕五行环境辨识地图
- 围绕灵根差异选择路线
- 围绕气脉冲突组织支线
- 围绕 Boss 房完成阶段收口

## 2. 与现有版本的衔接

当前版本主线已经做到：

- `灵脊岭 -> 练气境`
- `天穹残宫 -> 天封核心`
- `q_foundation_oath` 以“半步筑基”作为当前版本的终点占位

练气期新内容建议这样接上：

1. `天穹残宫` 仍保留为“练气开场楔子”
说明玩家已经见过更高层的破败真相，但还没有能力真正处理

2. `道基初誓` 不再只是版本终章文案
后续正式落地时，可把它改写成“清玄给出五脉集线索”的过渡任务

3. 新阶段的主题从“追查旧灾”转成“先把自己修成能承灾的人”
这能避免练气期继续重复凡人篇那种一路追真相的节奏

一句话概括衔接逻辑：

- 凡人篇解决“你有没有资格碰门”
- 练气期解决“你有没有本事在门后活下去”

## 3. 练气期的实现约束

这轮设计仍必须服从当前项目已经稳定的实现能力。

### 3.1 地图与任务约束

- 地图仍是静态 JSON 配置的格子地图
- 1Hz tick 下，不适合做超大空旷地图
- 主线和支线应继续以 `talk / kill / collect / realm_stage` 这类稳妥目标为主
- 不应把推进建立在护送、多阶段机关、实时地形重组、复杂潜入脚本上

### 3.2 因约束带来的设计口径

- `寒汐泽` 的“潮汐改路”先用雾桥岔路、浅滩高低区和文案表达，不依赖实时改图
- `青萝谷` 的“藤墙重生”先用多层环路、半封闭采集区和怪群阻挡表达，不依赖动态刷墙
- 五行 Boss 房的开门条件优先设计成：
  - 收集试印
  - 击杀区域精英
  - 回据点交付后再进入

也就是说，练气期设计要先做到“空间与叙事已经成立”，再为以后更复杂脚本预留想象空间。

## 4. 总体结构

### 4.1 阶段主据点更名

原暂定名：`灵枢城`

调整后名称：`五脉集`

调整理由：

- 练气期阶段太早，不适合直接使用“城”级命名
- 当前阶段更像乱世里由低阶修士、自保散修和小型坊市拼出来的据点
- “集”比“城”更符合练气玩家活动中心的体量
- 名称直接点出“五脉汇集”的功能定位，方便玩家理解这一阶段围绕五行展开

`五脉集` 的定位不是宗门山门，也不是完整仙城，而是：

- 练气修士的临时中枢
- 五行材料、低阶法器、修炼情报的交换点
- 玩家从凡人边镇转入修士活动圈后的第一个稳定据点

### 4.2 总地图关系

```text
云来镇 / 灵脊岭 / 天穹残宫旧线
            ↓
          五脉集
  ├─ 金：裂锋原
  │   └─ Boss 房：断兵冢
  ├─ 木：青萝谷
  │   └─ Boss 房：万藤心窟
  ├─ 水：寒汐泽
  │   └─ Boss 房：月井寒宫
  ├─ 火：赤陨庭
  │   └─ Boss 房：地火炉心
  └─ 土：厚脉岭
      └─ Boss 房：镇岳石府
```

建议理解：

- `五脉集` 负责集散、接支线、交材料、准备筑基
- 五个区域分别对应一条五行修炼线
- 每个区域最终都有一个 Boss 房作为阶段收口点
- 主线只要求玩家完成若干脉系试炼，不要求五区必须线性全清

## 5. 五脉集定位

### 5.1 剧情定位

天穹残宫坠落之后，周边地脉被撕成明显的五行失衡带。部分幸存修士围绕一处尚未断绝的脉眼，搭起一片供低阶修士调息、交易、换材和暂避灵灾的聚落，这里便是 `五脉集`。

它和云来镇的差异必须很明确：

- 云来镇是凡人的边镇据点
- 五脉集是练气修士的低阶据点
- 云来镇重在“活命”
- 五脉集重在“怎么修下去”

### 5.2 功能分区建议

- `引气台`
玩家进入练气期后的第一落点，用来解释灵根与五行纳气

- `五脉堂`
五行支线的主要接取与交付点，也是试印回收地点

- `百工坊`
五行材料换低阶法器、符箓与练气装备的区域

- `静室区`
修炼、调息、闭关相关文案与功能区

- `行修街`
散修摊位、残卷贩子、临时雇修聚集区，承担“多人修士据点”的烟火感

- `筑基台`
后期远景目标点，前期可见但不急着开放

### 5.3 关键 NPC 建议

| NPC | 身份 | 作用 |
| --- | --- | --- |
| 闻脉师·陆微 | 五脉集引路人 | 负责把玩家从“练气了”正式带入“五行修法” |
| 五脉执记·苏简 | 记录试印的文修 | 负责主线收束、统计试炼进度 |
| 百工老匠·郭火 | 炼材老匠 | 提供五行材料、法器、锻造口径 |
| 静室执事·许灯 | 调息管事 | 负责净气、闭关、稳脉方向的支线 |
| 守台人·石姑 | 筑基台守人 | 前期只给压力，不给立即突破的承诺 |

NPC 的口径重点：

- 不把玩家当天命主角
- 默认把玩家视为刚进修士圈的新人
- 更关心你脉象稳不稳、能不能活着回来，而不是你有没有豪言壮语

### 5.4 气质要求

- 不是仙门盛景，而是勉强维持秩序的低阶修士聚落
- 有坊市味、避难点味和前线补给点味
- 要保留乱世下的紧张感，不要做成安稳富足的仙城

## 6. 五个五行区域

## 6.1 金行区域：裂锋原

定位：

- 覆盖 `练气一层 -> 练气巅峰` 的完整金行线路
- 偏“收束、锐化、定形”
- 更适合走攻击、破甲、凝锋路线的玩家优先进入

环境关键词：

- 断裂石原
- 金砂风暴
- 残兵插地
- 被旧阵切开的锋痕地貌

空间结构建议：

- 外环是可绕行的碎岩带
- 中段是金砂横扫的开阔锋原
- 内圈收成残兵阵地
- 最深处接 `断兵冢`

怪物与资源建议：

- 兵魂、残甲傀、刃砂灵
- 产出偏金行矿砂、残兵铁片、锐化类材料
- 适合投放破甲、暴击、凝锋方向装备

Boss 房：`断兵冢`

- Boss：`庚金镇墓将`
- 关键词：高破防、高反击、硬碰硬惩罚明显
- 叙事作用：告诉玩家“金行不是只会打得更疼，而是先把灵气压成形”

## 6.2 木行区域：青萝谷

定位：

- 覆盖 `练气一层 -> 练气巅峰` 的完整木行线路
- 偏“生长、牵引、养脉”
- 更适合需要续航、回复、养脉的玩家先站稳

环境关键词：

- 古藤盘谷
- 灵木药圃
- 草木灵雾
- 层层攀附的谷壁与半封路藤墙

空间结构建议：

- 谷口为安全采药带
- 中层是多岔路藤桥
- 深处为被妖藤占住的药圃
- 最深处接 `万藤心窟`

怪物与资源建议：

- 寄生妖藤、木魅、吸灵花妖
- 产出偏木髓、药藤、养脉叶
- 适合投放回复、气脉恢复、修炼加成类装备

Boss 房：`万藤心窟`

- Boss：`噬脉藤母`
- 关键词：召唤藤蔓、持续吸灵、场地封锁
- 叙事作用：告诉玩家“木行的危险不是不够强，而是会慢慢把你缠死”

## 6.3 水行区域：寒汐泽

定位：

- 覆盖 `练气一层 -> 练气巅峰` 的完整水行线路
- 偏“流转、洗练、化解”
- 是五脉阶段最重要的“调和地图”

环境关键词：

- 寒泉浅泽
- 月照水道
- 雾桥
- 冷潮浸骨的低洼湿地

空间结构建议：

- 外层是可摸索的浅泽与雾桥
- 中层做成双路水道
- 内层收成月井周边的静冷区域
- 最深处接 `月井寒宫`

实现口径说明：

- “潮汐改路”先靠浅滩、石桥和雾区做路径观感
- 不要求真实动态涨落

怪物与资源建议：

- 寒泽螭影、冰纹水鬼、月沼灵
- 产出偏寒髓、水魄、净脉露
- 适合投放减伤、净化、真气回复、异常抗性类装备

Boss 房：`月井寒宫`

- Boss：`玄溟螭影`
- 关键词：减速、冻结、幻身换位、控制压力高
- 叙事作用：负责把“你会吸气”推进到“你会不会把杂气洗干净”

## 6.4 火行区域：赤陨庭

定位：

- 覆盖 `练气一层 -> 练气巅峰` 的完整火行线路
- 偏“爆发、淬体、炼气”
- 是高风险高收益地图

环境关键词：

- 坠火坑
- 熔裂石庭
- 焦土廊道
- 热浪翻涌的高温区

空间结构建议：

- 外围是可短暂停留的焦土带
- 中层是被熔流切开的庭廊
- 内圈是高密度精英区
- 最深处接 `地火炉心`

怪物与资源建议：

- 炽骨傀、火纹蜥、陨火侍灵
- 产出偏火晶、炎髓、淬体灰
- 适合投放爆发、灼烧、攻速、短时间强化类装备

Boss 房：`地火炉心`

- Boss：`离火炉君`
- 关键词：高爆发、灼烧叠层、阶段性全场升温
- 叙事作用：把“练气”从单纯纳气推进到“能不能承受气入体后的烈性”

## 6.5 土行区域：厚脉岭

定位：

- 覆盖 `练气一层 -> 练气巅峰` 的完整土行线路
- 偏“承载、镇压、固脉”
- 是练气期收尾最关键的“承基地图”

环境关键词：

- 黄岩台地
- 塌陷地脉
- 石柱阵
- 重压型狭道与坠岩区

空间结构建议：

- 山脚是较开阔的采脉区
- 中层是重压狭道与坠石道
- 高处是石柱阵台地
- 最深处接 `镇岳石府`

怪物与资源建议：

- 岩甲傀、地脉鼍、镇纹石灵
- 产出偏土魄、承脉石、镇压类材料
- 适合投放防御、稳脉、护盾、抗打断类装备

Boss 房：`镇岳石府`

- Boss：`玄岳地傀`
- 关键词：高血高抗、震地击退、土墙与落石压制
- 叙事作用：明确告诉玩家，筑基前最重要的不是“吸得更多”，而是“装得下”

## 6.6 五行线路铺怪硬规则

为满足“每一行都能从练气初段一路打到练气巅峰”的要求，五行图必须统一遵守以下规则：

- 每条五行线都完整覆盖 `realmLv 19-28`，即 `练气一层 -> 练气巅峰`
- `realmLv 29-30` 的 `练气大圆满 / 半步筑基` 不再放在普通野外刷怪层，而放在 Boss 房、门前精英和收束任务里
- 每条线的怪物强度必须按路径推进递增，不能出现“前面是练气二层，拐个弯突然跳到练气八层”的断档
- 每条线都保留自己的路线语言，不强行做成同一套刷图模板

推荐按当前运行口径拆成 6 个梯段：

| 梯段 | 对应境界 | 用途 |
| --- | --- | --- |
| T1 | `realmLv 19-20` | 入门适应段 |
| T2 | `realmLv 21-22` | 第一轮成形段 |
| T3 | `realmLv 23-24` | 中段关口段 |
| T4 | `realmLv 25-26` | 高压精英段 |
| T5 | `realmLv 27-28` | 巅峰前夜段 |
| T6 | `realmLv 29-30` | Boss 房与半步筑基收口段 |

| 土：厚脉岭 | 层台递进。先横向绕山，再纵向登高，最后压进石府，强调“越往上越重”。 | `黄岩鼍兽`、`沉脉石奴` | `坠岩甲灵`、`土纹守卒` | `镇脉岩魈`、`厚土负碑者` | `地脉石傀`、`裂岭重卫` | `岳纹镇灵`、`玄垣岩将` | `镇岳石府` 放 `府门镇脉兽`、Boss `玄岳地傀` |

### 6.7.1 各线的路线语言要求

- 金线必须干脆，视野开阔、岔路少、怪物正面拦路多，打起来像在顶着锋压硬推
- 木线必须缠绕，岔路、短环和半封闭区多，玩家会反复“看似靠近，实际被带偏”
- 水线必须旋进，地图记忆点应围绕“这一圈比上一圈更冷、更静、更危险”
- 火线必须层层下压，路径像在往炉里走，回头路短，但每次切层都更灼烈
- 土线必须先绕后升，前半段像找路，后半段像扛压，越靠近石府越沉重

### 6.7.2 怪物分布密度建议

- `19-22` 以普通怪为主，让玩家先熟悉这一行的节奏与词条
- `23-24` 开始加入区域精英，作为本线中段关口
- `25-26` 明显提高精英占比，让玩家感到“这条线开始咬人”
- `27-28` 的普通怪数量可以略降，但每一组都更有压迫感
- `29-30` 只保留 Boss 房守卫、门前精英和最终首领，不再做野外大面积散刷

## 7. 五脉试炼的主线组织方式

练气期主线必须明显短于凡人篇，只保留以下功能：

1. 把玩家从 `天穹残宫` 和 `道基初誓` 引入 `五脉集`
2. 让玩家理解“练气期围绕五行灵气展开”
3. 引导玩家按自身灵根优先选择一条本命脉
4. 在完成若干脉系试炼后，把目标收束到筑基准备

### 7.1 主线里程碑

建议主线只保留 4 个阶段：

1. `入集`
说明为什么练气玩家要离开云来镇，转入 `五脉集`

2. `识脉`
由 NPC 明确点出灵根与五行纳气差异，并让玩家确认自己的本命脉方向

3. `试修`
要求玩家完成若干五行区域试炼，而不是一条线打到底

4. `收基`
把练气末段收束到筑基台与半步筑基准备

### 7.2 试炼印记规则建议

为避免练气期重新变成线性清图，建议主线使用“三印收基”结构：

- `本命印`
来自玩家优先选择的一条五行区域

- `调和印`
来自偏净化、续航、平衡的区域，优先由 `木 / 水` 提供

- `承基印`
来自偏承载、淬炼、定形的区域，优先由 `金 / 火 / 土` 提供

这样做的好处：

- 主线只要求 `3` 枚关键印记，不要求五区全清
- 其余区域可自然成为支线、装备、材料和流派补完内容
- 玩家会觉得自己是在“组装一套适合自己的气海结构”，而不是照着地图列表打卡

## 8. 章节骨架建议

这里不再使用凡人篇那种长章节推图，而是把练气期写成《寻仙录》的阶段节点。

## 8.1 第一节点：入集听脉

建议对应区间：`realmLv 19-20`

主要地图：

- `天穹残宫`
- `五脉集`

核心内容：

- 玩家带着 `天封核心` 或其余波线索，被引到 `五脉集`
- 闻脉师判断玩家虽已练气，却气海驳杂，不能直接往筑基冲
- 玩家第一次听懂“本命脉、调和脉、承基脉”这套说法

任务节点建议：

- 去五脉集见闻脉师·陆微
- 在引气台完成一次五脉观测
- 和五脉执记·苏简对话，领取第一份试修簿
- 在 `裂锋原 / 青萝谷` 中二选一作为初试区域

## 8.2 第二节点：本命试修

建议对应区间：`realmLv 20-23`

主要地图：

- `裂锋原`
- `青萝谷`
- 对应 Boss 房

核心内容：

- 玩家先走与自身灵根最契合的一条线
- 第一次体验“环境适配”与“区域气质”对成长路线的影响
- 打下第一枚 `本命印`

任务节点建议：

- 清区域怪并收对应材料
- 回 `五脉集` 交材料，换 Boss 房入场凭据
- 击败区域 Boss，拿到 `本命印`

## 8.3 第三节点：调和三印

建议对应区间：`realmLv 21-28`

主要地图：

- `寒汐泽`
- `赤陨庭`
- `厚脉岭`
- 玩家尚未走过的另外若干区域

核心内容：

- 主线要求玩家再拿到 `调和印` 与 `承基印`
- 支线厚度主要由五行区域各自承担
- 玩家会逐渐意识到：自己不是在“学一门功法”，而是在把整个气海打磨成稳定结构

任务节点建议：

- `寒汐泽` 负责解决杂气冲突
- `赤陨庭` 负责解决烈性淬炼
- `厚脉岭` 负责解决承载问题
- 其余未走区域提供补完路线、材料与流派奖励

## 8.4 第四节点：聚脉收基

建议对应区间：`realmLv 28-30`

主要地图：

- `五脉集`
- `筑基台`
- 可选再回一张玩家最关键的五行地图做最终补材

核心内容：

- 玩家把三印交回五脉堂
- 五脉执记判断其气海结构是否勉强成形
- 守台人明确告知：半步筑基只是“能开始立基”，不是已经立成

任务节点建议：

- 交三印
- 补最后一批聚脉材料
- 在筑基台完成“半步筑基”收束任务

这里的情绪重点不是大胜，而是：

- 你终于有资格动手立基
- 但前面的路比凡人篇更真、更重，也更容易走偏

## 9. 五行支线承担的核心内容

练气期的内容厚度主要由五行支线承担，而不是主线承担。

支线应重点表达：

- 你能不能感到这股五行灵气
- 你能不能把它吸进体内
- 吸进去后会不会和已有气脉冲突
- 你要为这条脉付出什么代价
- 你最后把自己修成什么样的气海结构

因此五行支线的重点不是普通跑腿，而应是：

- 环境辨识
- 灵气适配
- 调息与取舍
- 脉系成长
- 区域 Boss 收口

### 9.1 每条五行支线都应自带的叙事句式

- 金：你能不能把散气收成锋
- 木：你能不能让气脉活下来
- 水：你能不能把乱气洗干净
- 火：你能不能扛住烈气入体
- 土：你能不能让气海承得住

这样玩家在看地图、掉落和 NPC 说话时，会更容易记住每张图到底在修什么。

## 10. 为什么采用这套结构

- 阶段辨识清楚：玩家会明确感觉自己已从凡人地图进入修士地图
- 地图组织稳定：1 个据点辐射 5 个区域，天然适合多人分流
- 内容投放明确：每个区域都能挂自己的材料、怪群、功法、装备和支线
- 练气味道更正：重点从“继续刷怪升级”转为“围绕灵根和灵气修行”
- 对长期任务友好：主线短，支线厚，符合《寻仙录》阶段性收束的结构
- 筑基更好承接：五行区域的成长结果可以自然回收到筑基准备
