## 1. v1 问题诊断

### 1.1 形状雷同

三张战斗图（霜刃渊、焚木荒台、玄壤深渊）使用几乎相同的模板：
- 椭圆形边界（逗号围成的碎石带）
- 中轴直线道路（`==`）贯穿南北
- 上下各一个传送点，位于中轴线上
- 左右对称布局

玩家进入任何一张图的体验几乎一致，无法通过空间感知区分"我在哪"。

### 1.2 空间叙事缺失

设计文档写了"外围适应区→中段压力区→深处高压区"三段递进，但实际地图：
- 没有瓶颈通道标记分区边界
- 没有地形宽窄变化暗示难度递增
- 没有视觉锚点（landmark 虽有文本但地形上无对应结构）

### 1.3 地形元素无功能

`铁铁`、`~~`、`火火`、`木木` 只是散落的 2 格装饰块：
- 不构成路径引导（玩家不需要绕行）
- 不形成视线遮挡（不影响战术）
- 不划分区域（不标记分区边界）

### 1.4 对比练气期差距

练气期的厚脉岭（deepvein_ridge）、灵脊岭（spirit_ridge）：
- 路径蜿蜒，有死角、有分支、有需要绕行的障碍
- 多种地形混合使用（崖壁、矿脉、泥地、小径、水域）
- 传送点分布在不同方位，不是简单的上下对称
- 空间有明确的"入口→过渡→核心"叙事

## 2. 设计哲学

### 2.1 核心原则：每张图必须有独特的空间签名

玩家闭眼回忆时能说出"霜刃渊是那个越走越窄的裂谷"、"焚木荒台是被地火裂缝切成碎片的荒原"、"玄壤深渊是要在洞穴网络里找路的迷宫"。

三张图的空间形态必须完全不同：

| 地图 | 空间形态 | 主轴方向 | 宽窄变化 | 路径特征 |
| --- | --- | --- | --- | --- |
| 霜刃渊 | 纵向裂谷 | 从上到下（由浅入深） | 入口宽→中段收窄→底部略开 | 主干道+侧洞 |
| 焚木荒台 | 横向荒原 | 从左到右（由外向内） | 整体开阔但被裂缝切割 | 多条平行路径 |
| 玄壤深渊 | 洞穴网络 | 无明确主轴 | 通道窄、房间宽 | 多分支多死角 |

### 2.2 分区必须可感知

每张图分三段，段与段之间必须有明确的地形瓶颈：
- 瓶颈可以是：窄通道（2-3格宽）、桥梁、阶梯、需要穿越的障碍带
- 进入新区域后地形风格应有可见变化（地面材质、障碍密度、空间开阔度）
- 怪物种类和密度在分区边界处有明显跳变

### 2.3 路径非线性

- 禁止全图只有一条直线主路
- 每张图至少有 2 条可选路径到达深处
- 死角和侧洞用于放置精英怪或高品级采集点（奖励探索）
- 主路可以稍宽（3-4格）、支路窄（1-2格）

### 2.4 地形元素必须有功能

每个地形块的放置必须满足以下至少一条：
1. **路径引导** — 迫使玩家绕行，形成路线选择
2. **视线遮挡** — 影响战术（绕过石柱可能遇到精英怪）
3. **分区标记** — 标识区域边界（如一排冰壁标记进入深层）
4. **资源承载** — 矿脉/采集点本身就是地形的一部分
5. **氛围营造** — 成片使用形成环境感（如大面积寒沼而非零星两格）

### 2.5 主城功能分区

云墟台不再是对称椭圆，改为功能分区布局：
- 北区：传送广场（天梯回阶）
- 中区：主街（杂货、丹药、材料商）
- 东区：炼器坊
- 西区：炼丹阁
- 南区：战斗图入口（霜刃渊裂口）
- 预留：宗门据点区域

## 4. 云墟台（yunxu_terrace）重设计

### 4.1 定位

筑基修士中立主城。悬于云层之上的石台，四周云海断崖。功能齐全的补给站和交易中心。

### 4.2 尺寸与形状

- 尺寸：50×45（从 60×60 缩小，去掉大量无用云层填充）
- 形状：不规则多边形，北宽南窄（北面是传送广场，南面收窄为通往霜刃渊的裂口）
- 边界：`云`（cloud）围成，不再是完美椭圆

### 4.3 功能分区布局

```
        ┌─────────────────────────┐
        │    北区·传送广场         │
        │        [天梯回阶]         │
        ├─────────────────────────┤
        │         主街             │
   ┌────┤  杂货  丹药  材料        ├────┐
   │西区│         ↕               │东区│
   │炼丹│                         │炼器│
   │阁  │      中央广场            │坊  │
   └────┤   （出生点/集合点）       ├────┘
        │                         │
        │      南区·裂口           │
        │    [霜刃渊入口]          │
        └────────┐     ┌──────────┘
                 │裂口 │
                 └─────┘
```

### 4.4 设计要点

- 主街用 `=`（road）铺设，移动快速
- 建筑用 `#`/`壁`/`角`/`檐` 等现有建筑字符
- 广场用 `.`（floor），周围用 `,`（grass）过渡
- 云海边缘用 `霞`（cloud_floor）做 1-2 格过渡带再接 `云`
- NPC 位置与功能分区对应，不再随意散落
- 南区裂口做成明显的地形收窄（从 20+ 格宽收到 4-5 格），暗示"前方危险"

## 5. 霜刃渊（frostblade_abyss）重设计

### 5.1 定位

筑基前期（31-34）战斗图。金水双行。纵向裂谷，越深越窄越冷。

### 5.2 空间叙事

玩家从顶部裂口进入，沿裂谷向下深入。裂谷由宽变窄，再在底部略微展开为一个冰窟。

```
入口（北）— 宽阔的裂谷上层，碎冰散落，视野开阔
    ↓
第一瓶颈 — 两侧冰壁收窄为 3 格通道
    ↓
中层（中）— 裂谷变窄但有侧洞分支，铁矿脉嵌入壁中
    ↓
第二瓶颈 — 冰桥横跨裂缝，只有 2 格宽
    ↓
深层（南）— 底部冰窟，略微开阔，寒沼遍布
    ↓
出口 — 冰窟尽头的裂缝通往焚木荒台
```

### 5.3 尺寸与形状

- 尺寸：40×60（窄长，强调纵深感）
- 形状：上宽下窄的不规则裂谷
  - 入口区宽约 24 格
  - 中层宽约 14-18 格
  - 深层宽约 20 格（底部冰窟展开）
- 边界：`崖`（cliff）为主，局部用 `铁`（矿脉嵌入崖壁）

### 5.4 三段分区

**上层·裂谷入口（y: 3-18）**
- 地面：`.` 为主，散落 `,`（碎冰碎石）
- 障碍：零星 `o`（落石），不密集
- 路径：较宽（12-20格），主路 `=` 偏左蜿蜒
- 怪物：寒铁蝎（普通）×4-5，分散在两侧
- 采集：1-2 个铁矿脉嵌入东壁

**中层·矿脉窄道（y: 20-38）**
- 地面：`.` + `;`（泥地增多，移速下降）
- 障碍：`铁` 矿脉从两壁突出，`o` 岩柱分割路径
- 路径：主路收窄为 4-5 格，东侧有一条 2 格宽的侧洞（通往精英怪）
- 瓶颈入口：两侧 `崖` 收窄到 3 格，地面变为 `:` 小径
- 怪物：寒铁蝎×2 + 霜鳞蛇×3（中层主力）
- 采集：3-4 个铁矿脉，1 个灵石矿脉（侧洞深处）

**深层·冰窟（y: 40-57）**
- 地面：`寒`（cold_bog）大面积覆盖，`.` 只在少数高台
- 障碍：`o` 冰柱密集，形成迷宫感
- 路径：无明确主路，需要在冰柱间穿行，`寒` 地面大幅减速
- 瓶颈入口：冰桥（2格宽 `=` 横跨 `空` void 裂缝）
- 怪物：冰晶蛛×3 + 铁壳龟（精英）×2
- 采集：2 个高品级铁矿脉
- 出口：南壁裂缝（传送至焚木荒台）

### 5.5 侧洞设计

中层东壁有一个 8×6 的侧洞：
- 入口只有 2 格宽（`:` 小径）
- 内部有 1 个灵石矿脉 + 1 个精英怪刷新点
- 死角设计：进去容易出来要绕，精英怪守在出口附近
- 奖励探索行为

## 6. 焚木荒台（blazewood_waste）重设计

### 6.1 定位

筑基中期（35-38）战斗图。火木双行。横向荒原，被地火裂缝切割成多个区块。

### 6.2 空间叙事

玩家从西侧入口进入一片焦灼荒原。地面被纵向的地火裂缝切成若干"岛"，玩家需要找到窄桥或绕行路径在岛间穿越，逐步向东深入。

```
入口（西）— 相对安全的焦土平台，残木稀疏
    →
第一裂缝带 — 3条纵向熔岩裂缝，需找桥通过
    →
中央荒原 — 最大的开阔区，焦木残桩密集，精英怪出没
    →
第二裂缝带 — 裂缝更宽更密，桥更少
    →
深处·焚心台 — 东端高台，Boss 领地
    →
出口（东壁下方裂缝）— 通往玄壤深渊
```

### 6.3 尺寸与形状

- 尺寸：65×45（横宽，强调水平展开）
- 形状：不规则横向带状，上下边界参差不齐
  - 北边界凹凸（崖壁+焦木带）
  - 南边界较平直（深渊边缘）
- 边界：北用 `崖`+`T`（焦木崖壁），南用 `空`（深渊边缘）

### 6.4 核心机制：地火裂缝

- 裂缝用 `熔`（molten_pool，移动消耗 800）铺设，宽 2-3 格
- 裂缝纵向贯穿地图（从北到南），将地图切成 3-4 个纵向条带
- 每条裂缝上有 1-2 座窄桥（2格宽 `=` road）
- 桥的位置不在同一水平线上，迫使玩家上下移动寻找通路
- 裂缝不是完全连续的，有些地方断开（可以从断口绕过，但路更长）

### 6.5 三段分区

**外围·焦土平台（x: 2-18）**
- 地面：`,`（焦土）为主，`.` 散落
- 障碍：`T`（焦木残桩）稀疏分布，不阻路
- 路径：开阔，主路 `=` 从入口向东延伸
- 怪物：焦木蜥（普通）×3，火行，分散巡逻
- 采集：1-2 个木行采集点（枯木残根）

**中央·裂缝荒原（x: 20-45）**
- 地面：`.` + `^`（丘陵起伏）+ `;`（灰烬泥）
- 障碍：`T` 焦木残桩密集（成片 3-5 个形成视线遮挡），`o` 焦岩
- 裂缝：2 条主裂缝纵贯，`熔` 铺设
- 路径：需要在裂缝间找桥，桥附近是争夺热点
- 怪物：焦木蜥×2 + 赤焰蜂（普通）×3 + 枯藤巨蟒（精英）×2
- 采集：火行+木行材料散布在各岛上

**深处·焚心台（x: 47-63）**
- 地面：`^`（高台）+ `熔`（台下熔岩）
- 障碍：`o` 焦岩柱环绕高台
- 路径：只有一座宽桥（3格）通往高台，桥两侧是 `熔`
- 怪物：焚心蛾王（Boss）×1 坐镇高台 + 赤焰蜂×2 护卫
- 采集：高品级火行材料（高台边缘）
- 出口：高台东壁下方裂缝（传送至玄壤深渊）

### 6.6 多路径设计

从入口到焚心台至少有 3 条路线：
1. **北线**：沿北壁走，桥在偏北位置，路程短但焦木密集（视线差）
2. **中线**：走中央主路，桥位置居中，最直观但怪物最密集
3. **南线**：沿南边缘走，需要在深渊边缘行走（`空` void 旁边），路窄但怪少

## 7. 玄壤深渊（darksoil_abyss）重设计

### 7.1 定位

筑基后期（39-42）战斗图。土行+混合。洞穴网络，多通道多房间。

### 7.2 空间叙事

玩家从西北入口进入一个庞大的地下洞穴系统。洞穴由多个大小不一的"厅"通过窄通道连接，没有明确的单一路径，需要探索才能找到最深处。

```
入口（西北）— 从焚木荒台裂缝落入的第一个洞厅
    ↓↘
分支 — 通道分叉，左通向矿脉厅，右通向蘑菇洞
    ↓
中央大厅 — 最大的洞穴空间，多条通道汇聚于此
    ↓↙↘
深层分支 — 左通向地脉泉（Boss1），右通向封印石室（Boss2）
    ↓
最深处 — 封印入口（预留金丹期衔接，不实装）
```

### 7.3 尺寸与形状

- 尺寸：55×55（接近正方，强调网络感而非方向性）
- 形状：多个不规则"房间"通过 2-3 格宽通道连接
- 边界：`o`（stone 洞壁）为主，局部 `崖`（坚硬岩层）

### 7.4 核心机制：房间+通道网络

地图由以下元素组成：
- **房间**（厅）：8×8 到 14×14 不等的开阔空间，地面为 `.` 或 `;`
- **通道**：2-3 格宽、5-12 格长的连接段，地面为 `:` 或 `;`
- **死角**：只有一个出口的小房间，放置精英怪或稀有采集点
- **岔路口**：通道分叉处，玩家需要选择方向

### 7.5 房间布局

```
    [入口厅]
     /    \
[矿脉厅]  [蘑菇洞]
     \    /
   [中央大厅]
    / |  \
[地脉泉] [过渡厅] [封印石室]
 (Boss1)    |      (Boss2)
         [深渊底]
         (预留)
```

**入口厅（12×10）**
- 位置：西北角
- 地面：`.` 为主，入口处有 `梯`（石阶从上方落入）
- 特征：洞顶有裂缝透光（描述性，不影响地形），相对安全
- 传送点：回焚木荒台

**矿脉厅（10×10）**
- 位置：西侧
- 地面：`.` + `;`（泥地）
- 特征：四壁嵌满 `铁` 和 `L` 矿脉，是主要采集区
- 怪物：穿山甲兽（普通）×3 守护矿脉
- 通道：从入口厅向南 1 条，从中央大厅向西 1 条

**蘑菇洞（8×8）**
- 位置：东北
- 地面：`;`（潮湿泥地）+ `%`（沼泽）
- 特征：木行采集点集中区，地面减速明显
- 怪物：毒菇蟾（普通）×2
- 通道：从入口厅向东南 1 条

**中央大厅（14×14）**
- 位置：地图中心偏南
- 地面：`.` 为主，中央有 `^`（土丘/石台）
- 特征：最大空间，多条通道汇聚，是交通枢纽
- 怪物：穿山甲兽×2 + 岩脊蜈蚣（精英）×1
- 通道：向北2条（通矿脉厅、蘑菇洞），向南3条（通地脉泉、过渡厅、封印石室）

**地脉泉（10×12）**
- 位置：西南
- 地面：`;` + `%`（泥泞）+ 中央 `~`（地下泉水）
- 特征：Boss1 领地，泉水环绕中央高台
- 怪物：玄土蟒王（Boss）×1
- 通道：从中央大厅向西南 1 条（3格宽，较好走）

**封印石室（10×10）**
- 位置：东南
- 地面：`.`，四壁 `#`（人工墙体，暗示远古修士建造）
- 特征：Boss2 领地，石室内有阵纹（描述性）
- 怪物：石傀儡（Boss）×1
- 通道：从中央大厅向东南 1 条（2格宽，需要穿过一段 `;` 泥地）

**深渊底（6×6）**
- 位置：正南最深处
- 地面：`.`
- 特征：预留金丹期入口，当前只有一个地标描述"封印的裂缝"
- 通道：从过渡厅向南 1 条（2格宽）

### 7.6 通道特征

- 通道不是直线，有 1-2 个拐角
- 部分通道有 `;`（泥地）减速
- 关键通道入口有 `o`（石柱）标记，帮助玩家辨认方向
- 死角通道比主通道更窄（2格 vs 3格）
