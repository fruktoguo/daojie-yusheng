# 筑基期地图设计

状态：草案 v1

适用范围：

- `realmLv 31-42`
- 筑基前期 31-34 / 筑基中期 35-38 / 筑基后期 39-42
- 采用"主城 + 三张线性递增战斗图"的结构
- 入口从天梯尽头接入主城

## 1. 设计结论

筑基期不再延续练气期的五行任选结构，改为线性难度递增：

- 1 个筑基修士中立主城
- 3 张线性解锁的战斗图，难度逐级递增
- 五行覆盖分散在三张图中（金水 / 火木 / 土+混合），保证总体五行齐全但不再追求平均
- 怪物全部为妖兽
- 不掉装备、不掉功法书，只掉带五行值的材料
- 采集点品级比练气期升一档

核心目标：

- 玩家已经立基成功，开始沉淀根基、建立宗门
- 战斗不再是为了获取功法和装备，而是为了获取材料（炼丹、炼器、宗门建设）
- 自由度进一步提高，主线不再强制推进，玩家按自身节奏刷图积累

一句话概括：

- 凡人篇解决"你能不能碰到门"
- 练气期解决"你进门以后会不会先把自己修坏"
- 筑基期解决"根基立住以后，你能不能撑起一方道场"

## 2. 与现有版本的衔接

当前版本已经做到：

- 归藏脉窟击败"未成道基"后推到 `realmLv 30`（半步筑基）
- 天梯从归藏脉窟道基古阵旁接出，终点描述为"筑基主城的方向已在云外露出轮廓"
- 天梯 `heaven_ladder` 尺寸 61x7，终点坐标 (57,3)

衔接方式：

1. 天梯尽头增加传送点，目标为筑基主城 `yunxu_terrace`
2. 主城设回程传送点到栖真渡（方便玩家回练气期区域）
3. 三张战斗图从主城线性解锁，每张图有回主城传送点

## 3. 实现约束

### 3.1 地图与玩法约束

- 地图仍是静态 JSON 配置的格子地图
- `1Hz tick` 下不做超大空旷图
- 玩法系统已完成，本次只做地图、采集、矿脉、怪物
- 不新增玩法机制
- 怪物掉落只给材料，材料必须带五行值字段

### 3.2 设计口径

- 三张战斗图都做成"外围适应区 + 中段压力区 + 深处高压区"
- 每张图 3-5 种怪物
- 每张图有两个传送点：一个回主城，一个通往下一张图（最后一张图只有回主城）
- 矿脉沿用灵石矿脉和玄铁矿脉，采集点按五行和品阶分配
- 所有材料带五行值，供炼丹炼器系统消耗

## 5. 主城：云墟台

### 5.1 地图定位

- 关键词：云上、中立、聚散、道场前站
- 天梯尽头的筑基修士中立聚集地
- 各方势力在此设有据点但互不干涉，是筑基修士交易、补给、建宗前的最后落脚点
- 比栖真渡更正式，但仍有边地感——不是仙城，是云上的修士集散台地

### 5.2 环境风格

- 整体是一座悬浮在云层之上的巨型石台
- 建筑半悬半嵌于崖壁，雾气从台地边缘翻涌
- 中心区域开阔平整，四周是悬崖和云海
- 石台上有明显的功能分区：商铺区、修炼区、宗门指引区、传送区
- 光照偏亮但带薄雾，给人"高处不胜寒但尚可栖身"的感觉

## 6. 第二张图：霜刃渊（金+水）

### 6.1 地图定位

- 关键词：寒锋、裂谷、冰脉、金铁
- 筑基前期的入门战斗图
- 以普通怪为主，少量精英散布在深处
- 让刚立基的玩家适应筑基期怪物强度

### 6.2 环境风格

- 高原裂谷地形，寒风如刃穿谷而过
- 谷壁裸露金属矿脉，反射冷光
- 谷底有冰脉暗河，水汽凝霜覆盖地表
- 越深处裂谷越窄，寒气越重，矿脉越密
- 空间语言：`切进去`——像被利刃劈开的山体，玩家沿着刀口往里走

### 6.3 空间结构

- `霜石外坪`（入口区）

- `裂谷主道`（中段）
  主要刷怪区，普通怪密集分布，金属矿脉和冰脉交替出现

- `寒铁深壑`（深处）
  精英怪出没区，矿脉品质更高，地形更窄更危险

### 6.4 传送点与等级坡度

## 9. 材料五行值规范

所有筑基期材料必须携带五行值字段，供炼丹炼器系统消耗。

### 9.1 五行值结构

```typescript
interface MaterialElementValues {
  metal: number;   // 金
  wood: number;    // 木
  water: number;   // 水
  fire: number;    // 火
  earth: number;   // 土
}
```

### 9.2 五行值分配原则

- 单属性材料：主属性值高（80-100），其余为 0 或极低（0-5）
- 双属性材料：两个属性各占 40-60，其余为 0
- 混合属性材料：五行均有值但分布不均，总值较高

### 9.3 品级与五行值总量对应

| 品级 | 单属性主值 | 双属性各值 | 混合总值 |
| --- | --- | --- | --- |
| mystic | 30-50 | 20-30 each | 60-80 |
| earth | 50-80 | 35-50 each | 80-120 |
| heaven | 80-120 | 55-75 each | 120-180 |

## 10. 难度设计原则

### 10.1 怪物强度递增

- 霜刃渊（31-34）：普通怪约等于练气期精英怪强度，精英怪需要认真打
- 焚木荒台（35-38）：普通怪需要注意走位，精英怪需要消耗品辅助，Boss 需要充分准备
- 玄壤深渊（39-42）：普通怪已有一定威胁，精英怪是硬仗，Boss 是筑基期顶级挑战

### 10.2 怪物数量分配

| 地图 | 普通怪种类 | 精英怪种类 | Boss种类 | 普通:精英:Boss 数量比 |
| --- | --- | --- | --- | --- |
| 霜刃渊 | 3 | 1 | 0 | 85:15:0 |
| 焚木荒台 | 2 | 2 | 1 | 45:45:10 |
| 玄壤深渊 | 1 | 2 | 2 | 30:40:30 |

### 10.3 刷新与分布

- 普通怪：count 3-5，radius 5-6，respawnSec 15-20
- 精英怪：count 1-2，radius 3-4，respawnSec 30-45
- Boss：count 1，radius 2，respawnSec 120-180
