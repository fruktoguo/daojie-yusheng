# 道劫余生知识库-剧情任务地图

本文档用于智能体检索《道劫余生》当前仓库中的剧情、主支线、地图推进与世界结构信息。  
口径只基于仓库真源，不扩写未实装剧情。

## 2. 世界主线脉络（检索关键词：道劫、旧灾、旧阵、漏口、封天核心）

当前主线叙事骨架是：云来镇在外患（匪、兽、怪）之下维持脆弱秩序，而更深层危机来自“更早的旧灾/旧阵崩坏外泄”。  
玩家从“活命跑腿”逐步进入“矿洞与遗迹调查”，再推进到“兽谷与灵岭叩门”，最终触及“天穹残宫封柱核心”，并在终章明确：这只是道劫外泄的第一层，主线尚未完结。

可概括为一条连续因果线：

1. 云来镇求生与北井异动（序章）
2. 荒野粮路与镇外生存权（第一章）
3. 青竹林追线，确认线索指向矿洞/遗迹（第二章）
4. 玄铁矿洞 + 断碑遗迹，旧纹与门钥浮出（第三章）
5. 噬魂兽谷 + 灵脊岭，凡武尽头叩门（第四章）
6. 天穹残宫夺回封柱核心（第五章）
7. 道劫未尽，目标转入半步筑基（终章）

## 3. 章节推进（主线）

## 序章·断道入镇（8 条）

- 核心推进：入镇落脚 -> 学会首门功法 -> 清理门外与流匪 -> 听到无名伤者线索 -> 北井异动 -> 达到炼体门槛。
- 关键地图：`yunlai_town`（主）、`wildlands`（结尾突破目标）
- 代表任务：`先进镇里`、`活命先练气`、`北井别回头`、`先把脚跟站住`

## 第一章·粮路讨生（6 条）

- 核心推进：围绕南门荒野粮路、生计与治安展开，逐步将“跑腿求生”转成“可持续镇外行动”。
- 关键地图：`wildlands`、`yunlai_town`
- 代表任务：`荒野先认路`、`路上要过买命钱`、`药铺要的不是可怜`、`把骨头练硬些`

## 第二章·青竹追线（6 条）

- 核心推进：进入青竹林运材线，出现“先救人再追问”的调查节奏，明确后续线索通往矿洞与遗迹。
- 关键地图：`bamboo_forest`
- 代表任务：`林口有人等你`、`翻车留下的东西`、`竹隐客的路`、`从林路走到矿路`

## 第三章·矿洞遗火（6 条）

- 核心推进：矿洞与遗迹双线并进，夺取信标/门钥，补齐旧纹信息，确立“真相成本很高”。
- 关键地图：`black_iron_mine`、`ancient_ruins`
- 代表任务：`矿路先问还剩什么`、`把失落信标抢回来`、`把门钥从死人手里拿走`、`先把自己练到能扛真相`

## 第四章·兽谷问门（6 条）

- 核心推进：噬魂兽谷的“漏口”风险实化，随后上灵脊岭获取行令并完成叩门前修为门槛。
- 关键地图：`beast_valley`、`spirit_ridge`
- 代表任务：`裂谷里还有活人`、`谷底那颗心`、`守岭的人没睡`、`叩门之前先成其身`

## 第五章·残宫封天（3 条）

- 核心推进：进入天穹残宫，确认封柱危机，回收坠星铁并夺回封天核心。
- 关键地图：`sky_ruins`
- 代表任务：`残宫里先听一句实话`、`先把坠星铁拿回来`、`把那颗封柱核心带走`

## 终章·道劫未尽（1 条）

- 核心推进：明确“当前只是第一层真相”，把目标推进到半步筑基，作为当前版本主线收束点。
- 关键地图：`sky_ruins`（目标历练图）、`yunlai_town`（交付）
- 代表任务：`道基初誓`（`targetRealmLv=30`，半步筑基）

## 5. 地图分层与推进结构

## 5.1 凡武期主干地图链

- 主城起点：`yunlai_town`（云来镇）
- 镇外过渡：`wildlands`（荒野）
- 调查推进：`bamboo_forest`（青竹林）
- 真相前层：`black_iron_mine`（玄铁矿洞）+ `ancient_ruins`（断碑遗迹）
- 高危门槛：`beast_valley`（噬魂兽谷）+ `spirit_ridge`（灵脊岭）
- 凡武尾段：`sky_ruins`（天穹残宫）

核心连通特征（按 portal）：

- `yunlai_town` 直连 `wildlands`、`bamboo_forest`、`spirit_ridge`，并连多个镇内室内图。
- `bamboo_forest` 向 `black_iron_mine` 与 `ancient_ruins` 分流，并可进入 `beast_valley`。
- `ancient_ruins`、`beast_valley`、`spirit_ridge`、`sky_ruins` 之间形成后段互联网。

## 5.2 练气期结构

- 主城：`qizhen_crossing`（栖真渡，推荐“练气一层-半步筑基”）
- 五行外域：
- 金：`cleft_blade_plain`（裂锋原）
- 木：`verdant_vine_vale`（青萝谷）
- 水：`cold_tide_marsh`（寒汐泽）
- 火：`emberfall_court`（赤陨庭）
- 土：`deepvein_ridge`（厚脉岭）
- 终图：`guizang_vein_cavern`（归藏脉窟，`29-30级`）
- 额外高危图：`ruined_cavern_manor`（破败洞府，推荐金丹一层）

连通特征（按 portal）：

- `qizhen_crossing` 向五行图放射，并与 `sky_ruins` 存在传送连接。
- 五行图汇流至 `guizang_vein_cavern`。
- `guizang_vein_cavern` 可进入 `ruined_cavern_manor`。

## 5.3 室内/子区域图层

- 云来镇室内：脚店二楼、药铺地窖、旧祠暗室、旧屋矿窖、南门门楼二层、湖底洞天。
- 凡武高危子图：荒骨风穴、青螂巢庭、玄脉熔心、断碑主殿、血祭坛、悬门旧关、封天核心井。
- 练气组合子图：`compose/` 下存在裂锋原、青萝谷、栖真渡相关拼图块（用于大图构建）。

## 6. 关键地图定位（检索关键词：地图职责、推进定位）

- `yunlai_town`：叙事起点与任务交付核心；承接生存教学与北井异动线。
- `wildlands`：第一阶段资源/治安/生存压力场；推动炼体与锻骨门槛。
- `bamboo_forest`：从“刷怪”转向“调查”的章节门；引出矿洞遗迹主线。
- `black_iron_mine`：矿路线索、信标争夺、炉心与矿监相关支线集中区。
- `ancient_ruins`：旧纹、门钥、残碑秩序线索集中区。
- `beast_valley`：旧阵漏口压力实化区域；主线风险显著抬升。
- `spirit_ridge`：凡武末段修为与资格门槛区，连接残宫。
- `sky_ruins`：凡武主线收束图；封柱核心与终章目标关联图。
- `qizhen_crossing`：练气阶段枢纽城，向五行图分发。
- `guizang_vein_cavern`：练气尾段汇流终图，挂接终章支线。
