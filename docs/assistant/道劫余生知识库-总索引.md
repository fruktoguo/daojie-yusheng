# 道劫余生知识库总索引

这组文档用于给《道劫余生》相关智能体提供检索入口。

建议检索顺序：

1. 先按问题类型命中对应专题库
2. 再回到本索引确认文档边界
3. 多专题问题允许交叉检索，不允许脱离文档事实自由脑补

## 使用原则

- 只以当前版本仓库内容为准
- 优先检索文档，不凭空补设定
- 查不到就明确说知识库未命中或当前版本未明确
- 涉及数值、路线、掉落、剧情时，优先给事实，再给简短归纳
- 智能体只回答《道劫余生》游戏内问题

### 剧情、地图、任务

### 怪物、掉落、采集、挖矿、刷图、练级路线
