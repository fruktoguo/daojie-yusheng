## 1. 怪物层级与经验机制（服务端规则）

### 1.1 怪物层级（`tier`）
- `mortal_blood`：凡血（普通怪）
- `variant`：异种（精英）
- `demon_king`：妖王（BOSS）

默认血脉经验倍率（`MONSTER_TIER_EXP_MULTIPLIERS`）：
- `mortal_blood = 1`
- `variant = 5`
- `demon_king = 100`

### 1.2 越级经验修正
- 玩家等级低于怪物：每级额外倍率  
  - 凡血：`1.1`
  - 异种：`1.25`
  - 妖王：`1.4`
- 玩家等级高于怪物：每级保留倍率 `0.5`
- 等级差上限按 `10` 级封顶计算（防止极端爆炸）

### 1.3 怪物等级经验衰减（怪物自身等级段）
- `1-18`：每级额外乘 `0.98`
- `19-30`：每级额外乘 `0.95`
- `31+`：每级额外乘 `0.92`

### 1.4 境界压制（战斗体感关键）
- 每差 1 级境界：高打低 `1.2` 倍，低打高 `0.8` 倍（幂次连乘）

## 3. 掉落类型与关键任务掉落

### 3.1 怪物掉落类型
- `material` 材料
- `equipment` 装备
- `skill_book` 功法/技能书

- `失落信标核心`：`m_mine_ghoul`（玄铁矿洞）
- `矿晶核`：`m_mine_ghoul`（玄铁矿洞）

- `遗迹钥石`：`m_rune_watcher`（断碑遗迹）
- `兽谷核心`：`m_valley_alpha`（噬魂兽谷）
- `谷皇獠牙`：`m_valley_emperor`（噬魂兽谷）
- `灵岭行令`：`m_ridge_ghost`（灵脊岭）
- `天封核心`：`m_star_devourer`（天穹残宫）

### 3.3 各区掉落结构特征（聚合）
- 云来镇：以基础材料为主，低概率出早期书/鞋
- 荒野/青竹林：材料 + 入门书，开始出现精英书/轻装
- 玄铁矿洞/断碑：任务道具密度显著上升，书和装备开始进入中期梯度
- 噬魂兽谷-天穹残宫：妖王条目中高概率书和装备明显增多
- 练气五行区：材料、装备、书三线并行，`Lv29` 段开始稳定出现破镜丹低概率条目
- 归藏脉窟：`Lv30` 终段，包含半基相关材料与高阶书条目

## 6. 采集与挖矿机制（公式摘录）

## 6.1 矿点掉落概率（按单次伤害）

来源常量：
- `ORE_REWARD_BASE_DAMAGE = 100`
- `ORE_REWARD_DAMAGE_SCALE = 3`

结算逻辑（`getOreRewardChanceBps`）：
- 若单次伤害 `<100`：概率为 `baseChanceBps/2`（至少 1，最多 10000）
- 若单次伤害 `>=100`：初始为 `baseChanceBps`
- 每跨过一次 `*3` 阈值，再叠加 `+baseChanceBps`

可检索表达：
- `chanceBps = baseChanceBps/2`（damage < 100）
- `chanceBps = baseChanceBps * (1 + n)`（damage >= 100，n 为跨过 3 倍阈值次数）

教程文本示例（已在仓库生成内容）：
- 灵石矿：基础 `20/10000`，`<100` 伤害为 `0.1%`，`300+` 为 `0.4%`，`900+` 为 `0.6%`
- 玄铁矿：基础 `50/10000`，`<100` 伤害为 `0.25%`，`300+` 为 `1.0%`，`900+` 为 `1.5%`
- 矿点掉落按每击独立判定，不看累计总伤害

## 6.2 草药采集耗时与采集技艺

草药基础采集 tick（服务端）：
- `nativeGatherTicks = ceil((草药等级 + 品阶值 - 1) * 0.5)`

采集技艺速度：
- `speedRate = gatherLevel * 0.02`
- `finalTicks = ceil(nativeGatherTicks * (1 / (1 + speedRate)))`

要点：
- 采集等级越高，单次采集耗时越低
- 草药节点有回生时间窗口（`refreshTicksMin/Max`）

## 7. 阶段性刷图与练级路线（基于现有等级带与任务链）

## 7.1 凡武段（1-18）
- `Lv1-3`：云来镇（鼠、暴徒） -> 荒野（野彘/匪徒/毒蛛）  
  目标：基础材料、早期装备与入门书。
- `Lv4-6`：青竹林 + 青螂巢庭  
  目标：`螳锋`、`翠竹心`、步法/内功入门书。
- `Lv6-11`：玄铁矿洞 + 断碑遗迹  
  目标：主线任务道具（信标核心/钥石/矿洞令牌）、中段装备书。
- `Lv10-16`：噬魂兽谷 -> 灵脊岭  
  目标：兽谷核心/谷皇獠牙/灵岭行令，准备终章前置。
- `Lv15-18`：天穹残宫 + 封天核心井  
  目标：`天封核心`、星陨系材料与终段功法书。

## 7.2 练气段（19-30）
- `Lv19-26`：先跑五行图（裂锋/青萝/寒汐/赤陨/厚脉）  
  目标：对应五行材料、装备、同系书；同步吃对应灵气资源点。
- `Lv27-29`：各图异种 + 妖王循环  
  目标：高价值书条目、`破镜丹`低概率条目、成套装备补齐。
- `Lv29-30`：归藏脉窟  
  目标：归藏脉核、半基灵胚、终段书与半步筑基前置积累。

## 7.3 风险提示
- `破败洞府` 为高危越级内容（`terrainRealmLv=43`），不属于常规练气刷图线。
