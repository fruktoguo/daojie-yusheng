### C1 [P1] GmMapEditor 异步方法 .catch(() => {}) 吞网络错误

- 文件：`packages/client` — GmMapEditor 相关文件
- 现象：异步方法 `.catch(() => {})` 吞掉网络错误，用户无反馈
- 影响：GM 操作失败时无提示，需手动刷新才能发现
- 修复方案：catch 内添加 toast/console.error 提示

### C2 [P1] MarketAuctionView 通过 as any 访问 tradeDialogView

- 文件：`packages/client` — MarketAuctionView 相关文件
- 现象：通过 `as any` 访问 tradeDialogView，运行时崩溃风险
- 影响：重构面板时编译不报错但运行时白屏
- 修复方案：补充正确类型声明，移除 as any

### C3 [P2] GmMapEditor 无 dispose，window 事件监听器永不移除

- 文件：`packages/client` — GmMapEditor 相关文件
- 现象：window 事件监听器永不移除
- 影响：当前单例使用无泄漏，仅代码规范问题
- 修复方案：添加 dispose 方法，注册时保存引用以便移除

### C4 [P2] MarketBrowseView/CraftWorkbenchModal 多处 as any 绕过类型检查

- 影响：重构时的隐患
- 修复方案：补充正确类型，移除 as any

### C5 [P2] localStorage 写入 catch{} 静默失败

- 文件：`packages/client` — 强化历史相关文件
- 现象：localStorage 写入 catch{} 静默失败，强化历史丢失无提示
- 影响：隐私模式下强化历史不保存，用户无感知
