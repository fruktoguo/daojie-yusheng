# 宗门地图与护宗大阵商业级开发计划

> 解决问题：实现完整的宗门系统——建宗令创建宗门、独立宗门地图、宗门核心交互、护宗大阵、重启恢复。

## 目标

玩家使用建宗令 → 创建宗门真源 → 创建独立宗门地图(3x3) → 主世界入口 → 宗门核心交互 → 护宗大阵生效 → 重启后全部恢复。

## 核心规则

- 宗门地图初始 3x3 全石头，中心为宗门管理核心
- 石头血量按圈翻倍：第1圈 100000，第2圈 200000，第3圈 400000...
- 宗门内置固脉：核心 17x17 范围内地块不自动复生、临时地块不消散，但不提供减伤
- 护宗大阵：本宗门玩家可通行，外人阻挡；灵力按 3 天半衰期衰减

## 领域模型

### sect_state 表
`sect_id / name / founder_player_id / leader_player_id / status / entrance_instance_id / entrance_x,y / sect_instance_id / core_x,y / expansion_radius / created_at / updated_at`

### sect_member 表
`sect_id / player_id / role(leader/elder/member) / joined_at / status`

### 宗门地图实例 ID
`sect:<sectId>:main`，kind=sect，persistent=true，ownerSectId=sectId

## 依赖关系

推荐顺序：宗门真源 → 地图实例+入口 → 核心交互 → 护宗大阵 → 扩圈+石头血量 → UI+GM+with-db
