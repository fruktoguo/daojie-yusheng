# ADR-0005: AOI 与视野同步

- **状态**: 已采纳
- **日期**: 2025-02-01

## 决策

采用基于格子的 AOI 系统，只同步玩家视野范围内的实体变化，避免全图广播。

## 核心规则

- 视野半径：15 格（切比雪夫距离）
- 实体进入/离开视野时发送 enter/leave 通知
- NPC 等静态实体在首包发送，不参与动态 AOI
- 传送时清除旧视野并全量发送新视野
- 断线重连时全量发送当前视野

## 同步优化

- 批量合并同 tick 内的多个 AOI 变化为单包发送
- 高频属性（位置、HP）只对视野内观察者广播
- 低频属性（装备外观）变化时增量通知

## 实体类型与 AOI 行为

| 类型 | AOI 行为 |
|------|----------|
| Player | 进入/离开视野时通知 |
| Monster | 进入/离开视野时通知 |
| NPC | 静态，首包发送 |
| Item | 进入/离开视野时通知 |

## 验证方式（5000 人 / 10000 图口径）

| 指标 | 正常范围 | 告警阈值 |
|------|----------|----------|
| AOI 更新延迟 | < 50ms | > 200ms |
| 单玩家视野实体数 | < 100 | > 300 |
| AOI 广播包大小 | < 1KB | > 5KB |
| 单地图 AOI 计算开销 | < 10ms | > 50ms |
