# ADR-0003: 网络同步分层

- **状态**: 已采纳
- **日期**: 2025-02-01

## 决策

采用分层同步。将数据按频率和重要性分层，按不同策略同步：

| 层次 | 内容 | 同步策略 |
|------|------|----------|
| 首包 | 地图静态、玩家完整状态 | 进入场景时全量 |
| 高频 | 位置、气机、战斗状态 | 每 tick 增量 |
| 低频 | 背包、属性、技能 | 变化时增量 |
| 按需 | 详情、历史、排行 | 请求时拉取 |

## 约束

- 高频包禁止混入静态数据、长文本、完整详情
- 能发 ID/revision/patch 的不发完整对象
- 能单播就不 AOI，能 AOI 就不全图

## 验证方式（5000 人 / 30Mbps 口径）

- [ ] 高频包大小不超过 1KB
- [ ] 100 玩家同屏带宽不超过 100KB/s
- [ ] 断线重连恢复时间不超过 3 秒
- [ ] 5000 人总出口带宽不超过 30Mbps
