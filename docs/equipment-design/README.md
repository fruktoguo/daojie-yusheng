# 装备设计

本目录用于集中整理《道劫余生》的装备设计文档。

- [装备框架设计](./equipment-framework.md)
- [物品强化系统 V1 设计](./item-enhancement-v1.md)

说明：

- 本目录先聚焦 `凡人篇`，即 `云来镇 -> 荒野 -> 青竹林 -> 玄铁矿洞 / 断碑遗迹 -> 噬魂兽谷 -> 灵脊岭` 这条前期主线
- 设计默认兼容当前项目的 `5` 个装备槽位：`weapon / head / body / legs / accessory`
- [装备框架设计](./equipment-framework.md) 负责定义装备配置模型、触发器、条件系统与运行时结算链路
- [凡人篇装备设计](./fanren-equipment-design.md) 负责基于该框架去设计前期装备池、剧情来源与流派分工
- [当前装备总表](./current-equipment-catalog.md) 负责汇总当前真源中的全部装备，并附带结构化 JSON 清单
