# 开发文档目录

本目录用于承载仍在设计、拆解、评审、待落地的开发文档。

放在这里的文档默认满足以下口径：

- 仍处于设计或待开发阶段，不视为当前主线已完成事实
- 允许描述目标架构、演进步骤、风险、验证计划
- 落点仍以 `packages/*` 主线为准，不恢复旧线目录作为新实现入口
- 文档完成落地并成为正式基线后，应视情况并入对应正式文档位置
