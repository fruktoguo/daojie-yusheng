# 术语表

供 AI agent 和人类开发者统一理解项目术语。

## 修仙体系

| 术语 | 英文 | 定义 |
|------|------|------|
| 境界 | realm | 修仙者的修为等级，如凡人、练气、筑基 |
| 灵气 | aura | 天地间的修仙能量，地图有灵气浓度 |
| 气机 | qi | 修仙者体内的能量，用于施法和战斗 |
| 功法 | technique | 修炼方法，包含多个技能，可升级层数 |
| 术法 | arts | 战斗技能类功法 |
| 层数 | layer | 功法的修炼等级 |
| 六维 | six attributes | 体质、神识、悟性、根骨、力量、经脉 |
| 丹药 | pill | 炼丹产出的消耗品，提供各种效果 |
| 法器 | artifact | 高阶装备，具有特殊能力 |
| 风水 | fengshui | 建筑的气场配置，影响修炼效率 |

## 境界等级

| 境界 | 英文 | 等级范围 |
|------|------|----------|
| 凡人 | mortal | 1-10 |
| 练气 | qi_refining | 11-20 |
| 筑基 | foundation | 21-30 |
| 金丹 | golden_core | 31-40 |
| 元婴 | nascent_soul | 41-50 |

## 运行时

| 术语 | 英文 | 定义 |
|------|------|------|
| 息 | tick | 服务端权威时间单位，当前 1Hz |
| 权威 | authority | 某状态的最终裁定方，通常是服务端 |
| 真源 | source of truth | 某状态的权威存储位置 |
| 意图 | intent | 玩家提交的操作请求，待服务端在 tick 内执行 |
| 投影 | projection | 服务端状态在客户端的只读映射 |

## 网络同步

| 术语 | 英文 | 定义 |
|------|------|------|
| AOI | Area of Interest | 视野范围，只向视野内玩家广播 |
| 单播 | unicast | 只发给单个玩家 |
| 全图广播 | map broadcast | 发给同一地图所有玩家 |
| 增量同步 | delta sync | 只发送变化部分，非全量 |
| 首包 | initial packet | 进入场景时的完整状态包 |

## 持久化

| 术语 | 英文 | 定义 |
|------|------|------|
| 落库 | persist / flush | 将内存状态写入数据库 |
| 在线态 | online state | Redis 中的实时状态，非持久化真源 |
| outbox | outbox | 待写入队列，用于异步持久化 |
| 刷盘 | flush | 批量将内存数据写入持久化存储 |

## 客户端

| 术语 | 英文 | 定义 |
|------|------|------|
| 表现层 | presentation | 纯显示逻辑，不影响游戏正确性 |
| 预测 | prediction | 客户端提前展示预期结果，待服务端确认 |
| HUD | heads-up display | 常驻界面元素，如血条、气机条 |
| 面板 | panel | 可打开关闭的功能界面 |

## 架构

| 术语 | 英文 | 定义 |
|------|------|------|
| 热路径 | hot path | 高频执行的代码路径，需严格优化 |
| 冷路径 | cold path | 低频执行的代码路径，可接受较高开销 |
| 边界 | boundary | 模块间的职责分界线 |
| 链路 | chain | 数据从输入到输出的完整流动路径 |

## 战斗系统

| 术语 | 英文 | 定义 |
|------|------|------|
| 物攻 | physAtk | 物理攻击力 |
| 法攻 | spellAtk | 法术攻击力 |
| 物防 | physDef | 物理防御力 |
| 法防 | spellDef | 法术防御力 |
| 破防 | breakPower | 穿透防御的能力 |
| 韧性 | resolvePower | 抵抗控制的能力 |
| 暴击 | crit | 造成额外伤害的概率 |
| 闪避 | dodge | 躲避攻击的概率 |

## 内容配置

| 术语 | 英文 | 定义 |
|------|------|------|
| 品阶 | grade | 物品或功法的稀有度等级 |
| 基准值 | baseline | 某等级某类型的标准数值 |
| 倾向 | tendency | 怪物属性相对基准的偏移百分比 |
| 刷新点 | spawn point | 怪物或资源出现的位置 |
