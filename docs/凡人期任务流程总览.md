# 凡人期任务流程总览

本文汇总当前版本凡人期主线与支线任务流程，方便查阅接取顺序、目标地图、奖励与建议推进境界。

说明：

- 范围覆盖 `序章` 到 `第五章`，以及对应章节支线。
- “参考境界”是推荐推进区间，不等同于严格接取门槛；涉及突破的任务以任务目标要求为准。
- 奖励只统计任务直接奖励，不包含打怪掉落、商店购买、地图探索拾取等额外来源。

## 推荐阅读顺序

如果只想快速推进凡人期，建议按以下顺序处理：

1. 每章先清主线，确保地图与下一阶段境界解锁。
2. 同章支线在主线目标地图顺手完成，能减少往返。
3. 序章与第一章完成后，再去云来镇左下角找文籍师叔补低阶锻体书。
4. 到第五章前，不必强刷高阶功法掉落，先用主线给的功法把境界推进上去更稳。
