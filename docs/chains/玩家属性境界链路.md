# 玩家属性境界链路

## 概述

玩家属性境界链路负责玩家六维属性结算、境界升级/突破、功法修炼、天门灵根和体修等全部成长线的运行时结算。

## 链路流程

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  属性来源   │────▶│  属性结算   │────▶│  面板输出   │────▶│  境界突破   │
│  (多维叠加) │     │  (六维→面板)│     │  (maxHp等)  │     │  (经验/条件)│
└─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘
```

## 核心文件

| 文件 | 职责 |
|------|------|
| `runtime/player/player-attributes.service.ts` | 属性结算器 |
| `runtime/player/player-progression.service.ts` | 成长结算器 |
| `runtime/player/player-realm-projection.helpers.ts` | 境界投影辅助 |
| `runtime/player/realm-runtime-exp.helpers.ts` | 经验计算辅助 |
| `runtime/player/player-buff-projection.helpers.ts` | Buff 投影辅助 |
| `runtime/player/runtime-player.entity.ts` | 玩家运行时实体 |

## 属性结算流程

```
PlayerAttributesService 属性结算
  │
  ├─▶ 基础属性（境界基准六维）
  │
  ├─▶ 装备加成
  │     - 装备基础属性
  │     - 强化加成
  │
  ├─▶ Buff 加成
  │     - 丹药 buff
  │     - 战斗 buff
  │     - 阵法 buff
  │
  ├─▶ 根骨加成
  │
  ├─▶ 功法加成
  │
  ├─▶ 折算最终六维
  │
  └─▶ 输出面板数值
        - maxHp / maxQi
        - 攻击 / 防御
        - 速度 / 闪避 / 暴击
        - 属性变化时按比例同步当前值
```

## 境界成长流程

```
PlayerProgressionService 成长结算
  │
  ├─▶ 境界升级
  │     - 修炼经验累积
  │     - 达到阈值自动升级
  │
  ├─▶ 境界突破
  │     - 突破条件检查（材料、属性、任务）
  │     - 突破成功/失败
  │
  ├─▶ 功法修炼
  │     - 功法经验分配
  │     - 功法等级提升
  │
  ├─▶ 天门灵根
  │     - HEAVEN_GATE_REALM_LEVEL = 18 开启
  │     - HEAVEN_GATE_MAX_SEVERED = 4 最多斩根数
  │
  ├─▶ 体修路线
  │
  └─▶ 根基底蕴
        - FOUNDATION_EXP_MULTIPLIER = 3
```

## 经验分配

```
战斗经验分配：
  │
  ├─▶ 击杀怪物 → 境界经验
  │     - SINGLE_COMBAT_REALM_EXP_CAP_MULTIPLIER = 5（单次上限倍率）
  │     - COMBAT_EXPERIENCE_ADVANTAGE_THRESHOLD ≥ 2（封顶倍率）
  │
  ├─▶ 修炼 → 修炼经验
  │
  └─▶ 制作 → 技能经验
```

## 关键约束

- **属性重算时机**: 装备变更、buff 变更、境界变更时触发
- **比例同步**: 属性变化时当前 HP/QI 按比例调整
- **服务端权威**: 所有属性计算由服务端执行
- **不参与 tick**: 属性结算由事件触发，非每 tick 重算
- **持久化**: 通过 flush 机制定期落库
