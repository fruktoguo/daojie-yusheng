# 链路文档索引

本目录描述系统核心数据流和职责边界，供 AI agent 理解跨模块改动影响范围。

## 链路列表

| 链路 | 描述 |
|------|------|
| [链路总览](链路总览.md) | 核心链路概述与分层 |
| [战斗链路](战斗链路.md) | 战斗系统数据流与目标架构 |

## 使用说明

- 新增链路文档时复制 `template.md`
- 链路文档描述"数据从哪来、经过谁、到哪去"
- 修改链路涉及的模块时，必须先读对应链路文档

## 与 ADR 的关系

- ADR 回答"为什么这样设计"
- 链路文档回答"数据怎么流动"
- 链路文档应引用相关 ADR
