# 宗门链路

## 概述

宗门链路负责宗门的创建、成员管理、领地扩展和护宗大阵的运行时编排。宗门拥有独立地图实例和完整的权限体系。

## 链路流程

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  创建宗门   │────▶│  成员管理   │────▶│  领地扩展   │────▶│  护宗大阵   │
│  (地图实例) │     │  (角色权限) │     │  (区块扩圈) │     │  (灵气运行) │
└─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘
```

## 核心文件

| 文件 | 职责 |
|------|------|
| `runtime/world/world-runtime-sect.service.ts` | 宗门运行时核心 |

## 创建宗门

```
WorldRuntimeSectService
  │
  ├─▶ 消耗建宗道具（useBehavior = 'create_sect'）
  ├─▶ 创建宗门地图实例
  │     - SECT_BASE_CLEAR_RADIUS = 1
  │     - SECT_FOUNDING_CLEAR_RADIUS = 2
  ├─▶ 放置入口/核心建筑
  └─▶ 持久化到 server_sect 表
```

## 成员管理

### 角色体系

| 角色 | 权限 |
|------|------|
| 宗主 | 全部权限 |
| 副宗主 | 大部分管理权限 |
| 长老 | 部分管理权限 |
| 内门弟子 | 基础权限 |
| 外门弟子 | 受限权限 |
| 杂役 | 最低权限 |
| 太上长老 | 荣誉角色 |

### 权限操作

- 护宗大阵开关
- 移除成员
- 修改职位
- 领地扩展

## 领地扩展

```
宗门领地扩展
  │
  ├─▶ SECT_EXPAND_CHUNK = 8（扩圈粒度）
  ├─▶ 消耗资源
  └─▶ 扩展地图边界
```

## 护宗大阵

```
护宗大阵运行时
  │
  ├─▶ SECT_GUARDIAN_INITIAL_AURA = 100000（初始灵气）
  ├─▶ SECT_INNATE_STABILIZER_RADIUS = 8（稳定半径）
  ├─▶ 灵气消耗/补充
  └─▶ 提供区域保护效果
```

## 关键约束

- **独立实例**: 每个宗门拥有独立地图实例
- **权限体系**: 操作受角色权限约束
- **持久化**: 宗门数据持久化到 `server_sect` 表
- **扩圈粒度**: 以 8 格为单位扩展
