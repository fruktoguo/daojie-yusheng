# 阵法链路

## 概述

阵法链路负责阵法的布置、开关、灵石补充和 tick 效果推进。支持临时阵法（阵盘布置）和持续性阵法（宗门护宗阵）两种生命周期。

## 链路流程

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  布阵       │────▶│  灵气注入   │────▶│  Tick 效果  │────▶│  灵气衰减   │
│  (阵盘消耗) │     │  (灵石补充) │     │  (区域效果) │     │  (半衰期)   │
└─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘
```

## 核心文件

| 文件 | 职责 |
|------|------|
| `runtime/world/world-runtime-formation.service.ts` | 阵法运行时核心 |
| `content/registries/formation-template.registry.ts` | 阵法模板注册表 |

## 布阵流程

```
WorldRuntimeFormationService
  │
  ├─▶ 消耗阵盘道具
  ├─▶ 消耗灵石
  ├─▶ 创建阵法实例
  │     - 类型: deployed（临时）/ persistent（持续）
  │     - 初始灵气
  │     - 效果范围
  └─▶ 注册到地图实例
```

## 生命周期

### 临时阵法（deployed）

```
阵盘布置 → 灵气注入 → 效果持续 → 灵气耗尽 → 消失
```

### 持续性阵法（persistent）

```
系统创建 → 持续运行 → 灵石补充维持 → 永久存在
```

## Tick 效果

```
每 tick 对每个活跃阵法：
  │
  ├─▶ 灵气衰减（半衰期机制）
  │
  ├─▶ 效果应用
  │     - 地形稳定
  │     - 灵气源
  │     - 边界屏障
  │     - 区域 buff
  │
  └─▶ 灵气归零 → 阵法失效/消失
```

## 操作

| 操作 | 说明 |
|------|------|
| 布阵 | 消耗阵盘 + 灵石 |
| 开关 | 切换阵法激活状态 |
| 补充 | 注入灵石延长寿命 |
| 拆除 | 主动移除阵法 |

## 关键约束

- **灵气衰减**: 半衰期机制，灵气随时间自然减少
- **持久化**: 阵法状态持久化到 `instance_formation_state` 表
- **模板注册**: 阵法定义通过 FormationTemplateRegistry 管理
- **效果范围**: 阵法效果限定在指定半径内
