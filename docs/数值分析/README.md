# 数值分析

本目录用于存放后续所有数据统计、数值分析、平衡阈值测算和配置对比结果。

当前约定：

- 只记录从当前生产主线配置、公式和运行时代码推导出的分析结果。
- 如果分析依赖特定公式，必须写明代码来源、统计口径和计算定义。
- 如果分析依赖内容配置，必须写明数据源路径。
- 分析文档不直接等同于玩法配置变更；需要落地调数时，仍以 `packages/server/data/content/**` 等配置真源为准。
