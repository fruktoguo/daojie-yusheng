# 1-18级普通玩家参考属性与怪物基准表

这份文档不是 GM 极限板，也不是刚升到该境界一瞬间的裸板。

这里统一采用更适合怪物策划的口径：

- 四维底盘：以 [breakthroughs.json](/home/yuohira/mud-mmo/packages/server/data/content/breakthroughs.json) 里的下一突破硬门槛为上沿，按普通玩家在当前级中后段的常见推进度估算。
- 装备：只取 [00_云来镇与荒野.json](/home/yuohira/mud-mmo/packages/server/data/content/items/装备/00_云来镇与荒野.json)、[01_青竹林.json](/home/yuohira/mud-mmo/packages/server/data/content/items/装备/01_青竹林.json)、[02_玄铁矿洞.json](/home/yuohira/mud-mmo/packages/server/data/content/items/装备/02_玄铁矿洞.json)、[03_断碑遗迹.json](/home/yuohira/mud-mmo/packages/server/data/content/items/装备/03_断碑遗迹.json)、[04_兽谷与灵脊岭.json](/home/yuohira/mud-mmo/packages/server/data/content/items/装备/04_兽谷与灵脊岭.json) 的常规掉落和任务装，按 4-5 件混搭估算，不算隐藏装、极限偏科装和刷满词条。
- 面板：按当前属性公式估算后，再额外整体上调 `50%`，然后统一抹零，作为怪物和 Boss 回标时更实用的统一战斗参考值，允许 `±10%~15%` 波动。
- 灵力：统一按生命值的一半取整。
- 产物：这张表同时作为 [monster-level-baselines.json](/home/yuohira/mud-mmo/packages/server/data/content/monster-level-baselines.json) 的设计来源。

## 怪物配置规则

后续怪物模板建议不要再直接写死最终面板，而是统一写“倍率”：

刷新时间也统一按血脉层次和等级走固定公式，怪物模板直接写计算后的 `respawnSec`：

- 普通怪：`tier = mortal_blood`，`respawnSec = Math.round(10 + (level - 1) * 0.1)`
- 精英怪：`tier = variant`，`respawnSec = Math.round(15 + (level - 1) * 0.2)`
- Boss：`tier = demon_king`，`respawnSec = Math.round(30 + (level - 1) * 1)`

这里的等级以模板 `level` 为准，`level = 1` 时分别对应 `10 / 15 / 30` 息。

- `100 = 100%`
- `50 = 50%`
- `135 = 135%`

推荐字段：

- `hpRate`
- `qiRate`
- `attackRate`
- `defenseRate`
- `accuracyRate`
- `moveSpeedRate`

示例：

```json
{
  "level": 10,
  "hpRate": 60,
  "qiRate": 40,
  "attackRate": 55,
  "defenseRate": 50,
  "accuracyRate": 85,
  "moveSpeedRate": 90
}
```

这只怪在 `Lv.10` 的实际面板就是：

- 生命：`1260 × 60% = 756`
- 灵力：`630 × 40% = 252`
- 攻击：`110 × 55% = 60.5`
- 防御：`170 × 50% = 85`
- 命闪：`150 × 85% = 127.5`
- 移速：`210 × 90% = 189`

实际落地时建议统一 `Math.round`。

## 读表结论

- `1-5` 级玩家还在凡俗境模板里，真正拉开差距的是装备和体魄 / 根骨提供的生命乘区，所以这段普通怪不能再按 `几十血、个位攻击` 去写。
- `6-12` 级玩家已经开始进入明显的中段膨胀区，原因不是单纯武器变强，而是四维底盘、装备值和 `realmLv` 的 `1.1` 指数乘区叠在一起。
- `13+` 以后最容易低估的是对抗值。当前公式里身法直接抬命闪，根骨直接抬化解，怪物如果还停留在旧模板的副属性水平，会出现“玩家几乎稳命中、怪几乎打不中”的情况。
- `16-18` 级已经是凡武末段。这个区间的普通玩家并不只是血量高，攻击、抗压和对抗值都已经到可以做 Boss 技能联动的水平，精英怪和 Boss 必须带技能，不然只会被平推。

## 给怪物设计的直接参考

- 普通怪：建议对标同级普通玩家的 `45%~65%` 生命、`35%~50%` 攻击、`35%~50%` 防御，命闪不要超过玩家。
- 精英怪：建议对标同级普通玩家的 `90%~130%` 生命、`70%~95%` 攻击、`60%~80%` 防御，至少带 `1` 个主动技能和 `1` 个明显战斗标签。
- 子图 Boss：建议对标同级普通玩家的 `180%~240%` 生命、`95%~115%` 攻击、`80%~100%` 防御，至少带 `2` 个技能，其中 `1` 个要改变站位或节奏。
- 主线 Boss：建议对标同级普通玩家的 `280%~380%` 生命、`115%~135%` 攻击、`100%~115%` 防御，必须有完整技能循环，不能只是大号精英怪。

## 备注

- 这份表优先服务怪物、精英和 Boss 数值回标，不追求和运行时某一套具体配装逐点完全一致。
- 后面如果要把怪物整批重做，最稳妥的做法是先按这张表定七张战斗图的玩家参考板，再反推普通怪、精英怪、Boss 的技能数和面板系数。
